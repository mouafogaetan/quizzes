PARTIE A : ÉVALUATION DES RESSOURCES (15 points)

Exercice 1: (4.5 points)

I) On considère l'équation (E) : $x^{4} - 8x^{3} + 19x^{2} - 8x + 1 = 0$.

1. Montrer que 0 n'est pas une solution de cette équation.

Si $x = 0$ était solution, alors en remplaçant dans (E) :

$0^4 - 8 \times 0^3 + 19 \times 0^2 - 8 \times 0 + 1 = 1 \neq 0$

Donc 0 n'est pas solution de (E).

2. Démontrer que (E) est équivalente à l'équation : $x^{2} - 8x + 19 - \frac{8}{x} + \frac{1}{x^{2}} = 0$.

Comme $x \neq 0$ (d'après la question 1), on peut diviser l'équation (E) par $x^2$ :

$\frac{x^4}{x^2} - \frac{8x^3}{x^2} + \frac{19x^2}{x^2} - \frac{8x}{x^2} + \frac{1}{x^2} = 0$

Ce qui donne : $x^2 - 8x + 19 - \frac{8}{x} + \frac{1}{x^2} = 0$

Les deux équations sont donc équivalentes pour $x \neq 0$.

3. On pose : $X = x + \frac{1}{x}$. Montrer que $x^{2} + \frac{1}{x^{2}} = X^{2} - 2$.

On a : $X = x + \frac{1}{x}$

Alors : $X^2 = \left(x + \frac{1}{x}\right)^2 = x^2 + 2 \cdot x \cdot \frac{1}{x} + \frac{1}{x^2} = x^2 + 2 + \frac{1}{x^2}$

Donc : $x^2 + \frac{1}{x^2} = X^2 - 2$

4. En déduire que si x est solution de (E), alors X est solution de l'équation (E') : $X^{2} - 8X + 17 = 0$.

D'après la question 2, si x est solution de (E), alors :

$x^2 + \frac{1}{x^2} - 8x - \frac{8}{x} + 19 = 0$

Or, d'après la question 3 : $x^2 + \frac{1}{x^2} = X^2 - 2$

Et on a aussi : $-8x - \frac{8}{x} = -8\left(x + \frac{1}{x}\right) = -8X$

Donc l'équation devient : $(X^2 - 2) - 8X + 19 = 0$

Soit : $X^2 - 8X + 17 = 0$

Ainsi, si x est solution de (E), alors X est solution de (E').

5. Résoudre l'équation (E') et en déduire les solutions de (E).

Résolution de (E') : $X^2 - 8X + 17 = 0$

Discriminant : $\Delta = (-8)^2 - 4 \times 1 \times 17 = 64 - 68 = -4 < 0$

L'équation (E') n'a pas de solutions réelles.

Par conséquent, l'équation (E) n'a pas de solutions réelles.

II) Soit P un polynôme de degré 3 à coefficients réels tel que :

$\begin{cases} P(0) = 0 \\ P(x+1) - P(x) = x^{2} \end{cases}$

6. En supposant que $P(x) = ax^{3} + bx^{2} + cx$, déterminer les réels a, b et c.

On a $P(x+1) - P(x) = a(x+1)^3 + b(x+1)^2 + c(x+1) - (ax^3 + bx^2 + cx)$

$= a[(x+1)^3 - x^3] + b[(x+1)^2 - x^2] + c[(x+1) - x]$

$= a(3x^2 + 3x + 1) + b(2x + 1) + c$

$= 3ax^2 + (3a + 2b)x + (a + b + c)$

Cette expression doit être égale à $x^2$ pour tout x, donc :

$\begin{cases} 3a = 1 \\ 3a + 2b = 0 \\ a + b + c = 0 \end{cases}$

De la première équation : $a = \frac{1}{3}$

De la deuxième : $3 \times \frac{1}{3} + 2b = 0 \Rightarrow 1 + 2b = 0 \Rightarrow b = -\frac{1}{2}$

De la troisième : $\frac{1}{3} - \frac{1}{2} + c = 0 \Rightarrow -\frac{1}{6} + c = 0 \Rightarrow c = \frac{1}{6}$

Donc : $P(x) = \frac{1}{3}x^3 - \frac{1}{2}x^2 + \frac{1}{6}x$

Exercice 2: (4.75 points)

1. Résoudre dans $\mathbb{R}$ l'équation $\sqrt{3x+1} = 2x-3$.

Conditions d'existence : $3x + 1 \ge 0 \Rightarrow x \ge -\frac{1}{3}$

Et $2x - 3 \ge 0 \Rightarrow x \ge \frac{3}{2}$ (car une racine carrée est toujours positive)

Donc domaine de validité : $x \ge \frac{3}{2}$

Élevons au carré les deux membres :

$(\sqrt{3x+1})^2 = (2x-3)^2$

$3x + 1 = 4x^2 - 12x + 9$

$4x^2 - 15x + 8 = 0$

Discriminant : $\Delta = (-15)^2 - 4 \times 4 \times 8 = 225 - 128 = 97$

$x = \frac{15 \pm \sqrt{97}}{8}$

Seule la solution $x = \frac{15 + \sqrt{97}}{8}$ est dans le domaine de validité car $\frac{15 + \sqrt{97}}{8} > \frac{15 + 9}{8} = 3 > \frac{3}{2}$

Et $x = \frac{15 - \sqrt{97}}{8} < \frac{15 - 9}{8} = \frac{6}{8} = 0.75 < \frac{3}{2}$ (non valide)

Solution : $x = \frac{15 + \sqrt{97}}{8}$

2. Résoudre dans $\mathbb{R}$ l'inéquation $\sqrt{x+2} \ge x$.

Condition d'existence : $x + 2 \ge 0 \Rightarrow x \ge -2$

Deux cas à considérer :

Cas 1 : $x < 0$ (alors $\sqrt{x+2} \ge 0 \ge x$, donc l'inégalité est toujours vraie)

Mais attention : il faut aussi que $x \ge -2$

Donc pour $x \in [-2, 0[$, l'inégalité est vraie

Cas 2 : $x \ge 0$ (on peut élever au carré car les deux membres sont positifs)

$(\sqrt{x+2})^2 \ge x^2$

$x + 2 \ge x^2$

$x^2 - x - 2 \le 0$

Racines du trinôme : $x = \frac{1 \pm \sqrt{1+8}}{2} = \frac{1 \pm 3}{2}$

Soit $x = -1$ et $x = 2$

Le trinôme est négatif entre les racines : $-1 \le x \le 2$

Mais comme on est dans le cas $x \ge 0$, on a $0 \le x \le 2$

En combinant les deux cas :

Solution : $x \in [-2, 2]$

3. Résoudre dans $\mathbb{R}^{2}$ les systèmes suivants :

a) $(S_1) : \begin{cases} -2x + 5y = 1 \\ 4x - 10y = -2 \end{cases}$

Calcul du déterminant principal :

$\Delta = \begin{vmatrix} -2 & 5 \\ 4 & -10 \end{vmatrix} = (-2)(-10) - (5)(4) = 20 - 20 = 0$

Le système peut être soit impossible, soit indéterminé.

On observe que la deuxième équation est exactement -2 fois la première :

$4x - 10y = -2 \Leftrightarrow -2(-2x + 5y) = -2(1)$

Les deux équations sont proportionnelles, donc le système est indéterminé.

Résolvons en fonction d'un paramètre :

De la première équation : $-2x + 5y = 1 \Rightarrow 5y = 1 + 2x \Rightarrow y = \frac{1 + 2x}{5}$

Solutions : $\left(x, \frac{1 + 2x}{5}\right)$ pour tout $x \in \mathbb{R}$

b) $(S_2) : \begin{cases} 3x - 4y = 7 \\ 2x + 3y = 1 \end{cases}$

Calcul du déterminant principal :

$\Delta = \begin{vmatrix} 3 & -4 \\ 2 & 3 \end{vmatrix} = (3)(3) - (-4)(2) = 9 + 8 = 17 \neq 0$

Le système admet une solution unique.

Calcul de x : $x = \frac{\begin{vmatrix} 7 & -4 \\ 1 & 3 \end{vmatrix}}{\Delta} = \frac{(7)(3) - (-4)(1)}{17} = \frac{21 + 4}{17} = \frac{25}{17}$

Calcul de y : $y = \frac{\begin{vmatrix} 3 & 7 \\ 2 & 1 \end{vmatrix}}{\Delta} = \frac{(3)(1) - (7)(2)}{17} = \frac{3 - 14}{17} = \frac{-11}{17}$

Solution unique : $\left(\frac{25}{17}, -\frac{11}{17}\right)$

c) $(S_3) : \begin{cases} 2x + 2y = 12 \\ xy = 16 \end{cases}$

Simplifions la première équation : $2x + 2y = 12 \Rightarrow x + y = 6$

On a donc : $\begin{cases} x + y = 6 \\ xy = 16 \end{cases}$

x et y sont les racines de l'équation du second degré : $t^2 - St + P = 0$ où S = 6 et P = 16

Soit : $t^2 - 6t + 16 = 0$

Discriminant : $\Delta = (-6)^2 - 4 \times 1 \times 16 = 36 - 64 = -28 < 0$

Le système n'a pas de solutions réelles.

Exercice 3: (5.75 points)

Soit $Q$ le polynôme de la variable réelle $x$ défini par $Q(x) = 3x^{3} + ax^{2} + bx + 2$.

1. Déterminer les réels a et b pour que -1 et 2 soient des racines de Q.

Si -1 est racine : $Q(-1) = 3(-1)^3 + a(-1)^2 + b(-1) + 2 = -3 + a - b + 2 = a - b - 1 = 0$

Si 2 est racine : $Q(2) = 3(2)^3 + a(2)^2 + b(2) + 2 = 24 + 4a + 2b + 2 = 4a + 2b + 26 = 0$

On a donc le système :

$\begin{cases} a - b = 1 \\ 4a + 2b = -26 \end{cases}$

Simplifions la deuxième équation : $2a + b = -13$

Résolvons :

De la première : $a = b + 1$

Substituons dans la deuxième : $2(b + 1) + b = -13 \Rightarrow 2b + 2 + b = -13 \Rightarrow 3b = -15 \Rightarrow b = -5$

Alors $a = -5 + 1 = -4$

a = -4, b = -5

On donne maintenant $Q(x) = 3x^{3} - 6x^{2} - 3x + 6$.

2. Montrer que 2 est une racine de Q.

$Q(2) = 3(2)^3 - 6(2)^2 - 3(2) + 6 = 3 \times 8 - 6 \times 4 - 6 + 6 = 24 - 24 - 6 + 6 = 0$

Donc 2 est bien une racine de Q.

3. Montrer que Q peut s'écrire sous la forme $Q(x) = (x-2)(cx^{2}+dx+e)$, où c, d, et e sont des réels à déterminer.

Effectuons la division euclidienne de Q(x) par (x-2) :

Donc $Q(x) = (x-2)(3x^2 + 0x - 3) = (x-2)(3x^2 - 3)$

c = 3, d = 0, e = -3

4. En déduire la résolution dans $\mathbb{R}$ de l'équation $Q(x) = 0$.

$Q(x) = (x-2)(3x^2 - 3) = 3(x-2)(x^2 - 1) = 3(x-2)(x-1)(x+1)$

Les racines de Q sont les solutions de :

$(x-2)(x-1)(x+1) = 0$

Solutions : x = -1, x = 1, x = 2

PARTIE B : ÉVALUATION DES COMPÉTENCES (5 points)

Tâches:

1. Déterminer les dimensions de la parcelle de maïs pour que sa surface soit maximale.

Soit x la longueur parallèle au mur, et y les deux autres côtés.

Longueur de grillage disponible : $x + 2y = 100$ m

Surface : $S = x \cdot y$

De la contrainte : $x = 100 - 2y$

Donc $S(y) = (100 - 2y)y = 100y - 2y^2$

Il s'agit d'une fonction du second degré concave (coefficient -2 < 0) qui atteint son maximum au sommet.

Abscisse du sommet : $y = -\frac{b}{2a} = -\frac{100}{2 \times (-2)} = 25$ m

Alors $x = 100 - 2 \times 25 = 50$ m

Dimensions optimales : longueur = 50 m, largeur = 25 m

2. Combien M. Jean devra-t-il prévoir pour acheter les semences de maïs si le prix du mètre carré est de 1850 F?

Surface maximale : $S = 50 \times 25 = 1250$ m²

Coût des semences : $1250 \times 1850 = 2 312 500$ F

M. Jean devra prévoir 2 312 500 F

3. Calculer la somme dont M. Jean disposera à la fin du troisième trimestre et vérifier si elle est suffisante pour couvrir les frais d'achat de semences de tomates.

Calculons d'abord le taux d'intérêt trimestriel t% :

Capital initial : $C_0 = 300 000$ F

Capital après 2 trimestres : $C_2 = 324 300$ F

Formule des intérêts composés : $C_n = C_0 (1 + i)^n$ où $i = \frac{t}{100}$

Donc $324 300 = 300 000 (1 + i)^2$

$(1 + i)^2 = \frac{324 300}{300 000} = 1.081$

$1 + i = \sqrt{1.081} = 1.04$ (car le taux est positif)

Donc $i = 0.04$, soit $t = 4\%$ par trimestre

Capital après 3 trimestres : $C_3 = 300 000 \times (1.04)^3 = 300 000 \times 1.124864 = 337 459.20$ F

Pour le champ de tomates :

Soit L et l les dimensions du champ rectangulaire.

Diagonale : $L^2 + l^2 = 20^2 = 400$

Périmètre : $2(L + l) = 56 \Rightarrow L + l = 28$

On a : $(L + l)^2 = L^2 + 2Ll + l^2 = 28^2 = 784$

Donc $400 + 2Ll = 784 \Rightarrow 2Ll = 384 \Rightarrow Ll = 192$

Surface du champ : 192 m²

Coût des semences de tomates : $192 \times 600 = 115 200$ F

Comparaison : $337 459.20$ F > $115 200$ F

Le capital de 337 459,20 F est suffisant pour couvrir les frais de 115 200 F.
