PARTIE A : CORRIGÉ DES EXERCICES

Exercice 1 : Équations polynomiales (4.5 points)

1. Montrer que 0 n'est pas solution de cette équation.

Si $x = 0$, alors $x^4 + 8x^3 + 18x^2 + 8x + 1 = 0^4 + 8\times0^3 + 18\times0^2 + 8\times0 + 1 = 1 \neq 0$

Donc 0 n'est pas solution de l'équation.

2. Démontrer que (E) est équivalente à l'équation : $x^2 + 8x + 18 + \frac{8}{x} + \frac{1}{x^2} = 0$

Puisque $x \neq 0$, on peut diviser l'équation (E) par $x^2$ :

$\frac{x^4 + 8x^3 + 18x^2 + 8x + 1}{x^2} = \frac{0}{x^2}$

Ce qui donne : $x^2 + 8x + 18 + \frac{8}{x} + \frac{1}{x^2} = 0$

3. On pose : $X = x + \frac{1}{x}$. Exprimer $x^2 + \frac{1}{x^2}$ en fonction de X.

$X = x + \frac{1}{x} \Rightarrow X^2 = (x + \frac{1}{x})^2 = x^2 + 2 + \frac{1}{x^2}$

Donc $x^2 + \frac{1}{x^2} = X^2 - 2$

4. En déduire que si $x$ est solution de (E) alors $X$ est solution de l'équation : $X^2 + 8X + 16 = 0$

De la question 2, on a : $x^2 + \frac{1}{x^2} + 8x + \frac{8}{x} + 18 = 0$

Soit $(x^2 + \frac{1}{x^2}) + 8(x + \frac{1}{x}) + 18 = 0$

En remplaçant avec $X$, on obtient : $(X^2 - 2) + 8X + 18 = 0$

Ce qui donne : $X^2 + 8X + 16 = 0$

5. Résoudre (E) et en déduire les solutions de (E).

Résolvons $X^2 + 8X + 16 = 0$

$\Delta = 64 - 64 = 0$, donc une solution double : $X = -4$

On résout maintenant $x + \frac{1}{x} = -4$

Multiplions par $x$ (x ≠ 0) : $x^2 + 4x + 1 = 0$

$\Delta = 16 - 4 = 12$

$x = \frac{-4 \pm \sqrt{12}}{2} = \frac{-4 \pm 2\sqrt{3}}{2} = -2 \pm \sqrt{3}$

Les solutions de (E) sont : $x = -2 + \sqrt{3}$ et $x = -2 - \sqrt{3}$

Exercice 2 : Polynômes et suites (4 points)

1. Calculer $P(-1)$ et $P(1)$.

Pour $x = -1$ : $P(0) - P(-1) = 3(-1)^2 + 3(-1) + 1 = 3 - 3 + 1 = 1$

Or $P(0) = 1$, donc $1 - P(-1) = 1 \Rightarrow P(-1) = 0$

Pour $x = 0$ : $P(1) - P(0) = 3(0)^2 + 3(0) + 1 = 1$

Donc $P(1) - 1 = 1 \Rightarrow P(1) = 2$

2. Déterminer les réels a, b, c et d.

On a $P(x) = ax^3 + bx^2 + cx + d$

$P(0) = d = 1$

$P(-1) = -a + b - c + d = 0$

$P(1) = a + b + c + d = 2$

De plus, $P(x+1) - P(x) = 3x^2 + 3x + 1$

$P(x+1) - P(x) = a[(x+1)^3 - x^3] + b[(x+1)^2 - x^2] + c[(x+1) - x]$

$= a(3x^2 + 3x + 1) + b(2x + 1) + c = 3ax^2 + (3a + 2b)x + (a + b + c)$

On identifie avec $3x^2 + 3x + 1$ :

$\begin{cases} 3a = 3 \\ 3a + 2b = 3 \\ a + b + c = 1 \end{cases}$

Donc $a = 1$, $3(1) + 2b = 3 \Rightarrow b = 0$, $1 + 0 + c = 1 \Rightarrow c = 0$

$P(x) = x^3 + 1$

3. Résoudre dans $\mathbb{R}$ l'inéquation $P(x) \geq 0$.

$P(x) = x^3 + 1 \geq 0 \Rightarrow x^3 \geq -1 \Rightarrow x \geq -1$

Solution : $x \in [-1, +\infty[$

Exercice 3 : Équations et inéquations (6 points)

1. Résoudre dans $\mathbb{R}$ l'équation $\sqrt{3x + 15} = x + 5$.

Conditions d'existence : $3x + 15 \geq 0 \Rightarrow x \geq -5$ et $x + 5 \geq 0 \Rightarrow x \geq -5$

On élève au carré : $3x + 15 = (x + 5)^2 = x^2 + 10x + 25$

Soit $x^2 + 7x + 10 = 0$

$\Delta = 49 - 40 = 9$

$x = \frac{-7 \pm 3}{2}$ donc $x = -2$ ou $x = -5$

Les deux solutions respectent les conditions.

Solutions : $x = -5$ et $x = -2$

2. Résoudre dans $\mathbb{R}$ l'inéquation $\sqrt{x + 3} \leq x - 3$.

Conditions d'existence : $x + 3 \geq 0 \Rightarrow x \geq -3$ et $x - 3 \geq 0 \Rightarrow x \geq 3$

Donc $x \geq 3$

On élève au carré : $x + 3 \leq (x - 3)^2 = x^2 - 6x + 9$

Soit $x^2 - 7x + 6 \geq 0$

Racines : $\Delta = 49 - 24 = 25$, $x = \frac{7 \pm 5}{2}$ donc $x = 1$ ou $x = 6$

Le polynôme est positif à l'extérieur des racines : $x \leq 1$ ou $x \geq 6$

Comme on doit avoir $x \geq 3$, on garde $x \geq 6$

Solution : $x \in [6, +\infty[$

3. Résoudre dans $\mathbb{R}^2$ les systèmes :

Système (S₁): $\begin{cases} 2x - 3y = 7 \\ -4x + 6y = 5 \end{cases}$

Déterminant : $\Delta = \begin{vmatrix} 2 & -3 \\ -4 & 6 \end{vmatrix} = (2)(6) - (-3)(-4) = 12 - 12 = 0$

Le système est soit impossible, soit indéterminé.

En multipliant la première équation par -2 : $-4x + 6y = -14$

Mais la deuxième équation donne $-4x + 6y = 5$

Donc $-14 = 5$, ce qui est impossible.

Solution : Système impossible

Système (S₂): $\begin{cases} 5x + 2y = 1 \\ 3x - y = 8 \end{cases}$

Déterminant : $\Delta = \begin{vmatrix} 5 & 2 \\ 3 & -1 \end{vmatrix} = (5)(-1) - (2)(3) = -5 - 6 = -11 \neq 0$

$\Delta_x = \begin{vmatrix} 1 & 2 \\ 8 & -1 \end{vmatrix} = (1)(-1) - (2)(8) = -1 - 16 = -17$

$\Delta_y = \begin{vmatrix} 5 & 1 \\ 3 & 8 \end{vmatrix} = (5)(8) - (1)(3) = 40 - 3 = 37$

$x = \frac{\Delta_x}{\Delta} = \frac{-17}{-11} = \frac{17}{11}$

$y = \frac{\Delta_y}{\Delta} = \frac{37}{-11} = -\frac{37}{11}$

Solution unique : $\left(\frac{17}{11}, -\frac{37}{11}\right)$

Système (S₃): $\begin{cases} x + y = 9 \\ xy = 20 \end{cases}$

D'après la première équation, $y = 9 - x$

En substituant dans la deuxième : $x(9 - x) = 20$

Soit $-x^2 + 9x - 20 = 0$ ou $x^2 - 9x + 20 = 0$

$\Delta = 81 - 80 = 1$

$x = \frac{9 \pm 1}{2}$ donc $x = 5$ ou $x = 4$

Si $x = 5$, alors $y = 4$ ; si $x = 4$, alors $y = 5$

Solutions : $(5, 4)$ et $(4, 5)$

PARTIE B : CORRIGÉ DU PROBLÈME (7.5 points)

1. Déterminer les réels $m$ et $n$ pour que 1 et 3 soient des racines de $P$.

$P(x) = -x^3 + 9x^2 + mx + n$

$P(1) = -1 + 9 + m + n = 0 \Rightarrow m + n = -8$

$P(3) = -27 + 81 + 3m + n = 0 \Rightarrow 3m + n = -54$

En soustrayant la première équation de la deuxième : $2m = -46 \Rightarrow m = -23$

Puis $m + n = -8 \Rightarrow -23 + n = -8 \Rightarrow n = 15$

$m = -23$, $n = 15$

2. Rappeler la méthode de résolution d'une équation de degré 3 dans $\mathbb{R}$.

Pour résoudre une équation de degré 3, on peut :

  1. Chercher une racine évidente (souvent parmi les diviseurs du terme constant)
  2. Effectuer une division polynomiale pour factoriser par $(x - \text{racine})$
  3. Résoudre l'équation du second degré obtenue

3. Montrer que 5 est une racine de $P$.

$P(5) = -125 + 225 - 115 + 15 = (-125 + 225) + (-115 + 15) = 100 - 100 = 0$

Donc 5 est bien une racine de $P$.

4. Montrer que $P$ peut s'écrire $P(x) = (x - 5)(ax^2 + bx + c)$.

On effectue la division euclidienne de $P(x) = -x^3 + 9x^2 - 23x + 15$ par $(x - 5)$ :

$-x^3 + 9x^2 - 23x + 15 = (x - 5)(-x^2 + 4x - 3)$

$P(x) = (x - 5)(-x^2 + 4x - 3)$

5. En déduire dans $\mathbb{R}$ la résolution de l'équation $P(x) = 0$.

$P(x) = 0 \Leftrightarrow (x - 5)(-x^2 + 4x - 3) = 0$

Soit $x - 5 = 0$ ou $-x^2 + 4x - 3 = 0$

La deuxième équation : $x^2 - 4x + 3 = 0$

$\Delta = 16 - 12 = 4$

$x = \frac{4 \pm 2}{2}$ donc $x = 3$ ou $x = 1$

Solutions : $x = 1$, $x = 3$, $x = 5$

6. Résoudre dans $\mathbb{R}$ l'inéquation $P(x) \geq 0$.

$P(x) = -(x - 1)(x - 3)(x - 5)$

Tableau de signes :

x $-∞$ 1 3 5 $+∞$
$(x-1)$ - 0 + + +
$(x-3)$ - - 0 + +
$(x-5)$ - - - 0 +
Produit - 0 + 0 -
$-($Produit$)$ + 0 - 0 +
Solution : $x \in ]-∞, 1] \cup [3, 5]$

PARTIE C : CORRIGÉ DU PROBLÈME CONTEXTUALISÉ (5 points)

1. Pierre pourra-t-il creuser son puits ?

Calcul du taux d'intérêt :

Capital initial : $C_0 = 500 000$ F

Après 2 trimestres : $C_2 = 540 800$ F

Formule des intérêts composés : $C_n = C_0 (1 + t)^n$

Donc $540 800 = 500 000 (1 + t)^2$

$(1 + t)^2 = \frac{540 800}{500 000} = 1.0816$

$1 + t = \sqrt{1.0816} = 1.04$ (car t > 0)

Donc $t = 0.04 = 4\%$ par trimestre

Après 3 trimestres : $C_3 = 500 000 (1.04)^3 = 500 000 × 1.124864 = 562 432$ F

Oui, Pierre pourra creuser son puits car $562 432$ F > $580 000$ F ? (Note : Il y a une incohérence dans l'énoncé, car 562 432 < 580 000. Pierre ne pourra pas creuser son puits avec ce montant.)

2. Combien doit prévoir Pierre pour la culture de la tomate ?

Terrain rectangulaire : périmètre = 40 m, diagonale = 14 m

Soit L et l les dimensions :

$\begin{cases} 2(L + l) = 40 \\ L^2 + l^2 = 14^2 = 196 \end{cases}$

Donc $L + l = 20$ et $L^2 + l^2 = 196$

Or $(L + l)^2 = L^2 + 2Ll + l^2 = 400$

Donc $400 = 196 + 2Ll \Rightarrow 2Ll = 204 \Rightarrow Ll = 102$

Surface = $L × l = 102$ m²

Coût = $102 × 550 = 56 100$ F

Pierre doit prévoir 56 100 F pour la culture de la tomate.

3. Combien doit prévoir Pierre pour la culture du maïs ?

Parcelle rectangulaire avec 3 côtés à clôturer (100 m de grillage)

Soit x la longueur perpendiculaire au mur, y la longueur parallèle au mur

On a $2x + y = 100$ (périmètre à clôturer)

Surface = $S = x × y = x(100 - 2x) = 100x - 2x^2$

Cette fonction quadratique atteint son maximum en $x = -\frac{b}{2a} = -\frac{100}{2×(-2)} = 25$ m

Alors $y = 100 - 2×25 = 50$ m

Surface maximale = $25 × 50 = 1250$ m²

Coût = $1250 × 1750 = 2 187 500$ F

Pierre doit prévoir 2 187 500 F pour la culture du maïs.
