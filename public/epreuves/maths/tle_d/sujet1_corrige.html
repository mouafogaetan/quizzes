Exercice 1 /5 points

1. Démontrer que le polynôme P admet une racine imaginaire pure $z_0 = iy$, où $y$ est un nombre réel à déterminer.

Soit $z_0 = iy$ une racine imaginaire pure de $P(z) = z^3 + (2i - 4\sqrt{2})z^2 + 16z - 16i - 16\sqrt{2}$.

On a $P(iy) = (iy)^3 + (2i - 4\sqrt{2})(iy)^2 + 16(iy) - 16i - 16\sqrt{2}$

$= -iy^3 + (2i - 4\sqrt{2})(-y^2) + 16iy - 16i - 16\sqrt{2}$

$= -iy^3 -2iy^2 + 4\sqrt{2}y^2 + 16iy - 16i - 16\sqrt{2}$

$= (4\sqrt{2}y^2 - 16\sqrt{2}) + i(-y^3 - 2y^2 + 16y - 16)$

Pour que $P(iy) = 0$, il faut que les parties réelle et imaginaire soient nulles :

$\begin{cases} 4\sqrt{2}y^2 - 16\sqrt{2} = 0 \\ -y^3 - 2y^2 + 16y - 16 = 0 \end{cases}$

De la première équation : $4\sqrt{2}(y^2 - 4) = 0 \Rightarrow y^2 = 4 \Rightarrow y = 2$ ou $y = -2$

Testons $y = 2$ dans la seconde équation : $-8 - 8 + 32 - 16 = 0$ ✓

Testons $y = -2$ dans la seconde équation : $8 - 8 - 32 - 16 = -48 \neq 0$

Donc $z_0 = 2i$ est une racine imaginaire pure de P.

2. Déterminer les nombres $a$ et $b$ tels que $P(z) = (z - z_0)(z^2 + az + b)$.

On effectue la division polynomiale de $P(z)$ par $(z - 2i)$ :

$P(z) = (z - 2i)(z^2 + az + b) = z^3 + az^2 + bz - 2iz^2 - 2aiz - 2bi$

$= z^3 + (a - 2i)z^2 + (b - 2ai)z - 2bi$

On identifie avec $P(z) = z^3 + (2i - 4\sqrt{2})z^2 + 16z - 16i - 16\sqrt{2}$ :

$\begin{cases} a - 2i = 2i - 4\sqrt{2} \\ b - 2ai = 16 \\ -2bi = -16i - 16\sqrt{2} \end{cases}$

De la première équation : $a = 4i - 4\sqrt{2}$

De la troisième équation : $2bi = 16i + 16\sqrt{2} \Rightarrow bi = 8i + 8\sqrt{2} \Rightarrow b = 8 + \frac{8\sqrt{2}}{i} = 8 - 8\sqrt{2}i$

Vérifions avec la seconde équation : $b - 2ai = 8 - 8\sqrt{2}i - 2i(4i - 4\sqrt{2}) = 8 - 8\sqrt{2}i - 8i^2 + 8\sqrt{2}i = 8 + 8 = 16$ ✓

Donc $a = 4i - 4\sqrt{2}$ et $b = 8 - 8\sqrt{2}i$.

3. Déterminer les racines carrées de $-8 + 8i\sqrt{2}$.

On cherche $z = x + iy$ tel que $z^2 = -8 + 8i\sqrt{2}$.

On a $(x + iy)^2 = x^2 - y^2 + 2ixy = -8 + 8i\sqrt{2}$

Donc $\begin{cases} x^2 - y^2 = -8 \\ 2xy = 8\sqrt{2} \end{cases}$

De plus, $|z^2| = \sqrt{(-8)^2 + (8\sqrt{2})^2} = \sqrt{64 + 128} = \sqrt{192} = 8\sqrt{3}$

Donc $|z| = \sqrt{8\sqrt{3}} = 2\sqrt{2}\sqrt[4]{3}$

On peut aussi résoudre le système :

De la deuxième équation : $y = \frac{4\sqrt{2}}{x}$

En substituant dans la première : $x^2 - \frac{32}{x^2} = -8$

Multiplions par $x^2$ : $x^4 + 8x^2 - 32 = 0$

Posons $X = x^2$ : $X^2 + 8X - 32 = 0$

$\Delta = 64 + 128 = 192 = 64 \times 3$

$X = \frac{-8 \pm 8\sqrt{3}}{2} = -4 \pm 4\sqrt{3}$

Comme $X = x^2 \geq 0$, on prend $X = -4 + 4\sqrt{3}$

Donc $x = \pm \sqrt{4(\sqrt{3} - 1)} = \pm 2\sqrt{\sqrt{3} - 1}$

Et $y = \frac{4\sqrt{2}}{x}$

Les racines carrées sont $z = \pm \left(2\sqrt{\sqrt{3} - 1} + i\frac{2\sqrt{2}}{\sqrt{\sqrt{3} - 1}}\right)$

4. Résoudre dans $\mathbb{C}$ l'équation $(E')$ : $z^2 + (4i - 4\sqrt{2})z + 8 - 8i\sqrt{2} = 0$.

On reconnaît le facteur $z^2 + az + b$ de la question 2 avec $a = 4i - 4\sqrt{2}$ et $b = 8 - 8\sqrt{2}i$.

Calculons le discriminant : $\Delta = a^2 - 4b$

$\Delta = (4i - 4\sqrt{2})^2 - 4(8 - 8\sqrt{2}i) = 16i^2 - 32i\sqrt{2} + 32 - 32 + 32\sqrt{2}i = -16$

Les solutions sont : $z = \frac{-a \pm \sqrt{\Delta}}{2} = \frac{-4i + 4\sqrt{2} \pm 4i}{2}$

Donc $z_1 = \frac{-4i + 4\sqrt{2} + 4i}{2} = 2\sqrt{2}$

Et $z_2 = \frac{-4i + 4\sqrt{2} - 4i}{2} = \frac{4\sqrt{2} - 8i}{2} = 2\sqrt{2} - 4i$

Les solutions de $(E')$ sont $z_1 = 2\sqrt{2}$ et $z_2 = 2\sqrt{2} - 4i$.

5. En déduire l'ensemble solution de l'équation $P(z) = 0$.

D'après la factorisation de la question 2, on a :

$P(z) = (z - 2i)(z^2 + (4i - 4\sqrt{2})z + 8 - 8\sqrt{2}i)$

Les racines de $P(z)$ sont donc :

- $z_0 = 2i$ (racine imaginaire pure)

- Les racines de $z^2 + (4i - 4\sqrt{2})z + 8 - 8\sqrt{2}i = 0$

D'après la question 4, ces racines sont $2\sqrt{2}$ et $2\sqrt{2} - 4i$

Donc l'ensemble solution est $S = \{2i, 2\sqrt{2}, 2\sqrt{2} - 4i\}$.

Exercice 2 /5 points

1. Développer $(1 - \sqrt{3})^2$.

$(1 - \sqrt{3})^2 = 1^2 - 2 \times 1 \times \sqrt{3} + (\sqrt{3})^2 = 1 - 2\sqrt{3} + 3 = 4 - 2\sqrt{3}$

$(1 - \sqrt{3})^2 = 4 - 2\sqrt{3}$

2. Résoudre dans $\mathbb{C}$ l'équation $z^2 - (1 + \sqrt{3})z + \sqrt{3} = 0$.

Calculons le discriminant : $\Delta = (1 + \sqrt{3})^2 - 4\sqrt{3} = 1 + 2\sqrt{3} + 3 - 4\sqrt{3} = 4 - 2\sqrt{3}$

Les solutions sont : $z = \frac{1 + \sqrt{3} \pm \sqrt{4 - 2\sqrt{3}}}{2}$

On remarque que $4 - 2\sqrt{3} = (\sqrt{3} - 1)^2$ car $(\sqrt{3} - 1)^2 = 3 - 2\sqrt{3} + 1 = 4 - 2\sqrt{3}$

Donc $\sqrt{\Delta} = \sqrt{3} - 1$

Les solutions sont :

$z_1 = \frac{1 + \sqrt{3} + \sqrt{3} - 1}{2} = \frac{2\sqrt{3}}{2} = \sqrt{3}$

$z_2 = \frac{1 + \sqrt{3} - \sqrt{3} + 1}{2} = \frac{2}{2} = 1$

Les solutions sont $z_1 = \sqrt{3}$ et $z_2 = 1$.

3. Résoudre dans $\mathbb{C}$ les équations $z + \frac{1}{z} = 1$ et $z + \frac{1}{z} = \sqrt{3}$.

Première équation : $z + \frac{1}{z} = 1$

Multiplions par $z$ (avec $z \neq 0$) : $z^2 + 1 = z \Rightarrow z^2 - z + 1 = 0$

$\Delta = 1 - 4 = -3 = 3i^2$

Les solutions sont : $z = \frac{1 \pm i\sqrt{3}}{2}$

Deuxième équation : $z + \frac{1}{z} = \sqrt{3}$

Multiplions par $z$ (avec $z \neq 0$) : $z^2 + 1 = \sqrt{3}z \Rightarrow z^2 - \sqrt{3}z + 1 = 0$

$\Delta = 3 - 4 = -1 = i^2$

Les solutions sont : $z = \frac{\sqrt{3} \pm i}{2}$

Pour $z + \frac{1}{z} = 1$ : $S = \left\{\frac{1 + i\sqrt{3}}{2}, \frac{1 - i\sqrt{3}}{2}\right\}$

Pour $z + \frac{1}{z} = \sqrt{3}$ : $S = \left\{\frac{\sqrt{3} + i}{2}, \frac{\sqrt{3} - i}{2}\right\}$

4. Soit P le polynôme complexe défini par :

$$P(z) = z^4 - (1 + \sqrt{3})z^3 + (2 + \sqrt{3})z^2 - (1 + \sqrt{3})z + 1$$

a) Vérifier que pour tout $z$ non nul, on a :

$$\frac{P(z)}{z^2} = \left(z + \frac{1}{z}\right)^2 - (1 + \sqrt{3})\left(z + \frac{1}{z}\right) + \sqrt{3}$$

On a :

$\frac{P(z)}{z^2} = \frac{z^4 - (1 + \sqrt{3})z^3 + (2 + \sqrt{3})z^2 - (1 + \sqrt{3})z + 1}{z^2}$

$= z^2 - (1 + \sqrt{3})z + (2 + \sqrt{3}) - (1 + \sqrt{3})\frac{1}{z} + \frac{1}{z^2}$

$= \left(z^2 + \frac{1}{z^2}\right) - (1 + \sqrt{3})\left(z + \frac{1}{z}\right) + (2 + \sqrt{3})$

Or, $\left(z + \frac{1}{z}\right)^2 = z^2 + 2 + \frac{1}{z^2}$, donc $z^2 + \frac{1}{z^2} = \left(z + \frac{1}{z}\right)^2 - 2$

Ainsi :

$\frac{P(z)}{z^2} = \left(z + \frac{1}{z}\right)^2 - 2 - (1 + \sqrt{3})\left(z + \frac{1}{z}\right) + (2 + \sqrt{3})$

$= \left(z + \frac{1}{z}\right)^2 - (1 + \sqrt{3})\left(z + \frac{1}{z}\right) + \sqrt{3}$

L'égalité est vérifiée.

b) En déduire l'ensemble solution de l'équation $P(z) = 0$.

Pour $z \neq 0$, $P(z) = 0 \Leftrightarrow \frac{P(z)}{z^2} = 0$

D'après la question a), cela équivaut à :

$\left(z + \frac{1}{z}\right)^2 - (1 + \sqrt{3})\left(z + \frac{1}{z}\right) + \sqrt{3} = 0$

Posons $X = z + \frac{1}{z}$, alors on a $X^2 - (1 + \sqrt{3})X + \sqrt{3} = 0$

D'après la question 2, les solutions de cette équation sont $X = 1$ et $X = \sqrt{3}$

Donc on a deux cas :

Cas 1 : $z + \frac{1}{z} = 1$

D'après la question 3, les solutions sont $z = \frac{1 \pm i\sqrt{3}}{2}$

Cas 2 : $z + \frac{1}{z} = \sqrt{3}$

D'après la question 3, les solutions sont $z = \frac{\sqrt{3} \pm i}{2}$

L'ensemble solution est $S = \left\{\frac{1 + i\sqrt{3}}{2}, \frac{1 - i\sqrt{3}}{2}, \frac{\sqrt{3} + i}{2}, \frac{\sqrt{3} - i}{2}\right\}$

Exercice 3 /4 points

On considère la fonction $f$ définie par :

$$ f(x) = \begin{cases} \frac{\sqrt{3x+7}-\sqrt{6x-2}}{x-3} & \text{pour tout } x > 3 \\ \frac{x^3 - 6x^2 + 12x - 8}{9-x^2} & \text{pour tout } x < 3 \end{cases} $$

1. Justifier que l'ensemble de définition de $f$ est $\mathbb{R} \setminus [-3, 3]$.

Pour $x > 3$ :

- $\sqrt{3x+7}$ est définie si $3x+7 \geq 0 \Rightarrow x \geq -\frac{7}{3}$ (toujours vrai pour $x > 3$)

- $\sqrt{6x-2}$ est définie si $6x-2 \geq 0 \Rightarrow x \geq \frac{1}{3}$ (toujours vrai pour $x > 3$)

- Le dénominateur $x-3$ est non nul pour $x \neq 3$

Donc $f$ est définie sur $]3, +\infty[$

Pour $x < 3$ :

- Le dénominateur $9-x^2$ doit être non nul, donc $x \neq \pm 3$

- De plus, $9-x^2 > 0$ pour $x \in ]-3, 3[$, mais comme $x < 3$, on a $x \in ]-3, 3[$ pour que le dénominateur ne s'annule pas

Donc $f$ est définie sur $]-\infty, -3[ \cup ]-3, 3[$

L'ensemble de définition de $f$ est $\mathbb{R} \setminus [-3, 3]$.

2. Calculer les limites de $f$ en 3 par valeurs supérieures et par valeurs inférieures.

Limite à droite (x → 3⁺) :

$\lim\limits_{x \to 3^+} f(x) = \lim\limits_{x \to 3^+} \frac{\sqrt{3x+7}-\sqrt{6x-2}}{x-3}$

Forme indéterminée $\frac{0}{0}$. Multiplions par la quantité conjuguée :

$\frac{\sqrt{3x+7}-\sqrt{6x-2}}{x-3} \times \frac{\sqrt{3x+7}+\sqrt{6x-2}}{\sqrt{3x+7}+\sqrt{6x-2}} = \frac{(3x+7) - (6x-2)}{(x-3)(\sqrt{3x+7}+\sqrt{6x-2})}$

$= \frac{-3x+9}{(x-3)(\sqrt{3x+7}+\sqrt{6x-2})} = \frac{-3(x-3)}{(x-3)(\sqrt{3x+7}+\sqrt{6x-2})} = \frac{-3}{\sqrt{3x+7}+\sqrt{6x-2}}$

Donc $\lim\limits_{x \to 3^+} f(x) = \frac{-3}{\sqrt{16}+\sqrt{16}} = \frac{-3}{4+4} = -\frac{3}{8}$

Limite à gauche (x → 3⁻) :

$\lim\limits_{x \to 3^-} f(x) = \lim\limits_{x \to 3^-} \frac{x^3 - 6x^2 + 12x - 8}{9-x^2}$

Forme indéterminée $\frac{0}{0}$. Factorisons numérateur et dénominateur :

Numérateur : $x^3 - 6x^2 + 12x - 8 = (x-2)^3$

Dénominateur : $9-x^2 = (3-x)(3+x)$

Donc $f(x) = \frac{(x-2)^3}{(3-x)(3+x)} = -\frac{(x-2)^3}{(x-3)(x+3)}$

$\lim\limits_{x \to 3^-} f(x) = -\frac{(3-2)^3}{(3-3)(3+3)} = -\frac{1}{0^- \times 6} = +\infty$

$\lim\limits_{x \to 3^+} f(x) = -\frac{3}{8}$ et $\lim\limits_{x \to 3^-} f(x) = +\infty$

3. Dire en justifiant si l'on peut prolonger par continuité en 3.

Pour qu'un prolongement par continuité en 3 soit possible, il faudrait que les limites à gauche et à droite soient finies et égales.

Or, $\lim\limits_{x \to 3^+} f(x) = -\frac{3}{8}$ (finie) mais $\lim\limits_{x \to 3^-} f(x) = +\infty$ (infinie)

Donc on ne peut pas prolonger $f$ par continuité en 3.

4. Si oui, préciser ce prolongement.

Comme on ne peut pas prolonger $f$ par continuité en 3, cette question ne s'applique pas.

Exercice 4 /6 points

1. $\lim\limits_{x \to 4} \frac{\sqrt{x-4}}{5 - \sqrt{7x-4}}$

Forme indéterminée $\frac{0}{0}$. Multiplions par la quantité conjuguée du dénominateur :

$\frac{\sqrt{x-4}}{5 - \sqrt{7x-4}} \times \frac{5 + \sqrt{7x-4}}{5 + \sqrt{7x-4}} = \frac{\sqrt{x-4}(5 + \sqrt{7x-4})}{25 - (7x-4)} = \frac{\sqrt{x-4}(5 + \sqrt{7x-4})}{29 - 7x}$

$= \frac{\sqrt{x-4}(5 + \sqrt{7x-4})}{-7(x-4) + 1} = \frac{\sqrt{x-4}(5 + \sqrt{7x-4})}{-7(x-4) + 1}$

Quand $x \to 4$, $\sqrt{x-4} \to 0$ et $5 + \sqrt{7x-4} \to 5 + \sqrt{24} = 5 + 2\sqrt{6}$

Donc la limite est $\frac{0 \times (5 + 2\sqrt{6})}{1} = 0$

La limite est 0.

2. $\lim\limits_{x \to \frac{\pi}{3}} \frac{\sin(3x-\pi)}{6x-2\pi}$

Posons $t = 3x - \pi$, alors quand $x \to \frac{\pi}{3}$, $t \to 0$

Et $6x - 2\pi = 2(3x - \pi) = 2t$

Donc $\lim\limits_{x \to \frac{\pi}{3}} \frac{\sin(3x-\pi)}{6x-2\pi} = \lim\limits_{t \to 0} \frac{\sin t}{2t} = \frac{1}{2} \lim\limits_{t \to 0} \frac{\sin t}{t} = \frac{1}{2} \times 1 = \frac{1}{2}$

La limite est $\frac{1}{2}$.

3. $\lim\limits_{x \to -\infty} \left( \sqrt{3x^2 + 7 + 4x} \right)$

Quand $x \to -\infty$, $4x \to -\infty$ et $\sqrt{3x^2 + 7} \sim \sqrt{3}|x| = -\sqrt{3}x$ (car $x < 0$)

Donc $\sqrt{3x^2 + 7 + 4x} \sim -\sqrt{3}x + 4x = x(4 - \sqrt{3}) \to -\infty$

Mais attention, c'est une forme indéterminée $\infty - \infty$.

Multiplions par la quantité conjuguée :

$\sqrt{3x^2 + 7 + 4x} = \frac{(3x^2 + 7) - (4x)^2}{\sqrt{3x^2 + 7 - 4x}} = \frac{3x^2 + 7 - 16x^2}{\sqrt{3x^2 + 7 - 4x}} = \frac{-13x^2 + 7}{\sqrt{3x^2 + 7 - 4x}}$

Quand $x \to -\infty$, le numérateur $\sim -13x^2$ et le dénominateur $\sim \sqrt{3}|x| - 4x = -\sqrt{3}x - 4x = -x(\sqrt{3} + 4)$

Donc l'expression $\sim \frac{-13x^2}{-x(\sqrt{3} + 4)} = \frac{13x}{\sqrt{3} + 4} \to -\infty$

La limite est $-\infty$.

4. $\lim\limits_{x \to +\infty} \frac{\sqrt{x+9}-3}{x-4}$

Quand $x \to +\infty$, numérateur $\sim \sqrt{x}$ et dénominateur $\sim x$

Donc $\frac{\sqrt{x+9}-3}{x-4} \sim \frac{\sqrt{x}}{x} = \frac{1}{\sqrt{x}} \to 0$

La limite est 0.

5. $\lim\limits_{x \to 0} \frac{\tan(4x)}{\sin(5x)}$

$\frac{\tan(4x)}{\sin(5x)} = \frac{\sin(4x)}{\cos(4x)\sin(5x)} = \frac{4x}{5x} \times \frac{\frac{\sin(4x)}{4x}}{\cos(4x)\frac{\sin(5x)}{5x}} \to \frac{4}{5} \times \frac{1}{1 \times 1} = \frac{4}{5}$

La limite est $\frac{4}{5}$.

6. $\lim\limits_{x \to 2\pi} \frac{\sin(x)}{x-2\pi}$

Posons $t = x - 2\pi$, alors quand $x \to 2\pi$, $t \to 0$

$\frac{\sin(x)}{x-2\pi} = \frac{\sin(t + 2\pi)}{t} = \frac{\sin t}{t} \to 1$

La limite est 1.
