PARTIE A : ÉVALUATION DES RESSOURCES

EXERCICE 1 (5 points)

1. Montrer que : $(1 + 2\sqrt{3})^2 = 13 + 4\sqrt{3}$ (0,5 pt)

Méthode : Développement d'une identité remarquable

$(1 + 2\sqrt{3})^2 = 1^2 + 2 \times 1 \times 2\sqrt{3} + (2\sqrt{3})^2$
$= 1 + 4\sqrt{3} + 4 \times 3$
$= 1 + 4\sqrt{3} + 12$
$= 13 + 4\sqrt{3}$
Ainsi, $(1 + 2\sqrt{3})^2 = 13 + 4\sqrt{3}$

2. Montrer que $P(x)$ admet deux racines distinctes et préciser leurs signes. (1,5 pt)

Méthode : Calcul du discriminant et analyse des racines

$P(x) = 2x^2 - 2\sqrt{3}x - (1 + \sqrt{3})$
Discriminant : $\Delta = (-2\sqrt{3})^2 - 4 \times 2 \times [-(1 + \sqrt{3})]$
$\Delta = 12 + 8(1 + \sqrt{3}) = 12 + 8 + 8\sqrt{3} = 20 + 8\sqrt{3}$
$\Delta > 0$ donc deux racines réelles distinctes :
$x_1 = \frac{2\sqrt{3} - \sqrt{20 + 8\sqrt{3}}}{4}$
$x_2 = \frac{2\sqrt{3} + \sqrt{20 + 8\sqrt{3}}}{4}$
Produit des racines : $\frac{c}{a} = \frac{-(1 + \sqrt{3})}{2} < 0$
Les deux racines sont de signes opposés.

3. Résoudre dans $\mathbb{R}$ l'équation : $2x^2 - 1 - 2\sqrt{3}x - \sqrt{3} = 0$ (1,5 pt)

Méthode : Utilisation de la forme canonique

$2x^2 - 2\sqrt{3}x - (1 + \sqrt{3}) = 0$
On divise par 2 : $x^2 - \sqrt{3}x - \frac{1 + \sqrt{3}}{2} = 0$
Discriminant : $\Delta = (-\sqrt{3})^2 - 4 \times 1 \times [-\frac{1 + \sqrt{3}}{2}] = 3 + 2(1 + \sqrt{3}) = 5 + 2\sqrt{3}$
Solutions : $x = \frac{\sqrt{3} \pm \sqrt{5 + 2\sqrt{3}}}{2}$
$S = \left\{ \frac{\sqrt{3} - \sqrt{5 + 2\sqrt{3}}}{2}, \frac{\sqrt{3} + \sqrt{5 + 2\sqrt{3}}}{2} \right\}$

4. En déduire l'ensemble des solutions de l'inéquation : $2x^2 - 1 - 2\sqrt{3}x - \sqrt{3} < 0$ (1,5 pt)

Méthode : Étude du signe d'un polynôme du second degré

Le coefficient de $x^2$ est positif (a = 2 > 0), donc le polynôme est négatif entre les racines.
Soit $r_1$ et $r_2$ les racines avec $r_1 < r_2$
$S = \left] \frac{\sqrt{3} - \sqrt{5 + 2\sqrt{3}}}{2}, \frac{\sqrt{3} + \sqrt{5 + 2\sqrt{3}}}{2} \right[$

EXERCICE 2 (4 points)

1. Résoudre dans $\mathbb{R}$ l'inéquation : $\sqrt{3x + 4} \geq x$ (2 pts)

Méthode : Inéquation avec racine carrée - Analyse de domaine et cas

Condition d'existence : $3x + 4 \geq 0 \Rightarrow x \geq -\frac{4}{3}$
Cas 1 : Si $x < 0$, alors $\sqrt{3x+4} \geq 0 \geq x$ toujours vrai
Cas 2 : Si $x \geq 0$, on élève au carré : $3x + 4 \geq x^2$
$x^2 - 3x - 4 \leq 0$
Racines : $x = -1$ et $x = 4$
Solution dans [0, +∞[ : $x \in [0, 4]$
Solution finale : $S = \left[-\frac{4}{3}, 4\right]$

2. Résoudre dans $\mathbb{R}^2$ le système : $\begin{cases} 4x + 4y = -4 \\ -3xy = 6 \end{cases}$ (2 pts)

Méthode : Substitution après simplification

Simplification : $\begin{cases} x + y = -1 \\ xy = -2 \end{cases}$
x et y sont les racines de $t^2 - St + P = 0$ où S = -1, P = -2
Équation : $t^2 + t - 2 = 0$
Discriminant : $\Delta = 1 + 8 = 9$
Racines : $t = \frac{-1 \pm 3}{2}$
Soit $t = 1$ ou $t = -2$
Les solutions sont $(1, -2)$ et $(-2, 1)$

EXERCICE 3 (3,5 points)

1. La solution du système $(S)$ : $\begin{cases} x + 2y + z = 2 \\ 3x - 4y - 2z = 1 \\ x + 5y - 2z = 8 \end{cases}$ est :

a) $(1; 1; -1)$    b) $(-1; 1; 1)$    c) $(1; -1; 1)$ (1,5 pt)

Méthode : Vérification des propositions

Test de (1, 1, -1) : L1: 1 + 2×1 + (-1) = 2 ✓ L2: 3×1 - 4×1 - 2×(-1) = 3 - 4 + 2 = 1 ✓ L3: 1 + 5×1 - 2×(-1) = 1 + 5 + 2 = 8 ✓
Test de (-1, 1, 1) : L1: -1 + 2×1 + 1 = 2 ✓ L2: 3×(-1) - 4×1 - 2×1 = -3 - 4 - 2 = -9 ≠ 1 ✗
Test de (1, -1, 1) : L1: 1 + 2×(-1) + 1 = 0 ≠ 2 ✗
La bonne réponse est a) $(1; 1; -1)$

2. En déduire l'ensemble solution du système :

$\begin{cases} (x-1) + 2(y-3) + (z+1) = 2 \\ 3(x-1) - 4(y-3) - 2(z+1) = 1 \\ (x-1) + 5(y-3) - 2(z+1) = 8 \end{cases}$ (2 pts)

Méthode : Changement de variable

Posons $X = x - 1$, $Y = y - 3$, $Z = z + 1$
Le système devient : $\begin{cases} X + 2Y + Z = 2 \\ 3X - 4Y - 2Z = 1 \\ X + 5Y - 2Z = 8 \end{cases}$
C'est exactement le système (S) dont la solution est $X = 1$, $Y = 1$, $Z = -1$
Donc : $x - 1 = 1 \Rightarrow x = 2$
$y - 3 = 1 \Rightarrow y = 4$
$z + 1 = -1 \Rightarrow z = -2$
La solution est $(2; 4; -2)$
PARTIE B : ÉVALUATION DES COMPÉTENCES

SOLUTION DE LA SITUATION PROBLÈME

1. Déterminer l'intervalle auquel doit appartenir $x$ pour que l'entreprise réalise un bénéfice. (2,25 pts)

Méthode : Modélisation du bénéfice

Coût de production : $C(x) = x^2 + 1,000,000$
Recette : $R(x) = 2500x$
Bénéfice : $B(x) = R(x) - C(x) = 2500x - x^2 - 1,000,000$
On veut $B(x) > 0$ : $-x^2 + 2500x - 1,000,000 > 0$
Soit $x^2 - 2500x + 1,000,000 < 0$
Discriminant : $\Delta = 6,250,000 - 4,000,000 = 2,250,000$
Racines : $x = \frac{2500 \pm 1500}{2}$ → $x = 500$ et $x = 2000$
L'entreprise réalise un bénéfice pour $x \in ]500; 2000[$

2. Déterminer le nombre d'enfants de Monsieur KOFFI. (2,25 pts)

Méthode : Mise en équation du problème

Soit $n$ le nombre d'enfants de M. KOFFI
Nombre total d'enfants présents : $n + 4$
Part initiale : $\frac{50}{n}$
Part réelle : $\frac{50}{n+4}$
Équation : $\frac{50}{n} - \frac{50}{n+4} = 2$
$\frac{50(n+4) - 50n}{n(n+4)} = 2$ → $\frac{200}{n(n+4)} = 2$
$n(n+4) = 100$ → $n^2 + 4n - 100 = 0$
Discriminant : $\Delta = 16 + 400 = 416$
$n = \frac{-4 \pm \sqrt{416}}{2} = \frac{-4 \pm 4\sqrt{26}}{2} = -2 \pm 2\sqrt{26}$
Seule la solution positive est acceptable : $n = -2 + 2\sqrt{26} \approx -2 + 10.2 = 8.2$
M. KOFFI a 8 enfants (valeur entière la plus proche)

3. Déterminer la valeur du taux d'augmentation $t$. (2,25 pts)

Méthode : Calcul de taux d'évolution successif

Prix initial : 2500 FCFA
Prix final : 3025 FCFA
Deux augmentations successives de taux $t$ : $2500 \times (1+t)^2 = 3025$
$(1+t)^2 = \frac{3025}{2500} = 1.21$
$1+t = \sqrt{1.21} = 1.1$ (car $t > 0$)
$t = 0.1 = 10\%$
Le taux d'augmentation $t$ est de 10%
