EXERCICE 1 (5,25 points)

1. Montrer que $(\sqrt{3} + \sqrt{2})^2 = 5 + 2\sqrt{6}$.

$(\sqrt{3} + \sqrt{2})^2 = (\sqrt{3})^2 + 2 \cdot \sqrt{3} \cdot \sqrt{2} + (\sqrt{2})^2 = 3 + 2\sqrt{6} + 2 = 5 + 2\sqrt{6}$

2. a) Résoudre dans $\mathbb{R}$ l'équation $(E) : 3x^2 - (\sqrt{3} - \sqrt{2})x - \frac{\sqrt{6}}{3} = 0$.

Calcul du discriminant :
$\Delta = (\sqrt{3} - \sqrt{2})^2 - 4 \cdot 3 \cdot (-\frac{\sqrt{6}}{3}) = (3 - 2\sqrt{6} + 2) + 4\sqrt{6} = 5 + 2\sqrt{6}$
Les solutions sont :
$x_1 = \frac{(\sqrt{3} - \sqrt{2}) - \sqrt{5 + 2\sqrt{6}}}{6}$ et $x_2 = \frac{(\sqrt{3} - \sqrt{2}) + \sqrt{5 + 2\sqrt{6}}}{6}$

b) En déduire la résolution de $(I) : 3x^2 - (\sqrt{3} - \sqrt{2})x - \frac{\sqrt{6}}{3} < 0$.

Le polynôme est du signe de $-a$ à l'intérieur des racines et de $a$ à l'extérieur.
Comme $a = 3 > 0$, le polynôme est négatif entre les racines :
$S = \left]\frac{(\sqrt{3} - \sqrt{2}) - \sqrt{5 + 2\sqrt{6}}}{6}, \frac{(\sqrt{3} - \sqrt{2}) + \sqrt{5 + 2\sqrt{6}}}{6}\right[$

3. Résoudre dans $\mathbb{R}$ :

$(E_1) : \sqrt{3x^2 - 2x + 2} + 4 - 3x = 2 - x$

Conditions d'existence : $3x^2 - 2x + 2 \geq 0$ (toujours vrai car discriminant négatif)
Isoler la racine : $\sqrt{3x^2 - 2x + 2} = -2 - 2x$
La racine carrée est toujours positive, donc $-2 - 2x \geq 0 \Rightarrow x \leq -1$
Élever au carré : $3x^2 - 2x + 2 = 4 + 8x + 4x^2$
Simplifier : $-x^2 - 10x - 2 = 0 \Rightarrow x^2 + 10x + 2 = 0$
Solutions : $x = -5 \pm \sqrt{23}$
Seule $x = -5 - \sqrt{23}$ vérifie $x \leq -1$
$S = \{-5 - \sqrt{23}\}$

$(E_2) : x^4 - 7x^2 + 12 = 0$

Poser $X = x^2$ : $X^2 - 7X + 12 = 0$
Solutions : $X = 3$ ou $X = 4$
Donc $x = \pm\sqrt{3}$ ou $x = \pm 2$
$S = \{-\sqrt{3}, \sqrt{3}, -2, 2\}$

$(I_1) : \sqrt{3x - 7} < -3x + 8$

Conditions d'existence : $3x - 7 \geq 0 \Rightarrow x \geq \frac{7}{3}$
Le côté droit doit être positif : $-3x + 8 > 0 \Rightarrow x < \frac{8}{3}$
Donc $x \in \left[\frac{7}{3}, \frac{8}{3}\right[$
Élever au carré : $3x - 7 < 9x^2 - 48x + 64$
Simplifier : $0 < 9x^2 - 51x + 71$
Le discriminant de $9x^2 - 51x + 71$ est négatif, donc toujours positif
Solution : $S = \left[\frac{7}{3}, \frac{8}{3}\right[$

$(E_3) : \sqrt{4x + 5} = 0$

$\sqrt{4x + 5} = 0 \Rightarrow 4x + 5 = 0 \Rightarrow x = -\frac{5}{4}$
$S = \{-\frac{5}{4}\}$
EXERCICE 2 (4,75 points)

1. Montrer que 1 est une racine du polynôme $P$.

$P(1) = -3(1)^3 + 6(1)^2 + 15(1) - 18 = -3 + 6 + 15 - 18 = 0$
Donc 1 est bien une racine de $P$.

2. Déterminer $a$, $b$ et $c$ tels que $P(x) = (x-1)(ax^2 + bx + c)$.

On effectue la division polynomiale ou on développe :
$(x-1)(ax^2 + bx + c) = ax^3 + bx^2 + cx - ax^2 - bx - c = ax^3 + (b-a)x^2 + (c-b)x - c$
On identifie avec $P(x) = -3x^3 + 6x^2 + 15x - 18$ :
$a = -3$, $b - a = 6 \Rightarrow b = 3$, $c - b = 15 \Rightarrow c = 18$, $-c = -18$
Donc $P(x) = (x-1)(-3x^2 + 3x + 18)$

3. Résoudre $-3x^2 + 3x + 18 = 0$ et en déduire une factorisation de $P(x)$.

$-3x^2 + 3x + 18 = 0$
Diviser par -3 : $x^2 - x - 6 = 0$
Discriminant : $\Delta = 1 + 24 = 25$
Solutions : $x = \frac{1 \pm 5}{2}$ donc $x = 3$ ou $x = -2$
Factorisation : $-3x^2 + 3x + 18 = -3(x-3)(x+2)$
Donc $P(x) = (x-1)(-3)(x-3)(x+2) = -3(x-1)(x-3)(x+2)$

4. Résoudre $P(x) = 0$, puis :

a) $P(x) \leq 0$

$P(x) = 0 \Rightarrow x = -2, 1, 3$
Tableau de signes :
$P(x) = -3(x+2)(x-1)(x-3)$
Sur $]-\infty, -2[$ : négatif
Sur $]-2, 1[$ : positif
Sur $]1, 3[$ : négatif
Sur $]3, +\infty[$ : positif
Donc $P(x) \leq 0$ pour $x \in ]-\infty, -2] \cup [1, 3]$

b) $P(x) > 0$

D'après le tableau de signes :
$P(x) > 0$ pour $x \in ]-2, 1[ \cup ]3, +\infty[$

5. Sans effectuer de calculs, comparer $P(-5)$ et $P(2022)$.

Le polynôme est de degré 3 avec coefficient dominant négatif (-3).
Donc $\lim_{x \to -\infty} P(x) = +\infty$ et $\lim_{x \to +\infty} P(x) = -\infty$.
Ainsi, $P(-5) > 0$ et $P(2022) < 0$, donc $P(-5) > P(2022)$.
EXERCICE 3 (5,5 points)

1. Résoudre dans $\mathbb{R}$ l'équation : $-2x - 3\sqrt{x + 20} = 0$.

Isoler la racine : $3\sqrt{x + 20} = -2x$
Condition : $-2x \geq 0 \Rightarrow x \leq 0$
Élever au carré : $9(x + 20) = 4x^2$
$4x^2 - 9x - 180 = 0$
Discriminant : $\Delta = 81 + 2880 = 2961$
$x = \frac{9 \pm \sqrt{2961}}{8}$
Seule $x = \frac{9 - \sqrt{2961}}{8}$ (négative) vérifie $x \leq 0$
$S = \left\{\frac{9 - \sqrt{2961}}{8}\right\}$

2. On donne $Q(x) = -4x^2 + (3 - 4\sqrt{2})x + 3\sqrt{2}$.

a) Montrer que $Q$ a exactement deux racines distinctes.

Calcul du discriminant :
$\Delta = (3 - 4\sqrt{2})^2 - 4 \cdot (-4) \cdot 3\sqrt{2} = 9 - 24\sqrt{2} + 32 + 48\sqrt{2} = 41 + 24\sqrt{2} > 0$
Donc deux racines réelles distinctes.

b) Déterminer $x_1 + x_2$ et $x_1x_2$.

Relations de Viète :
$x_1 + x_2 = \frac{3 - 4\sqrt{2}}{-4} = \frac{4\sqrt{2} - 3}{4}$
$x_1x_2 = \frac{3\sqrt{2}}{-4} = -\frac{3\sqrt{2}}{4}$

c) Montrer que $-\sqrt{2}$ est une racine de $Q$ et en déduire l'autre racine.

$Q(-\sqrt{2}) = -4(2) + (3 - 4\sqrt{2})(-\sqrt{2}) + 3\sqrt{2} = -8 -3\sqrt{2} + 8 + 3\sqrt{2} = 0$
Donc $-\sqrt{2}$ est bien une racine.
D'après les relations de Viète : $x_1 + x_2 = \frac{4\sqrt{2} - 3}{4}$
Si $x_1 = -\sqrt{2}$, alors $x_2 = \frac{4\sqrt{2} - 3}{4} + \sqrt{2} = \frac{4\sqrt{2} - 3 + 4\sqrt{2}}{4} = \frac{8\sqrt{2} - 3}{4}$

3. Résoudre les systèmes :

$(S_1) : \begin{cases} x^2 + y^2 = 130 \\ xy = 27 \\ x < y \end{cases}$

On a $(x+y)^2 = x^2 + y^2 + 2xy = 130 + 54 = 184$
Donc $x+y = \pm\sqrt{184} = \pm 2\sqrt{46}$
$(x-y)^2 = x^2 + y^2 - 2xy = 130 - 54 = 76$
Donc $x-y = \pm\sqrt{76} = \pm 2\sqrt{19}$
Comme $x < y$, $x-y$ est négatif
On résout les systèmes :
Cas 1: $x+y = 2\sqrt{46}$ et $x-y = -2\sqrt{19}$
$x = \sqrt{46} - \sqrt{19}$, $y = \sqrt{46} + \sqrt{19}$
Cas 2: $x+y = -2\sqrt{46}$ et $x-y = 2\sqrt{19}$
$x = -\sqrt{46} + \sqrt{19}$, $y = -\sqrt{46} - \sqrt{19}$
Mais $x < y$ dans les deux cas
$S = \left\{(\sqrt{46} - \sqrt{19}, \sqrt{46} + \sqrt{19}), (-\sqrt{46} + \sqrt{19}, -\sqrt{46} - \sqrt{19})\right\}$

$(S_2) : \begin{cases} 3x + 4y + z = 5 \\ -2x + 3y + 5z = 9 \\ 4x + 6y - z = -3 \end{cases}$

Méthode de Gauss ou combinaisons linéaires :
L1 + L3 : $7x + 10y = 2$
5×L1 + L2 : $13x + 23y = 34$
On résout le système $2\times2$ :
$7x + 10y = 2$ et $13x + 23y = 34$
Solution : $x = -6$, $y = 4.4$
Puis $z = 5 - 3(-6) - 4(4.4) = 5 + 18 - 17.6 = 5.4$
$S = \{(-6, 4.4, 5.4)\}$

PARTIE B : ÉVALUATION DES COMPÉTENCES (4,5 points)

1. Déterminer les dimensions du terrain de M. KANDÉ.

Soit $L$ la longueur et $l$ la largeur.
On a : $L \times l = 525$ et $2(L + l) = 92 \Rightarrow L + l = 46$
$L$ et $l$ sont solutions de $X^2 - 46X + 525 = 0$
Discriminant : $\Delta = 2116 - 2100 = 16$
Solutions : $X = \frac{46 \pm 4}{2} = 25$ ou $21$
Donc les dimensions sont 25m et 21m.

2. Déterminer la valeur de $x$ pour laquelle le salaire est de $60000$ FCFA.

$f(x) = -x^2 + 5x + 6$ (en dizaines de milliers de FCFA)
On veut $f(x) = 6$ (car 60000 FCFA = 6 × 10000 FCFA)
$-x^2 + 5x + 6 = 6 \Rightarrow -x^2 + 5x = 0 \Rightarrow x(-x + 5) = 0$
Solutions : $x = 0$ ou $x = 5$
Mais $x$ est choisi entre 0 et 5, donc les deux valeurs sont possibles.

3. Déterminer le prix de chaque article.

Soit $g$ le prix d'une guirlande et $b$ le prix d'une boule.
Système d'équations :
$8g + 7b = 6500$
$10g + 6b = 6800$
Multiplier la première équation par 3 et la seconde par 3.5 :
$24g + 21b = 19500$
$35g + 21b = 23800$
Soustraire : $11g = 4300 \Rightarrow g = 390.91$ FCFA
Remplacer : $8(390.91) + 7b = 6500 \Rightarrow 3127.28 + 7b = 6500 \Rightarrow 7b = 3372.72 \Rightarrow b = 481.82$ FCFA
Donc une guirlande coûte environ 391 FCFA et une boule environ 482 FCFA.
