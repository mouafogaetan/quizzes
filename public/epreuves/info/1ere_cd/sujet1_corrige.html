Structures de données et algorithmes

PARTIE 1 : ÉVALUATION DES RESSOURCES (11 Points)

1) Définir les termes suivants : (2 points)

a) Type abstrait de données :

Un type abstrait de données (TAD) est un modèle mathématique qui définit un ensemble de données et les opérations que l'on peut effectuer sur ces données, sans spécifier leur implémentation concrète.

b) Interpréteur :

Un interpréteur est un programme qui analyse et exécute directement le code source d'un programme sans étape préalable de compilation. Il traduit et exécute les instructions ligne par ligne.

2) (3 points)

a) Différence entre Répéter...Jusqu'à et Tant que...Faire :

La boucle Répéter...Jusqu'à exécute le bloc d'instructions au moins une fois, puis teste la condition de sortie. La boucle Tant que...Faire teste d'abord la condition d'entrée, et n'exécute le bloc que si cette condition est vraie.

b) Quatre structures de données fondamentales :

1. Tableaux (Arrays)
2. Listes chaînées (Linked Lists)
3. Piles (Stacks)
4. Files (Queues)

c) Syntaxe de la boucle for en langage C :

for (initialisation; condition; incrémentation) { // instructions }

3) Calcul de la moyenne des notes (1.5 points)

a) Structure de données appropriée :

Un tableau (array) serait la structure de données la plus appropriée.

b) Déclaration de la structure Notes :

// En algorithmique Variable Notes : Tableau[1..40] de Réel // En langage C float Notes[40];

4) Algorithme de saisie et calcul de la somme (2.5 points)

Algorithme CalculSommeNotes Variables i, somme: Entier Notes: Tableau[1..40] de Réel Début somme ← 0 Pour i de 1 à 40 faire Ecrire("Entrez la note de l'élève ", i) Lire(Notes[i]) somme ← somme + Notes[i] FinPour Ecrire("La somme des notes est : ", somme) Fin

5) Traduction en langage C (2 points)

#include int main() { float Notes[40]; float somme = 0; int i; for (i = 0; i < 40; i++) { printf("Entrez la note de l'élève %d : ", i+1); scanf("%f", &Notes[i]); somme += Notes[i]; } printf("La somme des notes est : %.2f\n", somme); return 0; }

PARTIE 2 : ÉVALUATION DES COMPÉTENCES (07 points)

1) Analyse de l'algorithme (2.5 points)

a) Condition de continuation de la boucle :

i ≤ n

b) Variable de résultat :

s (contient le résultat du calcul)

c) Instruction d'initialisation :

i ← 1
s ← 0

d) Instruction de modification :

i ← i + 1 (incrémentation du compteur)

2) Valeurs de i et s si n = 0 (1 point)

i = 1, s = 0
La condition (i ≤ n) devient (1 ≤ 0) qui est fausse, donc la boucle ne s'exécute pas.

3) Résultat si n = 4 (1 point)

Le résultat est : 10
L'algorithme calcule la somme des entiers de 1 à 4 : 1+2+3+4 = 10

4) Fonction mathématique calculée (0.5 point)

L'algorithme calcule la somme des n premiers entiers naturels : $S = \sum_{i=1}^{n} i = \frac{n(n+1)}{2}$

5) Réécriture avec une boucle Pour (2 points)

Algorithme Suite Var i, n, s : Entier Début Ecrire("Entrez un entier positif") Lire(n) s ← 0 Pour i de 1 à n faire s ← s + i FinPour Ecrire("Le résultat est :", s) Fin

La qualité d'un programme se mesure non seulement à son exactitude, mais aussi à sa clarté.
