Corrigé — Contrôle de Chimie Acides/Bases

Niveau : Terminale Scientifique • Durée : 2h30 • Coefficient : 2

Corrigé officiel
Exercice 1 : Vérification des prérequis

1. Définitions :

  • Acide : espèce chimique capable de céder un proton $H^+$ (selon Brønsted).
  • Base : espèce chimique capable de capter un proton $H^+$.
  • Ampholyte : espèce chimique pouvant se comporter à la fois comme acide et comme base (ex : $HCO_3^-$).
  • Indicateur coloré : molécule organique dont la couleur dépend du pH (ex : phénolphtaléine).
  • Teinte sensible : domaine de pH dans lequel la couleur d’un indicateur change perceptiblement.

2. pH d’un acide fort : $pH = -\log C$ ; pH d’une base forte : $pH = 14 + \log C$.

3. QCM :

  • (a) Réponse : C ($9,6\times 10^{-14}$).
  • (b) Réponse : D ($3,6\times10^{-7}\,%$).
  • (c) Réponse : A (pH proche de 2).
Exercice 2 : Acide monochloroéthanoïque

Données : $pK_a=2,9$, pH mesuré = 1,0.

1. $[H_3O^+] = 10^{-1}=0,10\,\mathrm{mol.L^{-1}}$ ; $[OH^-] = 10^{-13}\,\mathrm{mol.L^{-1}}$ ; $[ClCH_2COO^-]=[H_3O^+] - [OH^-] \approx 0,10$ ; $[ClCH_2COOH] = C - [ClCH_2COO^-]$.

2. La relation $pH = pK_a + \log\dfrac{[A^-]}{[HA]}$ permet d’obtenir $C \approx 0,12\,\mathrm{mol.L^{-1}}$.

3. Pour $pH=2,9$, $[H_3O^+] = 1,26\times10^{-3}$ ; dilution : $V_f = C_i/C_f \times V_i \Rightarrow V_f \approx 9,5\,\mathrm{L}$. On ajoute donc environ $9,4\,\mathrm{L}$ d’eau.

Exercice 3 : Acide benzoïque

1. Équations de pH :

  • (a) $pH \approx \dfrac{1}{2}(pK_a - \log C_0)$.
  • (b) $pH = 7 - \dfrac{1}{2}(pK_e - pK_a - \log C_0)$.
  • (c) $[H_3O^+] = \sqrt{K_a(C_0 - [H_3O^+])}$.
  • (d) Équation exacte : $K_a = \dfrac{[H_3O^+]^2}{C_0-[H_3O^+]}$.

2. Calcul : $[H_3O^+] \approx \sqrt{K_aC_0} = \sqrt{6,3\times10^{-7}} = 7,94\times10^{-4}$ ; $pH \approx 3,10$.

3. $[A^-] = [H_3O^+] \approx 7,94\times10^{-4}$ ; $[HA] = C_0 - [A^-] \approx 9,2\times10^{-3}$.

4. Équation : $C_6H_5COOH + H_2O \rightleftharpoons C_6H_5COO^- + H_3O^+$. Couple mis en jeu : $C_6H_5COOH/C_6H_5COO^-$.

Exercice 4 : Situation‑problème

1. Avec $pH=11,3$, $[OH^-]=2,0\times10^{-3}$, on en déduit que Y est une base faible. Le degré d’ionisation $\alpha=0,20$ confirme.

2. Classement :

  • Ultramajoritaires : molécules non ionisées $Y$.
  • Majoritaires : anions $Y^-$ (base conjuguée).
  • Minoritaires : $OH^-$.
  • Ultraminoritaires : $H_3O^+$.

3. La distribution des espèces en fonction du pH suit une courbe sigmoïde centrée autour du $pK_b$ (ou du $pK_a$ de l’acide conjugué). La fraction de $Y$ diminue quand le pH augmente, celle de $Y^-$ augmente.
