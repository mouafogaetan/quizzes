Corrigé du Contrôle de Chimie — Acides/Bases, pH, Titrage & Dilutions

Exercice 1 — Vérification des savoirs

  1. Définitions : Un acide de Brønsted est une espèce capable de céder un proton H$^+$. Une base est l’espèce capable de le capter. Un couple acide/base est constitué d’un acide et de sa base conjuguée (qui diffèrent par un proton).
  2. Couples avec l’eau : $\mathrm{H_3O^+}/\mathrm{H_2O}$ et $\mathrm{H_2O}/\mathrm{HO^-}$. Équations : $\mathrm{H_2O + H_2O \rightleftharpoons H_3O^+ + HO^-}$.
  3. pH‑mètre : On réalise un étalonnage (calibration) avec des solutions tampons de pH connu ; principe : ajuster la réponse de l’électrode à des valeurs de référence.
  4. Vrai/Faux :
    1. Vrai : à 25 °C, pH > 7 signifie basique.
    2. Faux : il faut une fiole jaugée pour la précision.
    3. Vrai : $K_e$ croît avec la température.
    4. Faux : la formule n’est valable que si $[\mathrm{H_3O^+}]$ >> $10^{-7}$.
  5. QCM :
    1. Réponse B : $5{,}6\times10^{-8}\%$.
    2. Réponse A : la solution est acide car $pH < pH_{neutre}$ à 80 °C.

Exercice 2 — Application

pH mesuré $=2{,}87 \Rightarrow [\mathrm{H_3O^+}] = 10^{-2.87} \approx 1,35\times10^{-3}$ mol·L$^{-1}$.
  1. L’acide éthanoïque est un acide faible, car $[\mathrm{H_3O^+}]$ est très inférieur à $C_0 = 0,1$ mol·L$^{-1}$.
  2. Équation : $\mathrm{CH_3COOH + H_2O \rightleftharpoons CH_3COO^- + H_3O^+}$. Couple : $\mathrm{CH_3COOH/CH_3COO^-}$.
  3. $[\mathrm{CH_3COO^-}] = [\mathrm{H_3O^+}] = 1,35\times10^{-3}$ mol·L$^{-1}$ ; $[\mathrm{CH_3COOH}] = 0,1 - 1,35\times10^{-3} \approx 9,87\times10^{-2}$ mol·L$^{-1}$.
  4. $K_a = \dfrac{[\mathrm{H_3O^+}]^2}{C_0 - [\mathrm{H_3O^+}]} \approx \dfrac{(1,35\times10^{-3})^2}{9,87\times10^{-2}} \approx 1,85\times10^{-5}$. Donc $pK_a \approx 4,73$.
  5. Comparaison : $pK_a$(propanoïque) $=4,87$ > $4,73$. Donc l’acide éthanoïque est légèrement plus fort.

Exercice 3 — Utilisation

  1. $[\mathrm{H_3O^+}] = 10^{-2.95} = 1,12\times10^{-3}$ mol·L$^{-1}$. $\alpha_1 = [\mathrm{H_3O^+}]/C_0 = 1,12\times10^{-3}/2,0\times10^{-2} = 0,056$. Donc acide faible.
  2. Dans $B_1$, $C’=2,0\times10^{-4}$ mol·L$^{-1}$. $[\mathrm{H_3O^+}] = 10^{-4.05} = 8,91\times10^{-5}$. $\alpha_2 = 0,445$. Plus élevé : la dilution favorise la dissociation.
  3. Mélange : Volume total = 200 mL ; donc $[\mathrm{B}] = 1,0\times10^{-2}$ mol·L$^{-1}$ ; $[\mathrm{C}] = 7,5\times10^{-4}$ mol·L$^{-1}$. pH = 3,18 → $[\mathrm{H_3O^+}] = 6,6\times10^{-4}$. L’ion commun $H_3O^+$ limite la dissociation → $\alpha_3$ diminue.

Exercice 4 — Situations expérimentales

A) Étapes :

  1. Dans une fiole jaugée (500 mL), introduire environ 200 mL d’eau distillée.
  2. À l’aide d’une pipette jaugée, prélever le volume de solution commerciale d’HCl calculé à partir des données (masse volumique et % massique → concentration ~12 mol·L$^{-1}$, volume requis ~20,8 mL).
  3. Transvaser soigneusement la pipette dans la fiole.
  4. Compléter avec de l’eau distillée jusqu’au trait de jauge.
  5. Homogénéiser par retournements. Sécurité : lunettes, gants, manipulation sous hotte.

B) On calcule les moles de chaque ion :

  • $S_1$: $n(H_3O^+) = 2,0\times10^{-3}$ mol
  • $S_2$: $n(K^+) = n(Cl^-) = 0,1$ mol
  • $S_3$: $pH=1,70 \Rightarrow [H_3O^+] = 2,0\times10^{-2}$ mol·L$^{-1}$ ; volume = 0,030 L ; $n(H_3O^+) = 6,0\times10^{-4}$ mol ; mais $H_2SO_4$ donne 2 H$^+$ → vérifier. Approximation : $n(H_3O^+) \approx 1,2\times10^{-3}$ mol ; $n(SO_4^{2-})$ identique.
Bilan : les charges positives (K$^+$, H$^+$) et négatives (Cl$^-$, SO$^4^{2-}$) s’équilibrent globalement. Conclusion : la solution $S_4$ est électriquement neutre.
