I. Les équipements de base d'un réseau

1. Définition

Les équipements de base d'un réseau sont les composants matériels et logiciels qui permettent de connecter des ordinateurs entre eux. L'objectif de la mise en réseau est de faciliter la communication et le partage de ressources (imprimantes, scanners, fichiers, etc.).

Les équipements requis pour créer un réseau local incluent :

  • Ordinateurs et périphériques
  • Système d'exploitation réseau
  • Câbles réseau
  • Serveur
  • Cartes réseau

a. L'ordinateur

L'ordinateur est l'élément principal d'un réseau. Pour être connecté, il doit être équipé d'une carte réseau et d'un système d'exploitation compatible réseau. Le système d'exploitation gère le fonctionnement de l'ordinateur ainsi que la communication avec les autres équipements du réseau.

b. La carte réseau (NIC)

La carte réseau (Network Interface Card - NIC) constitue l'interface entre l'ordinateur et le câble du réseau. Sa fonction est de préparer, envoyer et contrôler les informations circulant sur le réseau.

Chaque carte réseau possède une adresse physique unique appelée adresse MAC (Media Access Control). Exemple : SE:FF:56:A2:AF:15

c. Le serveur

On distingue deux types d'ordinateurs dans un réseau : les serveurs et les clients. Le serveur est un ordinateur performant doté d'une grande capacité mémoire et d'une puissance de calcul élevée. Il partage ses ressources (fichiers, espaces de stockage, imprimantes, etc.) avec les autres machines appelées clients.

d. Les câbles réseau

Le câblage réseau regroupe tous les moyens permettant d'établir une liaison de communication entre les différents équipements. Il est généralement constitué de câbles et de connecteurs spécifiques.

Le câble transporte les informations d'un ordinateur à un autre. Le choix du type de câble dépend de la structure du réseau à mettre en place. Les principaux types sont :

Câbles à paires torsadées

Il existe deux types de câbles à paires torsadées : blindés (STP) et non blindés (UTP). Les câbles blindés offrent une meilleure protection contre les interférences électromagnétiques.

Image d'illustration : Câble à paires torsadées avec connecteur RJ45

Câble coaxial

Le câble coaxial, parfois appelé câble BNC, est un fil de cuivre rigide entouré d'une gaine isolante en plastique. Similaire au câble d'antenne TV, il utilise des signaux électriques pour transmettre les données.

Image d'illustration : Câble coaxial

Fibre optique

La fibre optique conduit des signaux lumineux (photons) à travers une fibre de verre entourée d'une gaine réfléchissante qui isole le signal des perturbations extérieures.

Avantages :

  • Vitesses de transmission très élevées (jusqu'à 155 Mbps et au-delà)
  • Idéale pour le transfert de contenus vidéo, audio et multimédia
  • Imperméable aux perturbations électromagnétiques
  • Signal stable sur de longues distances sans dégradation

II. Les équipements d'interconnexion des réseaux

Un réseau local permet d'interconnecter les ordinateurs d'une organisation. Cependant, une organisation comporte généralement plusieurs réseaux locaux qu'il faut parfois relier entre eux. Pour cela, des équipements spécifiques sont nécessaires.

Définition

Un équipement d'interconnexion est un matériel qui permet de relier les ordinateurs d'un réseau ou plusieurs réseaux entre eux. Les principaux équipements sont :

  • Hubs (concentrateurs)
  • Modems
  • Switches (commutateurs)
  • Routeurs
  • Passerelles

a. Le Hub (concentrateur)

Le Hub est l'élément central d'un réseau en étoile. Il permet à tous les ordinateurs d'un réseau local de communiquer en reliant tous les câbles à lui. Lorsqu'un ordinateur envoie une information, le Hub la redistribue à tous les autres ordinateurs, mais seul le destinataire prévu ouvrira et utilisera le message.

Image d'illustration : Hub réseau

b. Le Switch (commutateur)

Le Switch est similaire au Hub mais avec une différence majeure. Lorsqu'il reçoit un message d'un ordinateur, il le renvoie directement au destinataire concerné sans que les autres ordinateurs en soient informés, améliorant ainsi la sécurité et l'efficacité du réseau.

Image d'illustration : Switch réseau

c. Le répéteur

Un répéteur a pour rôle de régénérer les signaux qu'il reçoit des équipements réseau afin de préserver leur intégrité sur des distances dépassant la longueur maximale normalement supportée par le câble.

d. La passerelle

Lorsqu'une organisation dispose de plusieurs réseaux locaux utilisant des protocoles de communication différents, une conversion de protocoles est nécessaire pour les relier. La passerelle est l'équipement qui permet cette conversion.

e. Le routeur

Un routeur est un périphérique d'inter-réseau qui permet de relier plusieurs réseaux locaux, même éloignés géographiquement. Il offre plusieurs interfaces de communication et est équipé d'un logiciel spécialisé qui optimise le routage des données d'un réseau à un autre.

Image d'illustration : Routeur

f. Le modem

Le modem (modulateur-démodulateur) est un périphérique qui permet de communiquer avec des utilisateurs distants via un réseau analogique (comme une ligne téléphonique). Il est essentiel pour se connecter à Internet.

Image d'illustration : Modem

Exercice d'évaluation

Associez chaque équipement à sa fonction :

Équipement Fonction
A) Routeur a) Interconnecte les ordinateurs
B) Passerelle b) Relie les réseaux de même technologie
C) Répéteur c) Interconnecte les réseaux différents
D) Carte réseau d) Relie les réseaux de protocole différent
E) Pont e) Interface entre l'ordinateur et le câble réseau
F) Hub ou Switch f) Amplifie les signaux pour prolonger la transmission
