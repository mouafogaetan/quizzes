Programmer en HTML - Insertion de médias, tableaux et formulaires

1. Insertion de fichiers multimédias

Images

Utilisation de la balise <img> avec les attributs obligatoires :

  • src : chemin vers l'image
  • alt : texte alternatif descriptif
<img src="mon-image.jpg" alt="Description de l'image">

Audio

Utilisation de la balise <audio> avec l'attribut controls :

<audio controls>
  <source src="musique.mp3" type="audio/mpeg">
  <source src="musique.ogg" type="audio/ogg">
</audio>

Vidéo

Utilisation de la balise <video> avec l'attribut controls :

<video controls width="600">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
</video>

Exercice d'application

Créez une page HTML qui affiche :

  1. Une image nommée "logo.png" avec le texte alternatif "Logo de l'école"
  2. Un lecteur audio pour le fichier "presentation.mp3"
  3. Un lecteur vidéo pour le fichier "tutoriel.mp4"
<!DOCTYPE html>
<html>
<head>
  <title>Exercice Multimédia</title>
</head>
<body>
  <img src="logo.png" alt="Logo de l'école">
  <audio controls>
    <source src="presentation.mp3" type="audio/mpeg">
  </audio>
  <video controls width="600">
    <source src="tutoriel.mp4" type="video/mp4">
  </video>
</body>
</html>

2. Création de tableaux

Structure de base d'un tableau HTML :

<table border="1">
  <caption>Titre du tableau</caption>
  <tr>
    <th>En-tête 1</th>
    <th>En-tête 2</th>
  </tr>
  <tr>
    <td>Donnée 1</td>
    <td>Donnée 2</td>
  </tr>
</table>

Attributs utiles :

  • border : épaisseur de la bordure
  • colspan : fusion de colonnes
  • rowspan : fusion de lignes

Exercice d'application

Créez un tableau HTML représentant un emploi du temps scolaire avec les colonnes : Heure, Lundi, Mardi, Mercredi, Jeudi, Vendredi.

Utilisez rowspan pour les heures qui couvrent plusieurs créneaux (ex: 8h-10h).

<table border="1">
  <caption>Emploi du temps</caption>
  <tr>
    <th>Heure</th>
    <th>Lundi</th>
    <th>Mardi</th>
    <th>Mercredi</th>
    <th>Jeudi</th>
    <th>Vendredi</th>
  </tr>
  <tr>
    <td rowspan="2">8h-10h</td>
    <td>Maths</td>
    <td>Français</td>
    <td>Sport</td>
    <td>Histoire</td>
    <td>Anglais</td>
  </tr>
  <tr>
    <td>Physique</td>
    <td>SVT</td>
    <td>Musique</td>
    <td>Géographie</td>
    <td>Arts</td>
  </tr>
</table>

3. Création de formulaires

Structure de base d'un formulaire :

<form method="post" action="traitement.php">
  <!-- Éléments du formulaire -->
  <input type="text" name="nom" placeholder="Votre nom">
  <input type="submit" value="Envoyer">
</form>

Éléments de formulaire courants

Type Balise Exemple
Texte <input type="text">
Mot de passe <input type="password">
Case à cocher <input type="checkbox"> Option 1
Bouton radio <input type="radio"> Option A
Option B
Liste déroulante <select>
Zone de texte <textarea>

Exercice d'application

Créez un formulaire d'inscription avec les champs suivants :

  • Nom (texte)
  • Email (email)
  • Mot de passe (password)
  • Date de naissance (date)
  • Niveau d'étude (liste déroulante avec options: Bac, Bac+2, Bac+3, Bac+5)
  • Acceptation des conditions (case à cocher)
  • Bouton de soumission
<form method="post" action="inscription.php">
  <label>Nom:</label>
  <input type="text" name="nom" required><br><br>

  <label>Email:</label>
  <input type="email" name="email" required><br><br>

  <label>Mot de passe:</label>
  <input type="password" name="password" required><br><br>

  <label>Date de naissance:</label>
  <input type="date" name="birthdate"><br><br>

  <label>Niveau d'étude:</label>
  <select name="niveau">
    <option value="bac">Bac</option>
    <option value="bac2">Bac+2</option>
    <option value="bac3">Bac+3</option>
    <option value="bac5">Bac+5</option>
  </select><br><br>

  <input type="checkbox" name="conditions" required>
  <label>J'accepte les conditions d'utilisation</label><br><br>

  <input type="submit" value="S'inscrire">
</form>

├── apropos.html
└── contact.html

Intégration du CSS

Trois méthodes pour intégrer CSS dans HTML :

  1. Inline : directement dans la balise HTML
    <p style="color: blue; font-size: 16px;">Texte en bleu</p>
  2. Interne : dans l'en-tête du document
    <head>
      <style>
        p {
          color: blue;
          font-size: 16px;
        }
      </style>
    </head>
  3. Externe : lien vers un fichier CSS séparé (méthode recommandée)
    <head>
      <link rel="stylesheet" href="styles/style.css">
    </head>

Exercice final

Créez une page web complète qui intègre :

  1. Un en-tête avec le nom de votre école
  2. Un menu de navigation vers les sections
  3. Un tableau avec des données
  4. Un formulaire de contact
  5. Un pied de page
  6. Un design responsive qui s'adapte aux mobiles

Conseils pour réussir cet exercice :

  • Utilisez des balises sémantiques HTML5 (header, nav, main, section, footer)
  • Structurez votre CSS avec des media queries pour le responsive
  • Testez votre site sur différentes tailles d'écran
  • Validez votre code HTML avec le validateur W3C
  • Optimisez les images pour le web
