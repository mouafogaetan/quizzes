Utilisation de l'invite de commandes

Apprendre à maîtriser l'interface en ligne de commande sous Windows

Introduction à l'invite de commandes

Compétences visées

  • Comprendre l'importance de l'utilisation de la ligne de commande
  • Savoir accéder à la ligne de commande

Pourquoi utiliser la ligne de commande ?

Avant l'avènement des interfaces graphiques, les ordinateurs fonctionnaient exclusivement en mode texte. L'apparition des premiers systèmes d'exploitation graphiques a progressivement transformé l'interaction "tout clavier" en interaction "tout souris" que nous connaissons aujourd'hui.

Malgré les avantages évidents des interfaces graphiques, la ligne de commande offre des fonctionnalités puissantes pour :

  • Automatiser des tâches répétitives ou complexes
  • Créer des sauvegardes ciblées de fichiers modifiés récemment
  • Extraire spécifiquement des éléments de multiples fichiers (comme la première page de plusieurs PDF)
  • Résoudre des problèmes système via le mode sans échec
  • Créer des fichiers batch (.bat ou .cmd) pour automatiser des processus
  • Renommer en masse des fichiers selon des règles précises

L'utilisation de la ligne de commande est particulièrement utile lors de connexions distantes, où les interfaces graphiques consomment plus de ressources réseau. Pour des connexions lentes, l'interaction en ligne de commande reste beaucoup plus fluide.

Accéder à l'invite de commande

L'invite (prompt)

L'invite, aussi appelé prompt, est le message indiquant que le système est prêt à recevoir des commandes. Il se présente généralement sous la forme :

C:\>

Cette invite indique :

  • Le lecteur actif (ici C:)
  • Le chemin du répertoire courant (après le backslash $\backslash$)
  • Le caractère > qui marque la fin de l'invite
  • Un curseur clignotant indiquant où taper les commandes

Structure d'une ligne de commande

Une ligne de commande suit généralement cette structure :

commande [options] [arguments]

Où :

  • Commande : Le mot-clé principal indiquant l'action à exécuter
  • Options (facultatives) : Modificateurs précédés d'un slash (/), qui changent le comportement de la commande
  • Arguments (facultatifs) : Compléments précisant les cibles ou paramètres de la commande

Note : Les éléments entre crochets [] sont optionnels. Les commandes, options et arguments sont séparés par des espaces, qu'il ne faut pas oublier.

Exemples de commandes de base

C:\> CD \Documents\recherche
# Se déplacer vers le répertoire \Documents\recherche
C:\> TIME
# Afficher l'heure système
C:\> CLS
# Effacer l'écran de l'invite de commande
C:\> VER
# Afficher la version du système d'exploitation

Utilisation pratique de l'invite de commandes

Compétences visées

  • Utiliser les commandes de manipulation de répertoires et fichiers
  • Savoir utiliser le système d'aide intégré

Scénario pratique

Imaginons qu'un ami vous demande de lui envoyer sur une clé USB plusieurs fichiers (audio, vidéo, etc.) qui sont dispersés à différents endroits sur votre ordinateur. Il connaît seulement les noms des fichiers mais pas leurs emplacements exacts.

Comment résoudre ce problème efficacement avec l'invite de commandes ?

Lancer l'invite de commandes

Windows propose plusieurs méthodes pour accéder à l'invite de commandes :

Méthode 1 : Via le menu Démarrer

  1. Ouvrir le menu Démarrer
  2. Aller dans "Programmes" → "Accessoires"
  3. Sélectionner "Invite de commandes"

Méthode 2 : Via la commande Exécuter

  1. Appuyer sur ⊞ Win + R
  2. Taper "cmd" dans la zone de dialogue
  3. Appuyer sur Entrée

Une fenêtre d'invite de commandes s'ouvrira, affichant quelque chose comme :

Microsoft Windows [version 10.0.14393]
(c) 2016 Microsoft Corporation. Tous droits réservés.
C:\Users\VotreNom>

Les commandes essentielles

1. Création d'un répertoire

La commande MD ou MKDIR (Make Directory) permet de créer un nouveau répertoire.

C:\> MD CoursInfo
# Crée un répertoire "CoursInfo" à l'emplacement courant

2. Navigation entre répertoires

La commande CD (Change Directory) permet de se déplacer dans l'arborescence.

C:\> CD Ubuntu
# Se déplace vers le répertoire Ubuntu
C:\Ubuntu> CD ..
# Remonte d'un niveau dans l'arborescence
C:\Ubuntu> CD \
# Retourne à la racine du lecteur

3. Suppression de répertoires et fichiers

Pour un répertoire : RD ou RMDIR (Remove Directory)

C:\Documents> RD CoursInfo
# Supprime le répertoire CoursInfo (doit être vide)

Pour un fichier : DEL ou ERASE

C:\Documents> DEL CoursInfo.txt
# Supprime le fichier CoursInfo.txt

4. Renommer répertoires et fichiers

La commande REN ou RENAME permet de renommer éléments.

C:\Documents> REN CoursInfo CoursMaths
# Renomme le répertoire CoursInfo en CoursMaths
C:\Documents> REN CoursInfo.txt CoursInformation.doc
# Renomme et change l'extension du fichier

5. Afficher le contenu d'un répertoire

La commande DIR (Directory) liste le contenu d'un répertoire.

C:\Documents> DIR
# Affiche tous les fichiers et répertoires dans Documents

On peut utiliser des caractères spéciaux pour filtrer les résultats :

  • ? représente un caractère unique
  • * représente une chaîne de caractères
DIR b??.*
# Fichiers de 3 caractères commençant par b
DIR *.com
# Tous les fichiers avec l'extension .com
DIR b*c*
# Fichiers dont le nom commence par b et l'extension par c

6. Copier et déplacer des fichiers

Copier : COPY

COPY Noel.pdf C:\Courrier
# Copie Noel.pdf vers le répertoire Courrier

Déplacer : MOVE

MODE Noel.pdf C:\Courrier
# Déplace Noel.pdf vers le répertoire Courrier

Copier des répertoires : XCOPY

XCOPY C:\Documents D:\Backup /E
# Copie récursive de Documents vers Backup

Obtenir de l'aide sur les commandes

Pour obtenir des informations sur une commande spécifique, utilisez :

HELP [commande]
# Affiche l'aide générale ou spécifique à une commande
DIR /?
# Affiche l'aide pour la commande DIR
