Condensé des Notions Essentielles de JavaScript

1. Introduction au JavaScript

Notions essentielles

  • JavaScript est un langage de script orienté objet exécuté côté client (navigateur)
  • Il permet de créer des pages web interactives et dynamiques
  • Limites : ne peut pas accéder directement aux bases de données ou au système de fichiers
  • On intègre JavaScript avec la balise <script>
Exercice d'application

Pourquoi ne peut-on pas utiliser uniquement le HTML pour créer des pages web interactives?

Le HTML est un langage de balisage qui permet de structurer le contenu d'une page web, mais il ne permet pas d'effectuer des opérations logiques, des calculs, ou d'interagir dynamiquement avec l'utilisateur. JavaScript est nécessaire pour ajouter de l'interactivité et des fonctionnalités dynamiques.

2. Variables et Opérations

Notions essentielles

  • Déclaration de variables: var, let, const
  • Types de données: nombres, chaînes, booléens, etc.
  • Opérateurs arithmétiques: +, -, *, /, %, **
  • Opérateurs de comparaison: ==, !=, ===, !==, >, <, >=, <=
  • Opérateurs logiques: &&, ||, !
Exemple de code
// Déclaration de variables
var nom = "Alice";
let age = 25;
const TAUX_TVA = 0.2;

// Opérations
let prixHT = 100;
let prixTTC = prixHT * (1 + TAUX_TVA);
console.log("Prix TTC: " + prixTTC);
Exercice d'application

Écrivez un script qui demande à l'utilisateur le rayon d'un cercle et calcule son aire (A = $\pi r^2$).

// Solution
let rayon = parseFloat(prompt("Entrez le rayon du cercle:"));
const PI = 3.14159;
let aire = PI * Math.pow(rayon, 2);
alert("L'aire du cercle est: " + aire.toFixed(2));

3. Structures de Contrôle

Notions essentielles

  • Conditionnelles: if, else if, else, switch
  • Boucles: for, while, do...while
Exemple de code
// Structure conditionnelle
let note = 15;
if (note >= 10) {
    console.log("Admis");
} else {
    console.log("Recalé");
}

// Boucle for
for (let i = 0; i < 5; i++) {
    console.log("Itération: " + i);
}

// Boucle while
let compteur = 0;
while (compteur < 3) {
    console.log("Compteur: " + compteur);
    compteur++;
}
Exercice d'application

Écrivez une fonction qui affiche tous les nombres pairs entre 0 et un nombre donné.

// Solution
function afficherNombresPairs(limite) {
    for (let i = 0; i <= limite; i++) {
        if (i % 2 === 0) {
            console.log(i);
        }
    }
}

afficherNombresPairs(10);

4. Tableaux

Notions essentielles

  • Déclaration: let arr = []; ou let arr = new Array();
  • Accès aux éléments: arr[index] (index commence à 0)
  • Propriété length: arr.length
  • Méthodes courantes: push(), pop(), shift(), unshift(), slice(), splice()
Exemple de code
// Création et manipulation de tableaux
let fruits = ["pomme", "banane", "orange"];
fruits.push("kiwi"); // Ajoute à la fin
fruits.pop(); // Supprime le dernier élément
fruits.unshift("fraise"); // Ajoute au début
fruits.shift(); // Supprime le premier élément

// Parcours de tableau
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}
Exercice d'application

Écrivez une fonction qui prend un tableau de nombres en paramètre et retourne la somme de ses éléments.

// Solution
function sommeTableau(tableau) {
    let somme = 0;
    for (let i = 0; i < tableau.length; i++) {
        somme += tableau[i];
    }
    return somme;
}

// Test
let nombres = [1, 2, 3, 4, 5];
console.log(sommeTableau(nombres)); // Affiche 15

5. Fonctions

Notions essentielles

  • Déclaration: function nomFonction(param1, param2) { ... }
  • Return: retourne une valeur (optionnel)
  • Portée des variables: globale vs locale
Exemple de code
// Déclaration de fonction
function calculerSurface(longueur, largeur) {
    return longueur * largeur;
}

// Appel de fonction
let surface = calculerSurface(5, 3);
console.log("Surface: " + surface); // Affiche 15

// Fonction fléchée (ES6)
const multiplier = (a, b) => a * b;
console.log(multiplier(4, 5)); // Affiche 20
Exercice d'application

Écrivez une fonction qui prend un tableau de nombres en paramètre et retourne le plus grand nombre.

// Solution
function trouverMaximum(nombres) {
    let max = nombres[0];
    for (let i = 1; i < nombres.length; i++) {
        if (nombres[i] > max) {
            max = nombres[i];
        }
    }
    return max;
}

// Test
let valeurs = [12, 45, 2, 37, 9];
console.log(trouverMaximum(valeurs)); // Affiche 45

6. Événements

Notions essentielles

  • Gestionnaires d'événements: onclick, onchange, onmouseover, etc.
  • Accès aux éléments: document.getElementById()
  • Modification du contenu: element.innerHTML, element.value
Exemple de code
<!-- HTML -->
<button id="monBouton">Cliquez-moi</button>
<p id="monParagraphe"></p>

<script>
// JavaScript
document.getElementById("monBouton").onclick = function() {
    document.getElementById("monParagraphe").innerHTML = "Vous avez cliqué!";
};
</script>
Exercice d'application

Créez un convertisseur de température Celsius vers Fahrenheit qui s'active lors du clic sur un bouton.

<!-- HTML -->
<input type="number" id="celsius" placeholder="Température en Celsius">
<button onclick="convertir()">Convertir</button>
<p id="resultat"></p>

<script>
// JavaScript
function convertir() {
    let celsius = document.getElementById("celsius").value;
    let fahrenheit = (celsius * 9/5) + 32;
    document.getElementById("resultat").innerHTML = 
        celsius + "°C = " + fahrenheit.toFixed(2) + "°F";
}
</script>
