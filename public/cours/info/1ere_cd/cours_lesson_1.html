Séquence 1: Installation d'un système d'exploitation Windows

Situation Problème

Le lycée technique de Douala a reçu un don de 15 ordinateurs de la part d'une entreprise locale. Ces ordinateurs sont équipés de disques durs vierges et doivent être configurés avec Windows 10 pour équiper le nouveau laboratoire d'informatique. Le proviseur a désigné une équipe d'élèves pour effectuer cette installation.

Problèmes à résoudre :

  • Comment accéder au BIOS/UEFI pour configurer l'ordre de démarrage?
  • Comment partitionner correctement le disque dur pour un usage éducatif?
  • Quelle clé produit utiliser pour une installation légale dans le contexte scolaire camerounais?
  • Comment créer des comptes utilisateurs adaptés aux besoins des élèves et professeurs?
  • Quelles configurations spécifiques appliquer pour un environnement scolaire?

Comment procéder pour installer et configurer Windows 10 sur ces postes en respectant le contexte camerounais ?

Activités d'apprentissage

Activité 1: Préparation et accès au BIOS/UEFI

Interface UEFI moderne

Objectif: Accéder au BIOS/UEFI et configurer l'ordre de démarrage.

Mise en situation: Vous devez installer Windows 10 à partir d'une clé USB sur un ordinateur neuf.

Tâche:

  1. Redémarrez l'ordinateur et appuyez sur la touche d'accès au BIOS/UEFI (F2, F10, Del, Échap selon le fabricant)
  2. Repérez l'onglet "Boot" ou "Démarrage"
  3. Modifiez l'ordre de priorité pour que l'ordinateur démarre d'abord sur la clé USB
  4. Activez le mode UEFI (si disponible) et sécurisez le boot
  5. Sauvegardez les modifications et quittez

Astuce: Sur les ordinateurs récents, vous pouvez accéder aux options de démarrage en maintenant la touche Maj enfoncée tout en cliquant sur "Redémarrer" dans Windows.

Activité 2: Processus d'installation de Windows

Écran d'installation de Windows 10

Objectif: Installer Windows 10 étape par étape.

Mise en situation: Installation de Windows 10 sur un poste du lycée technique de Douala.

Tâche:

  1. Démarrez sur le support d'installation (DVD ou clé USB)
  2. Sélectionnez la langue, le format horaire et le clavier (français, Cameroun)
  3. Cliquez sur "Installer maintenant"
  4. Entrez la clé produit (pour le contexte éducatif, utilisez une licence Education)
  5. Acceptez les termes du contrat de licence
  6. Choisissez "Installation personnalisée"

Activité 3: Partitionnement du disque dur

Interface de partitionnement durant l'installation

Objectif: Partitionner le disque dur selon les besoins éducatifs.

Mise en situation: Préparer les partitions pour un usage scolaire avec séparation du système et des données.

Tâche:

  1. Dans l'écran de choix de partition, sélectionnez "Nouveau" pour créer des partitions
  2. Créez une partition système d'au moins 100 Go pour Windows et les programmes
  3. Créez une partition données d'au moins 200 Go pour les documents des utilisateurs
  4. Optionnellement, créez une petite partition de recovery (10-15 Go)
  5. Sélectionnez la partition système pour l'installation
  6. Formatez les partitions en NTFS

Calcul d'espace: Pour un disque de 500 Go, on peut prévoir: $E_{\text{système}} = 100\text{Go}$, $E_{\text{données}} = 385\text{Go}$, $E_{\text{recovery}} = 15\text{Go}$

Activité 4: Configuration et création de comptes

Interface de création de comptes utilisateurs

Objectif: Configurer les paramètres initiaux et créer des comptes utilisateurs.

Mise en situation: Configuration finale d'un poste pour le laboratoire d'informatique.

Tâche:

  1. Personnalisez les paramètres de base (région: Cameroun, clavier: français)
  2. Pour un environnement scolaire, choisissez "Configurer pour un établissement scolaire"
  3. Créez un compte administrateur "AdminLabo" avec un mot de passe robuste
  4. Créez un compte standard "Eleve" sans mot de passe pour un usage général
  5. Configurez les paramètres de confidentialité selon la politique de l'établissement
  6. Attendez la fin de l'installation et des mises à jour initiales

Activité 5: Post-installation et configuration

Post-installation et configuration

Objectif: Finaliser l'installation et configurer les paramètres essentiels.

Mise en situation: Préparer le poste pour une utilisation en laboratoire scolaire.

Tâche:

  1. Installez les pilotes manquants (carte graphique, son, réseau)
  2. Activez Windows Update et installez les dernières mises à jour
  3. Installez les logiciels éducatifs nécessaires (suite bureautique, outils pédagogiques)
  4. Configurez les paramètres de sécurité (pare-feu Windows, contrôle de compte utilisateur)
  5. Créez un point de restauration système
  6. Testez tous les périphériques (imprimante, scanner, etc.)

Résumé

L'installation d'un système d'exploitation Windows suit un processus structuré qui nécessite une préparation et une configuration minutieuses.

Étapes clés de l'installation de Windows :

Étape 1: Préparation
  • Vérifier les exigences système : processeur 1 GHz, 2 Go de RAM (4 Go recommandés), 20 Go d'espace disque
  • Préparer le support d'installation (DVD ou clé USB bootable)
  • Sauvegarder les données importantes
Étape 2: Configuration du BIOS/UEFI
  • Accéder au BIOS/UEFI avec les touches F2, F10, Del ou Échap
  • Configurer l'ordre de démarrage (boot order) pour prioriser le support d'installation
  • Activer le mode UEFI pour les systèmes récents (offre plus de sécurité)
Étape 3: Installation proprement dite
  • Choisir la langue et les préférences régionales (Cameroun, français)
  • Entrer la clé produit (pour établissements scolaires, utiliser des licences Education)
  • Accepter les termes du contrat de licence
Étape 4: Partitionnement
  • Créer des partitions selon les besoins : système, données, recovery
  • Formater les partitions en NTFS ( système de fichiers standard pour Windows)
  • Calculer les tailles de partitions : $E_{\text{système}} = 100\text{Go}$, $E_{\text{données}} = \text{reste} - 15\text{Go}$, $E_{\text{recovery}} = 15\text{Go}$
Étape 5: Configuration finale
  • Créer les comptes utilisateurs (administrateur et standard)
  • Configurer les paramètres de confidentialité
  • Attendre la fin de l'installation et des mises à jour

Considerations pour le contexte camerounais :

  • Utiliser des versions Education de Windows pour les établissements scolaires
  • Configurer les paramètres régionaux pour le Cameroun
  • Prévoir une connexion internet stable pour les activations et mises à jour
  • Adapter les configurations aux contraintes électriques (onduleurs pour pallier les coupures)

Exercice d'application

Vous êtes technicien informatique au collège Saint Michel de Yaoundé. On vous demande d'installer Windows 10 Education sur 10 postes informatiques qui serviront aux élèves de première.

Caractéristiques techniques des ordinateurs :

  • Processeur : Intel Core i3 2.4 GHz
  • Mémoire RAM : 8 Go
  • Disque dur : 1 To (1000 Go)
  • Cartes réseau sans fil et Ethernet intégrées

Questions :

  1. Quelle est la procédure pour accéder au BIOS/UEFI sur ces machines si la touche standard est F2?
  2. Proposez un schéma de partitionnement pour un disque de 1 To avec justification des tailles.
  3. Quels types de comptes utilisateurs devez-vous créer et avec quels privilèges?
  4. Quelles configurations spécifiques devez-vous appliquer pour un environnement scolaire?
  5. Listez les étapes post-installation essentielles pour ces postes.

Correction proposée :

  1. Pour accéder au BIOS/UEFI :
    • Redémarrer l'ordinateur
    • Appuyer plusieurs fois sur F2 dès l'allumage
    • Si cela ne fonctionne pas, essayer les touches F10, Del ou Échap selon le fabricant
    • Dans le BIOS, naviguer jusqu'à l'onglet "Boot" et configurer l'ordre de démarrage
  2. Schéma de partitionnement pour 1 To :
    • Partition système (C:): 150 Go - Pour Windows et programmes
    • Partition données (D:): 800 Go - Pour documents des utilisateurs
    • Partition recovery: 50 Go - Pour sauvegardes et récupération
    • Calcul: $150 + 800 + 50 = 1000$ Go (1 To)
  3. Comptes utilisateurs à créer :
    • Compte administrateur : "AdminCollege" avec tous les privilèges, mot de passe robuste
    • Comptes élèves : Comptes standards avec limitations, pas de mot de passe pour faciliter l'accès
    • Compte professeur : Compte avec privilèges étendus mais sans accès administrateur complet
  4. Configurations spécifiques pour environnement scolaire :
    • Configurer les paramètres régionaux pour Cameroun
    • Désactiver les jeux et applications non éducatives
    • Configurer le navigateur avec une page d'accueil éducative
    • Configurer des restrictions d'accès à certains sites web
    • Installer les logiciels éducatifs nécessaires
  5. Étapes post-installation :
    1. Installer tous les pilotes manquants
    2. Mettre à jour Windows avec les derniers correctifs
    3. Installer les logiciels éducatifs (suite bureautique, outils pédagogiques)
    4. Configurer les imprimantes et périphériques partagés
    5. Créer un point de restauration système
    6. Test complet du poste et des périphériques

Réponse à la situation problème

Pour installer et configurer Windows 10 sur les 15 ordinateurs du lycée technique de Douala, l'équipe d'élèves pourrait procéder comme suit :

  1. Préparation des supports d'installation :
    • Télécharger l'image ISO de Windows 10 Education depuis le site Microsoft
    • Créer des clés USB bootables avec l'outil Media Creation Tool
    • Vérifier que les clés produits Education sont valides pour l'activation
  2. Accès et configuration du BIOS/UEFI :
    • Redémarrer chaque ordinateur et accéder au BIOS/UEFI avec la touche appropriée (F2, F10, Del ou Échap)
    • Configurer l'ordre de démarrage pour prioriser le boot sur USB
    • Vérifier que le mode UEFI est activé pour une meilleure sécurité
    • Sauvegarder les paramètres et quitter
  3. Processus d'installation :
    • Démarrer sur la clé USB d'installation
    • Sélectionner la langue française et les paramètres régionaux pour le Cameroun
    • Entrer la clé produit Windows 10 Education
    • Accepter les termes du contrat de licence
    • Choisir "Installation personnalisée"
  4. Partitionnement des disques durs :
    • Pour des disques de 500 Go, créer :
      • Partition système (C:): 100 Go - Pour le système et les programmes
      • Partition données (D:): 385 Go - Pour les documents des utilisateurs
      • Partition recovery: 15 Go - Pour les outils de récupération
    • Formater les partitions en NTFS
    • Sélectionner la partition système pour l'installation
  5. Configuration et création de comptes :
    • Personnaliser les paramètres pour un usage scolaire
    • Créer un compte administrateur "AdminLycée" avec mot de passe robuste
    • Créer un compte standard "Élève" sans mot de passe pour un accès facile
    • Configurer les paramètres de confidentialité adaptés à un environnement éducatif
  6. Post-installation :
    • Installer les pilotes manquants (à partir du site du fabricant si nécessaire)
    • Activer Windows Update et installer toutes les mises à jour disponibles
    • Installer les logiciels éducatifs requis (suite Office, logiciels techniques)
    • Configurer les périphériques partagés (imprimantes, scanner)
    • Créer un point de restauration système pour chaque poste
    • Documenter la configuration de chaque machine

Cette approche permettra d'installer Windows 10 de manière standardisée sur tous les postes, avec une configuration adaptée au contexte éducatif camerounais et aux ressources disponibles.
