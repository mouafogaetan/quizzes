Suites Numériques - Notions Essentielles

1. Définition et généralités

Une suite numérique est une fonction $U$ de $\mathbb{N}$ (ou d'une partie $I$ de $\mathbb{N}$) dans $\mathbb{R}$.

Notation :

On note $(U_n)_{n \in I}$ ou simplement $(U_n)$

Types de suites :
  • Formule explicite : $U_n = f(n)$ où $f$ est une fonction numérique
  • Formule de récurrence : $U_{n+1} = f(U_n)$ avec $U_0$ connu

Exercice : On donne $f(x) = \frac{2x+3}{x+1}$. Calcule $f(2)$, $f(5)$, $f(x+1)$.

$f(2) = \frac{2(2)+3}{2+1} = \frac{7}{3}$

$f(5) = \frac{2(5)+3}{5+1} = \frac{13}{6}$

$f(x+1) = \frac{2(x+1)+3}{(x+1)+1} = \frac{2x+5}{x+2}$

2. Limite d'une suite

Si $(U_n)$ est définie par $U_n = f(n)$ où $f$ est une fonction numérique, et si $\lim_{x \to +\infty} f(x) = l$, alors $\lim_{n \to +\infty} U_n = l$.

Exercice : On donne la suite $(U_n)$ définie par $U_n = \frac{3n+5}{n+1}$. Calcule sa limite.

On a $f(x) = \frac{3x+5}{x+1}$ et $\lim_{x \to +\infty} f(x) = 3$, donc $\lim_{n \to +\infty} U_n = 3$.

3. Monotonie d'une suite

Une suite $(U_n)$ est dite :

  • Croissante si pour tout $n \in I$, $U_{n+1} \geq U_n$
  • Décroissante si pour tout $n \in I$, $U_{n+1} \leq U_n$
  • Constante si pour tout $n \in I$, $U_{n+1} = U_n$
Propriétés :
  1. Si $(U_n)$ est définie par $U_n = f(n)$, alors $(U_n)$ et $f$ ont les mêmes variations sur $[0; +\infty[$
  2. Pour une suite à termes strictement positifs :
    • Si $\frac{U_{n+1}}{U_n} \geq 1$, la suite est croissante
    • Si $\frac{U_{n+1}}{U_n} \leq 1$, la suite est décroissante
    • Si $\frac{U_{n+1}}{U_n} = 1$, la suite est constante

Exercice : Montre que la suite $(U_n)$ définie par $U_n = \frac{n+2}{n+1}$ est strictement décroissante.

Calculons $\frac{U_{n+1}}{U_n} = \frac{(n+3)/(n+2)}{(n+2)/(n+1)} = \frac{(n+3)(n+1)}{(n+2)^2}$

Or $(n+3)(n+1) = n^2 + 4n + 3$ et $(n+2)^2 = n^2 + 4n + 4$

Donc $\frac{U_{n+1}}{U_n} = \frac{n^2+4n+3}{n^2+4n+4} < 1$ pour tout $n \in \mathbb{N}$

La suite est donc strictement décroissante.

4. Représentation graphique

Pour représenter graphiquement une suite définie par récurrence $U_{n+1} = f(U_n)$ :

  1. Représenter la fonction $f$ et la droite $y = x$
  2. Placer $U_0$ sur l'axe des abscisses
  3. Tracer la verticale jusqu'à la courbe de $f$ pour obtenir $f(U_0) = U_1$
  4. Tracer l'horizontale jusqu'à la droite $y = x$ pour reporter $U_1$ sur l'axe des abscisses
  5. Répéter le processus

Exemple : Représente les premiers termes de la suite $(U_n)$ définie par $U_0 = 0$ et $U_{n+1} = \sqrt{2U_n + 1}$.

5. Suites arithmétiques

Définition : Une suite $(U_n)$ est arithmétique s'il existe un réel $r$ (la raison) tel que pour tout $n$, $U_{n+1} - U_n = r$.

Propriétés :
  • Terme général : $U_n = U_p + (n-p)r$ (en particulier $U_n = U_0 + nr$)
  • Somme des termes : $S_n = U_p + U_{p+1} + \cdots + U_n = \frac{(n-p+1)(U_p + U_n)}{2}$
  • Monotonie :
    • Si $r > 0$, la suite est croissante
    • Si $r < 0$, la suite est décroissante
    • Si $r = 0$, la suite est constante

Exercice : On considère la suite $(U_n)$ définie par $U_n = \frac{2n+1}{3}$.

  1. Calcule $U_0$, $U_1$, $U_2$
  2. Démontre que $(U_n)$ est arithmétique et exprime $U_n$ en fonction de $n$
  3. Calcule $S_{10} = U_0 + U_1 + \cdots + U_{10}$
  4. Étudie la monotonie de $(U_n)$

1. $U_0 = \frac{1}{3}$, $U_1 = 1$, $U_2 = \frac{5}{3}$

2. $U_{n+1} - U_n = \frac{2(n+1)+1}{3} - \frac{2n+1}{3} = \frac{2}{3}$ constante, donc suite arithmétique de raison $r = \frac{2}{3}$.

$U_n = U_0 + n \times r = \frac{1}{3} + n \times \frac{2}{3} = \frac{2n+1}{3}$

3. $S_{10} = \frac{(10+1)(U_0 + U_{10})}{2} = \frac{11 \times (\frac{1}{3} + \frac{21}{3})}{2} = \frac{11 \times \frac{22}{3}}{2} = \frac{242}{6} = \frac{121}{3}$

4. Comme $r = \frac{2}{3} > 0$, la suite est croissante.

6. Suites géométriques

Définition : Une suite $(V_n)$ est géométrique s'il existe un réel $q$ (la raison) tel que pour tout $n$, $V_{n+1} = q \times V_n$.

Propriétés :
  • Terme général : $V_n = V_p \times q^{n-p}$ (en particulier $V_n = V_0 \times q^n$)
  • Somme des termes : $S_n = V_p + V_{p+1} + \cdots + V_n = V_p \times \frac{1-q^{n-p+1}}{1-q}$ (si $q \neq 1$)
  • Monotonie :
    • Si $V_0 > 0$ et $q > 1$, la suite est croissante
    • Si $V_0 > 0$ et $0 < q < 1$, la suite est décroissante
    • Si $V_0 < 0$ et $q > 1$, la suite est décroissante
    • Si $V_0 < 0$ et $0 < q < 1$, la suite est croissante
    • Si $q < 0$, la suite n'est pas monotone

Exercice : On considère les suites $(U_n)$ et $(V_n)$ définies par :

$\begin{cases} U_0 = 1 \\ U_{n+1} = 2U_n - 3 \end{cases}$ et $V_n = U_n - 3$

  1. Calcule $U_1$, $U_2$, $U_3$
  2. Démontre que $(V_n)$ est géométrique et précise son premier terme et sa raison
  3. Donne l'expression de $V_n$ en fonction de $n$
  4. On pose $S_n = V_0 + V_1 + \cdots + V_n$. Donne l'expression de $S_n$ en fonction de $n$
  5. Étudie la monotonie de $(V_n)$

1. $U_1 = 2 \times 1 - 3 = -1$, $U_2 = 2 \times (-1) - 3 = -5$, $U_3 = 2 \times (-5) - 3 = -13$

2. $V_{n+1} = U_{n+1} - 3 = (2U_n - 3) - 3 = 2U_n - 6 = 2(U_n - 3) = 2V_n$

Donc $(V_n)$ est géométrique de raison $q = 2$ et de premier terme $V_0 = U_0 - 3 = -2$

3. $V_n = V_0 \times q^n = -2 \times 2^n$

4. $S_n = V_0 \times \frac{1-q^{n+1}}{1-q} = -2 \times \frac{1-2^{n+1}}{1-2} = -2 \times \frac{1-2^{n+1}}{-1} = 2(1-2^{n+1})$

5. Comme $V_0 = -2 < 0$ et $q = 2 > 1$, la suite $(V_n)$ est décroissante.

7. Problème concret - Comparaison de suites

Problème : Gervais lance une balle jusqu'à 20m. Après chaque rebond, elle perd 1/4 de sa hauteur. Après le cinquième rebond, le ballon pourra-t-il encore s'élever de 30cm?

Soit $U_n$ la hauteur après le n-ième rebond, avec $U_0 = 20$.

Après chaque rebond, la balle perd 1/4 de sa hauteur, donc elle conserve 3/4 de sa hauteur.

Ainsi, $U_{n+1} = \frac{3}{4}U_n$

La suite $(U_n)$ est géométrique de raison $q = \frac{3}{4}$ et de premier terme $U_0 = 20$.

Donc $U_n = 20 \times \left(\frac{3}{4}\right)^n$

Après le 5ème rebond : $U_5 = 20 \times \left(\frac{3}{4}\right)^5 = 20 \times \frac{243}{1024} = \frac{4860}{1024} \approx 4.75$ m

4.75 m > 0.3 m, donc le ballon pourra encore s'élever de 30 cm après le 5ème rebond.

Le petit-frère de Gervais a tort.

Problème : Une entreprise propose deux types de rémunération :

  • Type 1 : 120 000 F/mois avec augmentation annuelle de 10 000 F
  • Type 2 : 110 000 F/mois avec augmentation annuelle de 8%

Quelle est la rémunération la plus avantageuse sur 5 ans?

Type 1 (suite arithmétique) :

$U_0 = 120000$, raison $r = 10000$

$U_n = 120000 + 10000n$

Au bout de 5 ans (n=4) : $U_4 = 120000 + 10000 \times 4 = 160000$ F/mois

Masse salariale sur 5 ans : $S = 12 \times (U_0 + U_1 + U_2 + U_3 + U_4)$

$= 12 \times (5 \times 120000 + 10000 \times (0+1+2+3+4))$

$= 12 \times (600000 + 100000) = 12 \times 700000 = 8400000$ F

Type 2 (suite géométrique) :

$V_0 = 110000$, raison $q = 1.08$

$V_n = 110000 \times (1.08)^n$

Au bout de 5 ans (n=4) : $V_4 = 110000 \times (1.08)^4 \approx 110000 \times 1.3605 \approx 149655$ F/mois

Masse salariale sur 5 ans : $S = 12 \times (V_0 + V_1 + V_2 + V_3 + V_4)$

$= 12 \times 110000 \times (1 + 1.08 + 1.08^2 + 1.08^3 + 1.08^4)$

$= 1320000 \times \frac{1-1.08^5}{1-1.08} \approx 1320000 \times 5.8666 \approx 7740000$ F

Le type 1 est plus avantageux avec 8 400 000 F contre 7 740 000 F pour le type 2.
