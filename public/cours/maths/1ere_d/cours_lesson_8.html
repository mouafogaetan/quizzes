Leçon I : Effectifs et fréquences cumulées
Notions essentielles
  • Effectif cumulé croissant (ECC) : $ECC_k = n_1 + n_2 + \ldots + n_k$
  • Effectif cumulé décroissant (ECD) : $ECD_k = n_k + n_{k+1} + \ldots + n_p$
  • Fréquence cumulée croissante (FCC) : $FCC_k = f_1 + f_2 + \ldots + f_k$
  • Fréquence cumulée décroissante (FCD) : $FCD_k = f_k + f_{k+1} + \ldots + f_p$

Leçon II : Caractéristiques de position
Notions essentielles
  • Classe modale : Classe ayant le plus grand effectif
  • Mode : Centre de la classe modale
  • Moyenne : $\overline{X} = \frac{\sum_{i=1}^{p} n_i \times c_i}{N}$
  • Médiane : Valeur qui partage la série en deux parties égales
  • Classe médiane : Classe contenant la médiane
Exercice d'application

Calculer la moyenne, déterminer la classe modale, le mode et la médiane de la série suivante :

Notes [0;5] [5;8] [8;10] [10;12] [12;15] [15;20]
Effectifs 10 5 13 7 8 4

Voir la solution

Solution :

1. Calcul des centres de classes :

Notes [0;5] [5;8] [8;10] [10;12] [12;15] [15;20]
Centres (cᵢ) 2.5 6.5 9 11 13.5 17.5

2. Moyenne : $\overline{X} = \frac{10×2.5 + 5×6.5 + 13×9 + 7×11 + 8×13.5 + 4×17.5}{47} = \frac{429.5}{47} = 9.14$

3. Classe modale : [8;10] (effectif le plus élevé = 13)

4. Mode : Centre de la classe modale = 9

5. Médiane : La moitié de l'effectif total est 23.5. La classe médiane est [8;10] car son ECC (28) est la première valeur ≥ 23.5.

Leçon III : Caractéristiques de dispersion
Notions essentielles
  • Écart moyen : $e_m = \frac{\sum n_i|c_i - \overline{X}|}{N}$
  • Variance : $V = \frac{\sum n_i(c_i - \overline{X})^2}{N}$ ou $V = \frac{\sum n_i c_i^2}{N} - \overline{X}^2$ (formule de Koenig)
  • Écart type : $\sigma = \sqrt{V}$
Exercice d'application

Calculer l'écart moyen, la variance et l'écart type de la série suivante :

Classe [0;2] [2;4] [4;6] [6;8] [8;10] [10;12]
Effectifs 19 21 15 25 8 7

Voir la solution

Solution :

1. Calcul des centres et autres valeurs nécessaires :

Classe [0;2] [2;4] [4;6] [6;8] [8;10] [10;12] Total
Centres (cᵢ) 1 3 5 7 9 11 -
nᵢ × cᵢ 19 63 75 175 72 77 481
nᵢ × cᵢ² 19 189 375 1225 648 847 3303

2. Moyenne : $\overline{X} = \frac{481}{95} = 5.06$

3. Écart moyen : $e_m = \frac{19|1-5.06| + 21|3-5.06| + 15|5-5.06| + 25|7-5.06| + 8|9-5.06| + 7|11-5.06|}{95} = \frac{243.5}{95} = 2.56$

4. Variance (formule de Koenig) : $V = \frac{3303}{95} - (5.06)^2 = 34.77 - 25.60 = 9.17$

5. Écart type : $\sigma = \sqrt{9.17} = 3.03$

Leçon IV : Représentations graphiques
Notions essentielles
  • Histogramme : Diagramme avec rectangles dont les bases sont proportionnelles aux amplitudes et les hauteurs proportionnelles aux effectifs (si amplitudes égales) ou aux densités (si amplitudes différentes)
  • Polygone des effectifs : Ligne brisée joignant les centres des classes
  • Courbes cumulatives :
    • ECC : points (borne supérieure, ECC)
    • ECD : points (borne inférieure, ECD)
  • L'abscisse du point d'intersection des courbes ECC et ECD donne la médiane
