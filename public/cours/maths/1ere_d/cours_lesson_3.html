TRIGONOMÉTRIE

Formules trigonométriques et résolution d'équations

Motivation

De nombreuses situations de la vie courante, telles que la détermination des dimensions d'un terrain, l'achat ou la vente de biens de consommation, ou les positions d'objets mobiles, font appel à la notion de trigonométrie. Ce chapitre nous donnera les outils nécessaires pour résoudre ces problèmes.

Cercle trigonométrique avec les angles remarquables

Objectifs pédagogiques
  • Déterminer les lignes trigonométriques des angles orientés
  • Reconnaître et utiliser les formules trigonométriques
Situation problème

Monsieur Adam est un jeune cultivateur. Il veut réparer les rayons des roues de son porte-tout (communément appelé "pousse-pousse") chez son mécanicien Kouam. Le mécanicien lui conseille qu'afin d'éviter les pertes de rayons en cas de surcharge, il devrait placer les rayons des roues de telle sorte que l'angle entre deux rayons consécutifs soit compris entre 5° et 10°. Les roues de monsieur Adam peuvent-elles supporter la surcharge ?

Lignes trigonométriques d'angles orientés associés

Soit $\theta$ un angle orienté. Les angles orientés $-\theta$, $\pi - \theta$, $\pi + \theta$, $\frac{\pi}{2} - \theta$ et $\frac{\pi}{2} + \theta$ sont dits associés à $\theta$ et on a les formules suivantes :

$\sin(-\theta) = -\sin\theta$

$\cos(-\theta) = \cos\theta$

$\tan(-\theta) = -\tan\theta$

$\sin(\pi - \theta) = \sin\theta$

$\cos(\pi - \theta) = -\cos\theta$

$\tan(\pi - \theta) = -\tan\theta$

$\sin(\pi + \theta) = -\sin\theta$

$\cos(\pi + \theta) = -\cos\theta$

$\tan(\pi + \theta) = \tan\theta$

Exemple :

Simplifier l'expression : $\sin(\pi - \theta) + \cos\left(\frac{\pi}{2} + \theta\right) - \sin(-\theta) + \cos(\pi + \theta)$

Formules trigonométriques
Formules d'addition

$\sin(a + b) = \sin a \cos b + \cos a \sin b$

$\sin(a - b) = \sin a \cos b - \cos a \sin b$

$\cos(a + b) = \cos a \cos b - \sin a \sin b$

$\cos(a - b) = \cos a \cos b + \sin a \sin b$

$\tan(a + b) = \frac{\tan a + \tan b}{1 - \tan a \tan b}$

$\tan(a - b) = \frac{\tan a - \tan b}{1 + \tan a \tan b}$

Formules de duplication

$\sin(2a) = 2 \sin a \cos a$

$\cos(2a) = \cos^2 a - \sin^2 a = 2\cos^2 a - 1 = 1 - 2\sin^2 a$

$\tan(2a) = \frac{2\tan a}{1 - \tan^2 a}$

Formules de linéarisation

$\sin^2 a = \frac{1 - \cos(2a)}{2}$

$\cos^2 a = \frac{1 + \cos(2a)}{2}$

$\sin^3 a = \frac{3\sin a - \sin(3a)}{4}$

$\cos^3 a = \frac{3\cos a + \cos(3a)}{4}$

Exemple :

Calculer la valeur exacte de $\sin(75^\circ)$ et $\cos(15^\circ)$ en utilisant les formules d'addition.

Situation problème

Pour régler ses problèmes de pertes de rayons en cas de surcharge, Adam consulte un autre mécanicien qui lui conseille de placer les rayons des roues de son porte-tout de telle sorte que l'angle formé par deux rayons consécutifs soit dans l'intervalle $[5^\circ, 10^\circ]$ et vérifie la relation $\cos(2\theta) = \frac{\sqrt{3}}{2}$. Très embarrassé, Adam ignore comment procéder. Aidez Adam à déterminer l'angle entre deux rayons consécutifs de son porte-tout.

Résolution d'équations trigonométriques
Équations du type $\cos x = a$

Soit à résoudre l'équation $\cos x = a$ :

  • Si $|a| > 1$, alors l'équation n'a pas de solution
  • Si $|a| \leq 1$, alors on cherche un angle $\theta \in [0, \pi]$ tel que $\cos \theta = a$

Les solutions sont alors : $x = \theta + 2k\pi$ ou $x = -\theta + 2k\pi$, $k \in \mathbb{Z}$

Exemple :

Résoudre $\cos x = \frac{1}{2}$

Solution : $x = \frac{\pi}{3} + 2k\pi$ ou $x = -\frac{\pi}{3} + 2k\pi$, $k \in \mathbb{Z}$

Équations du type $\sin x = a$

Soit à résoudre l'équation $\sin x = a$ :

  • Si $|a| > 1$, alors l'équation n'a pas de solution
  • Si $|a| \leq 1$, alors on cherche un angle $\theta \in [-\frac{\pi}{2}, \frac{\pi}{2}]$ tel que $\sin \theta = a$

Les solutions sont alors : $x = \theta + 2k\pi$ ou $x = \pi - \theta + 2k\pi$, $k \in \mathbb{Z}$

Exemple :

Résoudre $\sin x = \frac{\sqrt{2}}{2}$

Solution : $x = \frac{\pi}{4} + 2k\pi$ ou $x = \frac{3\pi}{4} + 2k\pi$, $k \in \mathbb{Z}$

Équations du type $a \cos x + b \sin x = c$

Pour résoudre ce type d'équation, on peut :

  1. Calculer $R = \sqrt{a^2 + b^2}$
  2. Diviser l'équation par $R$ : $\frac{a}{R} \cos x + \frac{b}{R} \sin x = \frac{c}{R}$
  3. Identifier $\frac{a}{R} = \cos \phi$ et $\frac{b}{R} = \sin \phi$
  4. L'équation devient $\cos(x - \phi) = \frac{c}{R}$
  5. Résoudre cette dernière équation
Exemple :

Résoudre $\sqrt{3} \cos x + \sin x = 1$

Résolution d'inéquations trigonométriques

Pour résoudre une inéquation trigonométrique :

  1. Résoudre l'équation trigonométrique associée
  2. Placer les solutions sur le cercle trigonométrique
  3. Déterminer les intervalles solutions en étudiant le signe
Exemple :

Résoudre $\sin x \geq \frac{1}{2}$ sur $[0, 2\pi]$

Solution : $x \in \left[\frac{\pi}{6}, \frac{5\pi}{6}\right]$

Résolution graphique

Pour résoudre graphiquement une équation ou inéquation trigonométrique :

  • Tracer le cercle trigonométrique et les droites correspondantes
  • Identifier les points d'intersection
  • Déterminer les angles correspondants
