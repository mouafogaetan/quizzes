Trigonométrie

Formules de transformation et applications pratiques

Introduction à la Trigonométrie

La trigonométrie offre des outils essentiels pour calculer des longueurs dans les triangles rectangles. Ses applications s'étendent à de nombreux domaines comme la cartographie, la construction, la topographie et la physique.

Application pratique : Problème de topographie

Un topographe doit déterminer la hauteur $h$ d'une colline. Il effectue deux mesures d'angles ($\alpha = 29^\circ$ et $\beta = 27^\circ$) à partir de deux points distants de 10 mètres. Le manuel indique la formule :

$h = l \times \frac{\sin\alpha \sin\beta}{\sin(\beta-\alpha)}$

Comment calculer $h$ sans connaître directement $\sin(\beta-\alpha)$ ?

Formules de transformation trigonométriques

Objectifs d'apprentissage

  • Exprimer $\cos(a \pm b)$ et $\sin(a \pm b)$ en fonction de $\cos a$, $\cos b$, $\sin a$ et $\sin b$
  • Exprimer $\tan(a + b)$ en fonction de $\tan a$ et $\tan b$
  • Calculer des valeurs exactes de cosinus, sinus et tangente pour certains angles remarquables

Formules fondamentales

Pour tous réels $\alpha$ et $\beta$ :

$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$

$\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$

$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$

$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$

Formules de duplication

Pour tout réel $\alpha$ :

$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha - 1 = 1 - 2\sin^2\alpha$

$\sin 2\alpha = 2\sin\alpha\cos\alpha$

Formules de linéarisation

Pour tout réel $\alpha$ :

$\cos^2\alpha = \frac{1 + \cos 2\alpha}{2}$

$\sin^2\alpha = \frac{1 - \cos 2\alpha}{2}$

Exemple d'application

Calcul de $\cos\left(\frac{\pi}{12}\right)$ :

On remarque que $\frac{\pi}{6} = 2 \times \frac{\pi}{12}$, donc :

$\cos\left(\frac{\pi}{6}\right) = 2\cos^2\left(\frac{\pi}{12}\right) - 1$

$\frac{\sqrt{3}}{2} = 2\cos^2\left(\frac{\pi}{12}\right) - 1$

$\cos^2\left(\frac{\pi}{12}\right) = \frac{\sqrt{3} + 2}{4}$

Comme $\frac{\pi}{12} \in \left]0; \frac{\pi}{2}\right[$, on a $\cos\left(\frac{\pi}{12}\right) > 0$, donc :

$\cos\left(\frac{\pi}{12}\right) = \frac{\sqrt{2} + \sqrt{6}}{4}$

Rappels des relations trigonométriques

Relations fondamentales

Pour tout réel $x$ :

$\cos^2 x + \sin^2 x = 1$

$\cos(-x) = \cos x$

$\sin(-x) = -\sin x$

$\cos(x + \pi) = -\cos x$

$\sin(x + \pi) = -\sin x$

$\cos\left(\frac{\pi}{2} - x\right) = \sin x$

$\sin\left(\frac{\pi}{2} - x\right) = \cos x$

Relations concernant la tangente

Pour $x \neq \frac{\pi}{2} + k\pi$ ($k \in \mathbb{Z}$) :

$\tan x = \frac{\sin x}{\cos x}$

$\tan(x + \pi k) = \tan x$

$1 + \tan^2 x = \frac{1}{\cos^2 x}$

$\tan(x + y) = \frac{\tan x + \tan y}{1 - \tan x \tan y}$

$\tan(2x) = \frac{2\tan x}{1 - \tan^2 x}$

$\tan(-x) = -\tan x$

Valeurs remarquables

$\alpha$ $0$ $\frac{\pi}{6}$ $\frac{\pi}{4}$ $\frac{\pi}{3}$ $\frac{\pi}{2}$
$\cos\alpha$ $1$ $\frac{\sqrt{3}}{2}$ $\frac{\sqrt{2}}{2}$ $\frac{1}{2}$ $0$
$\sin\alpha$ $0$ $\frac{1}{2}$ $\frac{\sqrt{2}}{2}$ $\frac{\sqrt{3}}{2}$ $1$
$\tan\alpha$ $0$ $\frac{\sqrt{3}}{3}$ $1$ $\sqrt{3}$ Non définie

Équations trigonométriques

Problème pratique

Un électricien doit déterminer quand la tension d'une batterie atteint 10 volts. La relation entre la tension et le temps $t$ (en secondes) est :

$u(t) = 10\sqrt{2} \sin t$

Pour quelles valeurs de $t$ dans l'intervalle $\left[0, \frac{\pi}{2}\right]$ la tension vaut-elle 10 volts ?

Résolution d'équations trigonométriques

Équation $\sin x = a$

Si $a \in [-1, 1]$, alors il existe $\beta$ tel que $\sin\beta = a$ :

$\sin x = a \Leftrightarrow \sin x = \sin\beta \Leftrightarrow \begin{cases} x = \beta + 2k\pi \\ x = \pi - \beta + 2k\pi \end{cases} $, avec $k \in \mathbb{Z}$

Si $a \notin [-1, 1]$, l'équation n'a pas de solution.

Équation $\cos x = a$

Si $a \in [-1, 1]$, alors il existe $\beta$ tel que $\cos\beta = a$ :

$\cos x = a \Leftrightarrow \cos x = \cos\beta \Leftrightarrow \begin{cases} x = \beta + 2k\pi \\ x = -\beta + 2k\pi \end{cases} $, avec $k \in \mathbb{Z}$

Si $a \notin [-1, 1]$, l'équation n'a pas de solution.

Équation $\tan x = a$

Pour tout $a \in \mathbb{R}$, il existe $\beta$ tel que $\tan\beta = a$ :

$\tan x = a \Leftrightarrow \tan x = \tan\beta \Leftrightarrow x = \beta + k\pi$, avec $k \in \mathbb{Z}$

Équation $a\cos x + b\sin x = c$

On pose $r = \sqrt{a^2 + b^2}$

L'équation devient : $\frac{a}{r}\cos x + \frac{b}{r}\sin x = \frac{c}{r}$

Il existe $\mu$ tel que $\cos\mu = \frac{a}{r}$ et $\sin\mu = \frac{b}{r}$

L'équation se transforme alors en : $\cos(x - \mu) = \frac{c}{r}$

Si $\left|\frac{c}{r}\right| \leq 1$, il existe des solutions.

Si $\left|\frac{c}{r}\right| > 1$, il n'y a pas de solution.

Exemple de résolution

Résoudre $\cos x + \sin x = \sqrt{6}$ :

$r = \sqrt{1^2 + 1^2} = \sqrt{2}$

$\frac{c}{r} = \frac{\sqrt{6}}{\sqrt{2}} = \sqrt{3} \notin [-1, 1]$

Donc l'équation n'a pas de solution dans $\mathbb{R}$.

Résolution d'inéquations trigonométriques

Inéquations avec sinus et cosinus

Pour résoudre $\sin x < a$, $\sin x \geq a$, $\cos x < a$ ou $\cos x \geq a$ :

Si $a \notin [-1, 1]$, l'ensemble des solutions est soit vide soit $\mathbb{R}$.

Si $a \in [-1, 1]$, on utilise le cercle trigonométrique pour déterminer les intervalles solutions.

Inéquations avec tangente

Pour résoudre $\tan x < a$ ou $\tan x \geq a$ :

On utilise la droite tangente au cercle trigonométrique pour déterminer les intervalles solutions.

Exemple de résolution

Résoudre $\sin(2x) > -\frac{1}{2}$ :

On pose $X = 2x$, donc $\sin X > -\frac{1}{2}$

$\sin X = -\frac{1}{2} \Leftrightarrow X = -\frac{\pi}{6} + 2k\pi$ ou $X = \frac{7\pi}{6} + 2k\pi$, $k \in \mathbb{Z}$

Sur un intervalle de longueur $2\pi$, la solution est $-\frac{\pi}{6} < X < \frac{7\pi}{6}$

Donc $-\frac{\pi}{6} + 2k\pi < 2x < \frac{7\pi}{6} + 2k\pi$

Soit $-\frac{\pi}{12} + k\pi < x < \frac{7\pi}{12} + k\pi$, $k \in \mathbb{Z}$
