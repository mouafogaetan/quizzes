Introduction aux nombres complexes

Les nombres complexes constituent une extension des nombres réels qui permet de résoudre des équations polynomiales qui n'ont pas de solutions réelles, comme $x^2 + 1 = 0$.

Ils sont définis comme des couples de nombres réels, notés $z = a + ib$, où :

  • $a$ est la partie réelle (notée $\text{Re}(z)$)
  • $b$ est la partie imaginaire (notée $\text{Im}(z)$)
  • $i$ est l'unité imaginaire vérifiant $i^2 = -1$
$\mathbb{N}$ ⊂ $\mathbb{Z}$ ⊂ $\mathbb{D}$ ⊂ $\mathbb{Q}$ ⊂ $\mathbb{R}$ ⊂ $\mathbb{C}$
Motivation et applications

Les nombres complexes ne sont pas qu'une abstraction mathématique. Ils ont de nombreuses applications pratiques :

  • Phénomènes ondulatoires : Les complexes permettent de modéliser facilement une grandeur angulaire (phase) et une grandeur scalaire (intensité)
  • Physique : Mécanique (oscillations), électronique (circuits résonnants)
  • Ondes et optique : Radars, laser, radio, interférences
  • Mécanique quantique : Fonction d'onde complexe
Problème concret :

Un champ rectangulaire a un périmètre de 16 m et une superficie de 40 m². Quelles sont ses dimensions?

En notant $x$ et $y$ les dimensions, on obtient le système :

$\begin{cases} x + y = 8 \\ xy = 40 \end{cases}$

L'équation du second degré correspondante $t^2 - 8t + 40 = 0$ a un discriminant négatif ($\Delta = -96$).

Seuls les nombres complexes permettent de résoudre formellement cette équation.

Forme algébrique d'un nombre complexe
Définition :

Un nombre complexe $z$ s'écrit sous forme algébrique $z = a + ib$, où :

  • $a$ et $b$ sont des nombres réels
  • $a$ est la partie réelle de $z$, notée $\text{Re}(z)$
  • $b$ est la partie imaginaire de $z$, notée $\text{Im}(z)$
  • $i$ est l'unité imaginaire vérifiant $i^2 = -1$
Propriétés :
  • Un nombre complexe est réel si et seulement si sa partie imaginaire est nulle
  • Un nombre complexe est imaginaire pur si et seulement si sa partie réelle est nulle
  • Deux nombres complexes sont égaux si et seulement s'ils ont mêmes parties réelle et imaginaire
Exemples :

$\text{Re}(3 + 2i) = 3$, $\text{Im}(3 + 2i) = 2$

$\text{Re}(-5i) = 0$, $\text{Im}(-5i) = -5$ (imaginaire pur)

$\text{Re}(7) = 7$, $\text{Im}(7) = 0$ (réel)

Opérations dans C

Soient $z = a + ib$ et $z' = a' + ib'$ deux nombres complexes.

Addition et soustraction :

$z + z' = (a + a') + i(b + b')$

$z - z' = (a - a') + i(b - b')$

Multiplication :

$z \times z' = (aa' - bb') + i(ab' + a'b)$

En particulier, pour tout réel $\lambda$ : $\lambda z = \lambda a + i\lambda b$

Inverse et division :

Si $z \neq 0$, alors $\frac{1}{z} = \frac{a}{a^2 + b^2} - i\frac{b}{a^2 + b^2}$

$\frac{z}{z'} = z \times \frac{1}{z'} = \frac{aa' + bb'}{a'^2 + b'^2} + i\frac{a'b - ab'}{a'^2 + b'^2}$ (pour $z' \neq 0$)

Exemple :

Soient $z_1 = 2 + 5i$ et $z_2 = 1 - 3i$

$z_1 + z_2 = (2 + 1) + i(5 - 3) = 3 + 2i$

$z_1 \times z_2 = (2 \times 1 - 5 \times (-3)) + i(2 \times (-3) + 5 \times 1) = 17 - i$

$\frac{z_1}{z_2} = \frac{2 + 5i}{1 - 3i} = \frac{(2 + 5i)(1 + 3i)}{(1 - 3i)(1 + 3i)} = \frac{-13 + 11i}{10} = -\frac{13}{10} + \frac{11}{10}i$

Conjugué d'un nombre complexe
Définition :

Soit $z = a + ib$ un nombre complexe. On appelle conjugué de $z$ le nombre complexe $\bar{z} = a - ib$.

Propriétés du conjugué :
  • $\overline{z + z'} = \bar{z} + \bar{z'}$
  • $\overline{z \times z'} = \bar{z} \times \bar{z'}$
  • $\overline{\left(\frac{z}{z'}\right)} = \frac{\bar{z}}{\bar{z'}}$ (pour $z' \neq 0$)
  • $\overline{\alpha z} = \alpha \bar{z}$ (pour $\alpha$ réel)
  • $\overline{(\bar{z})} = z$
  • $z + \bar{z} = 2\text{Re}(z)$
  • $z - \bar{z} = 2i\text{Im}(z)$
  • $z \times \bar{z} = a^2 + b^2 = |z|^2$
Exemple :

Soit $z = 3 + 4i$, alors $\bar{z} = 3 - 4i$

$z + \bar{z} = 6 = 2 \times \text{Re}(z)$

$z - \bar{z} = 8i = 2i \times \text{Im}(z)$

$z \times \bar{z} = 9 + 16 = 25 = |z|^2$

Module d'un nombre complexe
Définition :

Soit $z = a + ib$ un nombre complexe. On appelle module de $z$ le nombre réel positif $|z| = \sqrt{a^2 + b^2}$.

Propriétés du module :
  • $|z| = 0$ si et seulement si $z = 0$
  • $|z| = |\bar{z}| = |-z|$
  • $|z \times z'| = |z| \times |z'|$
  • $\left|\frac{z}{z'}\right| = \frac{|z|}{|z'|}$ (pour $z' \neq 0$)
  • $|z + z'| \leq |z| + |z'|$ (inégalité triangulaire)
  • $||z| - |z'|| \leq |z - z'|$
  • $|z|^2 = z \times \bar{z}$
Exemple :

$|3 + 4i| = \sqrt{9 + 16} = 5$

$|1 - i| = \sqrt{1 + 1} = \sqrt{2}$

$|5i| = \sqrt{0 + 25} = 5$

$|-7| = \sqrt{49 + 0} = 7$

Interprétation géométrique :

Si on représente le nombre complexe $z = a + ib$ par le point $M(a, b)$ dans le plan, alors $|z|$ représente la distance entre l'origine $O$ et le point $M$.

Équations du second degré dans C
Racines carrées d'un nombre complexe :

On appelle racine carrée d'un nombre complexe $z$ tout nombre complexe $\delta$ tel que $\delta^2 = z$.

Tout nombre complexe non nul admet exactement deux racines carrées opposées.

Résolution des équations du second degré :

Soit l'équation $az^2 + bz + c = 0$ avec $a \neq 0$.

On calcule le discriminant $\Delta = b^2 - 4ac$.

  • Si $\Delta \geq 0$ : solutions réelles $z = \frac{-b \pm \sqrt{\Delta}}{2a}$
  • Si $\Delta < 0$ : solutions complexes conjuguées $z = \frac{-b \pm i\sqrt{-\Delta}}{2a}$
Exemple :

Résoudre $z^2 - 6z + 13 = 0$

$\Delta = (-6)^2 - 4 \times 1 \times 13 = 36 - 52 = -16$

Les solutions sont : $z = \frac{6 \pm i\sqrt{16}}{2} = \frac{6 \pm 4i}{2} = 3 \pm 2i$

Relations entre coefficients et racines :

Si $z_1$ et $z_2$ sont les racines de $az^2 + bz + c = 0$, alors :

$z_1 + z_2 = -\frac{b}{a}$ et $z_1 \times z_2 = \frac{c}{a}$
