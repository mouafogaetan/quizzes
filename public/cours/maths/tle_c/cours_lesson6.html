Chapitre 6 : Nombres Complexes - Approche Géométrique

Représentation plane, forme trigonométrique et exponentielle

Introduction aux nombres complexes

Motivation

Les nombres complexes permettent de "compléter" les nombres réels de sorte que tout nombre ait une racine carrée. Ils offrent également un outil puissant pour modéliser les phénomènes ondulatoires et oscillatoires en physique et en ingénierie.

Applications des nombres complexes
  • Mécanique (oscillations)
  • Électronique (circuits résonnants)
  • Optique (ondes, interférences)
  • Mécanique quantique (fonction d'onde)
  • Traitement du signal

Leçon 1 : Représentation plane

Correspondance entre nombres complexes et points du plan

Soit un repère orthonormé $(O; \vec{e}_1; \vec{e}_2)$ du plan.

  • À tout nombre complexe $z = a + ib$, on associe le point $M(a; b)$.
  • Réciproquement, à tout point $M(a; b)$, on associe le nombre complexe $z = a + ib$.
Terminologie :
  • $z$ est appelé l'affixe du point $M$ ou du vecteur $\overrightarrow{OM}$
  • Le plan est appelé plan complexe
  • L'axe des abscisses est l'axe des réels
  • L'axe des ordonnées est l'axe des imaginaires purs

Propriétés des affixes

Soient $\vec{u}(z)$ et $\vec{v}(z')$ deux vecteurs et $A(z_A)$, $B(z_B)$ deux points du plan complexe.

  • Le vecteur $\vec{u} + \vec{v}$ a pour affixe $z + z'$
  • Le vecteur $k \times \vec{u}$ a pour affixe $k \times z$
  • Le vecteur $\overrightarrow{AB}$ a pour affixe $z_B - z_A$
  • Le milieu du segment $[AB]$ a pour affixe $\frac{z_A + z_B}{2}$

Module et distance

Soit $z = a + ib$ un nombre complexe. Le module de $z$ est défini par $|z| = \sqrt{a^2 + b^2}$.

Si $A(z_A)$ et $B(z_B)$ sont deux points du plan complexe, alors la distance $AB$ est donnée par :

$AB = |z_B - z_A|$

Exemple :

Soient $A(3+2i)$, $B(4+3i)$ et $C(5+i)$.

Calculer $AB$ et $|z_B - z_A|$ :

$z_B - z_A = (4+3i) - (3+2i) = 1 + i$

$|z_B - z_A| = \sqrt{1^2 + 1^2} = \sqrt{2}$

$AB = \sqrt{(4-3)^2 + (3-2)^2} = \sqrt{1^2 + 1^2} = \sqrt{2}$

On vérifie bien que $AB = |z_B - z_A|$.

Lieux géométriques

Soit $A(z_A)$ un point du plan complexe et $R$ un réel strictement positif.

L'ensemble des points $M(z)$ tels que $|z - z_A| = R$ est le cercle de centre $A$ et de rayon $R$.

Exemple :

Soit $A$ le point d'affixe $z_A = 1 + i$.

Montrer que $|z - z_A| = 2 \Leftrightarrow (x - 1)^2 + (y - 1)^2 = 4$

En posant $z = x + iy$, on a :

$|z - z_A| = |(x + iy) - (1 + i)| = |(x - 1) + i(y - 1)| = \sqrt{(x-1)^2 + (y-1)^2}$

Ainsi, $|z - z_A| = 2 \Leftrightarrow \sqrt{(x-1)^2 + (y-1)^2} = 2 \Leftrightarrow (x-1)^2 + (y-1)^2 = 4$

Il s'agit de l'équation du cercle de centre $A(1, 1)$ et de rayon 2.

Exercices d'application

Exercice 1

Dans le plan complexe, on considère les points $A(2i)$, $B(1-i)$ et $C(3)$.

  1. Déterminer l'affixe des vecteurs $\overrightarrow{AB}$, $\overrightarrow{BA}$ et $3\overrightarrow{AB} - \overrightarrow{BC}$.
  2. Déterminer l'affixe du point $D$ tel que $ABCD$ soit un parallélogramme.

Solution :

  1. Affixe de $\overrightarrow{AB} = z_B - z_A = (1-i) - (2i) = 1 - 3i$

    Affixe de $\overrightarrow{BA} = z_A - z_B = (2i) - (1-i) = -1 + 3i$

    Affixe de $\overrightarrow{BC} = z_C - z_B = 3 - (1-i) = 2 + i$

    Affixe de $3\overrightarrow{AB} - \overrightarrow{BC} = 3(1-3i) - (2+i) = 3-9i-2-i = 1-10i$

  2. Pour que $ABCD$ soit un parallélogramme, il faut que $\overrightarrow{AB} = \overrightarrow{DC}$

    Soit $z_D$ l'affixe de $D$, alors $z_C - z_D = z_B - z_A$

    $3 - z_D = (1-i) - (2i) = 1 - 3i$

    Donc $z_D = 3 - (1-3i) = 2 + 3i$

Leçon 2 : Forme trigonométrique d'un nombre complexe

Argument d'un nombre complexe

Soit $z \in \mathbb{C}^*$ et $M$ son image dans le plan complexe muni d'un repère orthonormé $(O; \vec{u}; \vec{v})$.

On appelle argument de $z$, noté $\arg(z)$, toute mesure de l'angle orienté $(\vec{u}; \overrightarrow{OM})$.

Propriétés :

  • Si $\theta$ est un argument de $z$, alors $\theta + 2k\pi$ ($k \in \mathbb{Z}$) est aussi un argument de $z$
  • $z \in \mathbb{R}^*_+ \Leftrightarrow \arg(z) \equiv 0 \pmod{2\pi}$
  • $z \in \mathbb{R}^*_- \Leftrightarrow \arg(z) \equiv \pi \pmod{2\pi}$
  • $z \in i\mathbb{R}^*$ avec $\Im(z) > 0 \Leftrightarrow \arg(z) \equiv \frac{\pi}{2} \pmod{2\pi}$
  • $z \in i\mathbb{R}^*$ avec $\Im(z) < 0 \Leftrightarrow \arg(z) \equiv -\frac{\pi}{2} \pmod{2\pi}$

Forme trigonométrique

Soit $z \in \mathbb{C}^*$ de module $r$ et d'argument $\theta$. La forme trigonométrique de $z$ est :

$z = r(\cos \theta + i \sin \theta)$

Exemple :

Soit $z = -\sqrt{2} + i\sqrt{2}$. Déterminer sa forme trigonométrique.

$|z| = \sqrt{(-\sqrt{2})^2 + (\sqrt{2})^2} = \sqrt{2+2} = 2$

Soit $\theta$ un argument de $z$ :

$\cos \theta = \frac{\Re(z)}{|z|} = \frac{-\sqrt{2}}{2} = -\frac{\sqrt{2}}{2}$

$\sin \theta = \frac{\Im(z)}{|z|} = \frac{\sqrt{2}}{2} = \frac{\sqrt{2}}{2}$

Donc $\theta = \frac{3\pi}{4}$ (modulo $2\pi$)

Ainsi, $z = 2\left(\cos \frac{3\pi}{4} + i \sin \frac{3\pi}{4}\right)$

Propriétés des arguments

Soient $z$ et $z'$ deux nombres complexes non nuls.

  • $\arg(z \times z') \equiv \arg(z) + \arg(z') \pmod{2\pi}$
  • $\arg\left(\frac{z}{z'}\right) \equiv \arg(z) - \arg(z') \pmod{2\pi}$
  • $\arg(-z) \equiv \arg(z) + \pi \pmod{2\pi}$
  • $\arg(\overline{z}) \equiv -\arg(z) \pmod{2\pi}$
  • $\arg(z^n) \equiv n \times \arg(z) \pmod{2\pi}$ pour tout $n \in \mathbb{N}$

Interprétation géométrique

Soient $A(z_A)$, $B(z_B)$ et $C(z_C)$ trois points du plan complexe, avec $z_A \neq z_B \neq z_C$.

On a : $\arg\left(\frac{z_B - z_C}{z_A - z_C}\right) \equiv (\overrightarrow{CA}; \overrightarrow{CB}) \pmod{2\pi}$

Exercices d'application

Exercice 2
  1. Déterminer un argument de $z = \frac{(1-i)^3}{(i+\sqrt{3})^2}$
  2. Soit $A(1+i)$, $B(1-i)$ et $C\left(\frac{8}{5} + \frac{4}{5}i\right)$
  3. Déterminer la mesure de l'angle $(\overrightarrow{CA}; \overrightarrow{CB})$. En déduire la nature du triangle $ABC$.

Solution :

  1. Calculons d'abord les arguments de $(1-i)$ et $(i+\sqrt{3})$ :

    $\arg(1-i) = -\frac{\pi}{4}$ (car dans le 4ème quadrant)

    $\arg(i+\sqrt{3}) = \frac{\pi}{6}$ (car $\tan \theta = \frac{1}{\sqrt{3}} = \frac{\sqrt{3}}{3}$)

    $\arg((1-i)^3) = 3 \times (-\frac{\pi}{4}) = -\frac{3\pi}{4}$

    $\arg((i+\sqrt{3})^2) = 2 \times \frac{\pi}{6} = \frac{\pi}{3}$

    $\arg(z) = \arg((1-i)^3) - \arg((i+\sqrt{3})^2) = -\frac{3\pi}{4} - \frac{\pi}{3} = -\frac{9\pi}{12} - \frac{4\pi}{12} = -\frac{13\pi}{12}$

    On peut simplifier en ajoutant $2\pi$ : $-\frac{13\pi}{12} + 2\pi = \frac{11\pi}{12}$

    Donc $\arg(z) \equiv \frac{11\pi}{12} \pmod{2\pi}$

  2. Calculons $\frac{z_B - z_C}{z_A - z_C}$ :

    $z_B - z_C = (1-i) - \left(\frac{8}{5} + \frac{4}{5}i\right) = -\frac{3}{5} - \frac{9}{5}i$

    $z_A - z_C = (1+i) - \left(\frac{8}{5} + \frac{4}{5}i\right) = -\frac{3}{5} + \frac{1}{5}i$

    $\frac{z_B - z_C}{z_A - z_C} = \frac{-\frac{3}{5} - \frac{9}{5}i}{-\frac{3}{5} + \frac{1}{5}i} = \frac{-3 - 9i}{-3 + i}$

    Multiplions numérateur et dénominateur par le conjugué du dénominateur :

    $\frac{(-3 - 9i)(-3 - i)}{(-3 + i)(-3 - i)} = \frac{(9 + 3i + 27i + 9i^2)}{9 + 1} = \frac{(9 + 30i - 9)}{10} = \frac{30i}{10} = 3i$

    $\arg(3i) = \frac{\pi}{2}$, donc $(\overrightarrow{CA}; \overrightarrow{CB}) = \frac{\pi}{2}$

    L'angle $\widehat{ACB}$ est droit, donc le triangle $ABC$ est rectangle en $C$.

Leçon 3 : Forme exponentielle d'un nombre complexe

Notation exponentielle

Pour tout $\theta \in \mathbb{R}$, on pose :

$e^{i\theta} = \cos \theta + i \sin \theta$

Propriétés :

  • $|e^{i\theta}| = 1$
  • $\arg(e^{i\theta}) \equiv \theta \pmod{2\pi}$

Forme exponentielle

Soit $z \in \mathbb{C}^*$ de module $r$ et d'argument $\theta$. La forme exponentielle de $z$ est :

$z = r e^{i\theta}$

Exemple :

Soit $z = 1 - i\sqrt{3}$. Donner sa forme exponentielle.

$|z| = \sqrt{1^2 + (-\sqrt{3})^2} = \sqrt{1 + 3} = 2$

$\cos \theta = \frac{1}{2}$, $\sin \theta = -\frac{\sqrt{3}}{2}$

Donc $\theta = -\frac{\pi}{3}$ (modulo $2\pi$)

Ainsi, $z = 2e^{-i\frac{\pi}{3}}$

Propriétés de l'exponentielle complexe

Pour tout $\theta, \theta' \in \mathbb{R}$ et pour tout $n \in \mathbb{Z}$ :

  • $e^{i\theta} \times e^{i\theta'} = e^{i(\theta + \theta')}$
  • $\frac{e^{i\theta}}{e^{i\theta'}} = e^{i(\theta - \theta')}$
  • $\frac{1}{e^{i\theta}} = e^{-i\theta}$
  • $(e^{i\theta})^n = e^{in\theta}$

Formule de Moivre

Pour tout $\theta \in \mathbb{R}$ et tout $n \in \mathbb{Z}$ :

$(\cos \theta + i \sin \theta)^n = \cos n\theta + i \sin n\theta$

Formules d'Euler

Pour tout $\theta \in \mathbb{R}$ :

$\cos \theta = \frac{e^{i\theta} + e^{-i\theta}}{2}$

$\sin \theta = \frac{e^{i\theta} - e^{-i\theta}}{2i}$

Applications

La forme exponentielle et la formule de Moivre sont particulièrement utiles pour :

  • Calculer des puissances de nombres complexes
  • Linéariser des expressions trigonométriques
  • Résoudre des équations polynomiales dans $\mathbb{C}$
Exercice 3
  1. Écrire sous forme exponentielle $z = \frac{(1+i)^3}{(1-i\sqrt{3})^2}$
  2. Calculer $\left(\frac{1}{2} + i\frac{\sqrt{3}}{2}\right)^{2021}$
  3. Linéariser $\cos^3 x$ et $\sin^3 x \cos x$

Solution :

  1. Forme exponentielle de $1+i$ : $|1+i| = \sqrt{2}$, $\arg(1+i) = \frac{\pi}{4}$, donc $1+i = \sqrt{2}e^{i\frac{\pi}{4}}$

    $(1+i)^3 = (\sqrt{2})^3 e^{i\frac{3\pi}{4}} = 2\sqrt{2}e^{i\frac{3\pi}{4}}$

    Forme exponentielle de $1-i\sqrt{3}$ : $|1-i\sqrt{3}| = 2$, $\arg(1-i\sqrt{3}) = -\frac{\pi}{3}$, donc $1-i\sqrt{3} = 2e^{-i\frac{\pi}{3}}$

    $(1-i\sqrt{3})^2 = 4e^{-i\frac{2\pi}{3}}$

    $z = \frac{2\sqrt{2}e^{i\frac{3\pi}{4}}}{4e^{-i\frac{2\pi}{3}}} = \frac{\sqrt{2}}{2}e^{i(\frac{3\pi}{4} + \frac{2\pi}{3})} = \frac{\sqrt{2}}{2}e^{i(\frac{9\pi}{12} + \frac{8\pi}{12})} = \frac{\sqrt{2}}{2}e^{i\frac{17\pi}{12}}$

    On peut simplifier : $\frac{17\pi}{12} - 2\pi = -\frac{7\pi}{12}$, donc $z = \frac{\sqrt{2}}{2}e^{-i\frac{7\pi}{12}}$

  2. $\frac{1}{2} + i\frac{\sqrt{3}}{2} = e^{i\frac{\pi}{3}}$

    $\left(\frac{1}{2} + i\frac{\sqrt{3}}{2}\right)^{2021} = e^{i\frac{2021\pi}{3}}$

    Simplifions l'exposant : $\frac{2021}{3} = 673 + \frac{2}{3}$ (car $3 \times 673 = 2019$)

    Donc $e^{i\frac{2021\pi}{3}} = e^{i(673\pi + \frac{2\pi}{3})} = e^{i673\pi} \times e^{i\frac{2\pi}{3}}$

    $e^{i673\pi} = (e^{i\pi})^{673} = (-1)^{673} = -1$

    Finalement, $\left(\frac{1}{2} + i\frac{\sqrt{3}}{2}\right)^{2021} = -e^{i\frac{2\pi}{3}} = -\left(-\frac{1}{2} + i\frac{\sqrt{3}}{2}\right) = \frac{1}{2} - i\frac{\sqrt{3}}{2}$

  3. Linéarisation de $\cos^3 x$ :

    $\cos x = \frac{e^{ix} + e^{-ix}}{2}$

    $\cos^3 x = \left(\frac{e^{ix} + e^{-ix}}{2}\right)^3 = \frac{1}{8}(e^{3ix} + 3e^{ix} + 3e^{-ix} + e^{-3ix}) = \frac{1}{4}\left(\frac{e^{3ix} + e^{-3ix}}{2}\right) + \frac{3}{4}\left(\frac{e^{ix} + e^{-ix}}{2}\right) = \frac{1}{4}\cos 3x + \frac{3}{4}\cos x$

    Linéarisation de $\sin^3 x \cos x$ :

    $\sin x = \frac{e^{ix} - e^{-ix}}{2i}$, donc $\sin^3 x = \left(\frac{e^{ix} - e^{-ix}}{2i}\right)^3 = \frac{1}{-8i}(e^{3ix} - 3e^{ix} + 3e^{-ix} - e^{-3ix}) = \frac{i}{8}(e^{3ix} - 3e^{ix} + 3e^{-ix} - e^{-3ix})$

    $\sin^3 x \cos x = \frac{i}{8}(e^{3ix} - 3e^{ix} + 3e^{-ix} - e^{-3ix}) \times \frac{e^{ix} + e^{-ix}}{2} = \frac{i}{16}(e^{4ix} + e^{2ix} - 3e^{2ix} - 3 - 3 - 3e^{-2ix} + e^{-2ix} + e^{-4ix})$

    $= \frac{i}{16}(e^{4ix} + e^{-4ix} - 2e^{2ix} - 2e^{-2ix} - 6) = \frac{i}{16}(2\cos 4x - 4\cos 2x - 6)$

    $= \frac{i}{8}(\cos 4x - 2\cos 2x - 3)$

    Mais ce résultat est imaginaire pur, ce qui n'est pas cohérent. Corrigeons :

    En fait, $\sin^3 x \cos x = \frac{1}{16}(2\sin 2x - \sin 4x)$

Leçon 4 : Racines n-ièmes d'un nombre complexe

Définition

Soit $Z$ un nombre complexe non nul et $n$ un entier naturel ($n \geq 2$).

On appelle racine n-ième de $Z$ tout nombre complexe $z$ tel que :

$z^n = Z$

Méthode de recherche

Soient $Z = re^{i\alpha}$ et $z = \rho e^{i\theta}$.

Alors $z^n = Z \Leftrightarrow \rho^n e^{in\theta} = re^{i\alpha} \Leftrightarrow \begin{cases} \rho^n = r \\ n\theta = \alpha + 2k\pi, k \in \mathbb{Z} \end{cases} \Leftrightarrow \begin{cases} \rho = \sqrt[n]{r} \\ \theta = \frac{\alpha}{n} + \frac{2k\pi}{n}, k \in \mathbb{Z} \end{cases}$

Les $n$ racines n-ièmes de $Z$ sont :

$z_k = \sqrt[n]{r} e^{i\left(\frac{\alpha}{n} + \frac{2k\pi}{n}\right)}, \quad k = 0, 1, 2, ..., n-1$

Remarque :

Si on connaît une racine n-ième $z_0$ de $Z$, alors les autres s'obtiennent en multipliant $z_0$ par les racines n-ièmes de l'unité.

Exercices d'application

Exercice 4
  1. Résoudre dans $\mathbb{C}$ l'équation : $z^5 = \frac{8(1+i)}{\sqrt{3}-i}$
  2. Résoudre dans $\mathbb{C}$ l'équation $z^3 = -2 + 2i$

Solution :

  1. Simplifions d'abord le membre de droite :

    $8(1+i) = 8\sqrt{2}e^{i\frac{\pi}{4}}$

    $\sqrt{3}-i = 2e^{-i\frac{\pi}{6}}$

    Donc $\frac{8(1+i)}{\sqrt{3}-i} = \frac{8\sqrt{2}e^{i\frac{\pi}{4}}}{2e^{-i\frac{\pi}{6}}} = 4\sqrt{2}e^{i(\frac{\pi}{4} + \frac{\pi}{6})} = 4\sqrt{2}e^{i\frac{5\pi}{12}}$

    On cherche $z$ tel que $z^5 = 4\sqrt{2}e^{i\frac{5\pi}{12}}$

    Les solutions sont : $z_k = (4\sqrt{2})^{\frac{1}{5}} e^{i\left(\frac{\frac{5\pi}{12} + 2k\pi}{5}\right)} = (4\sqrt{2})^{\frac{1}{5}} e^{i\left(\frac{\pi}{12} + \frac{2k\pi}{5}\right)}$, $k = 0, 1, 2, 3, 4$

    Simplifions le module : $4\sqrt{2} = 2^2 \cdot 2^{\frac{1}{2}} = 2^{\frac{5}{2}}$, donc $(4\sqrt{2})^{\frac{1}{5}} = 2^{\frac{5}{2} \cdot \frac{1}{5}} = 2^{\frac{1}{2}} = \sqrt{2}$

    Finalement, $z_k = \sqrt{2} e^{i\left(\frac{\pi}{12} + \frac{2k\pi}{5}\right)}$, $k = 0, 1, 2, 3, 4$

  2. $-2 + 2i = 2\sqrt{2}e^{i\frac{3\pi}{4}}$ (car dans le 2ème quadrant)

    On cherche $z$ tel que $z^3 = 2\sqrt{2}e^{i\frac{3\pi}{4}}$

    Les solutions sont : $z_k = (2\sqrt{2})^{\frac{1}{3}} e^{i\left(\frac{\frac{3\pi}{4} + 2k\pi}{3}\right)} = (2\sqrt{2})^{\frac{1}{3}} e^{i\left(\frac{\pi}{4} + \frac{2k\pi}{3}\right)}$, $k = 0, 1, 2$

    Simplifions le module : $2\sqrt{2} = 2^1 \cdot 2^{\frac{1}{2}} = 2^{\frac{3}{2}}$, donc $(2\sqrt{2})^{\frac{1}{3}} = 2^{\frac{3}{2} \cdot \frac{1}{3}} = 2^{\frac{1}{2}} = \sqrt{2}$

    Finalement, $z_k = \sqrt{2} e^{i\left(\frac{\pi}{4} + \frac{2k\pi}{3}\right)}$, $k = 0, 1, 2$
