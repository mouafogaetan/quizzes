Introduction aux Nombres Complexes

Intérêt et Applications

Les nombres complexes constituent un outil fondamental en mathématiques et en sciences physiques. Ils permettent notamment :

  • L'analyse des circuits électroniques en régime sinusoïdal
  • La modélisation en théorie de la diffusion de la chaleur, de l'électricité et de la lumière
  • La résolution d'équations polynomiales qui n'admettent pas de solutions réelles
Exemple Historique

Les nombres complexes ont été introduits pour résoudre des équations du type $x^2 + 1 = 0$ qui n'ont pas de solution dans l'ensemble des nombres réels $\mathbb{R}$.

Application en Physique

En électricité, l'analyse des circuits en courant alternatif utilise intensivement les nombres complexes pour représenter les impédances.

Leçon 1 : Généralités sur les Nombres Complexes

Définition Fondamentale

Un nombre complexe $z$ s'écrit sous la forme $z = a + bi$, où :

  • $a$ et $b$ sont des nombres réels
  • $i$ est l'unité imaginaire définie par $i^2 = -1$
Parties réelle et imaginaire

Pour $z = a + bi$ :

  • $\text{Re}(z) = a$ (partie réelle)
  • $\text{Im}(z) = b$ (partie imaginaire)
Types de nombres complexes
Nombres réels

Si $\text{Im}(z) = 0$, alors $z$ est un nombre réel.

Exemple : $z = 5$ (car $z = 5 + 0i$)

Nombres imaginaires purs

Si $\text{Re}(z) = 0$ et $\text{Im}(z) \neq 0$, alors $z$ est un imaginaire pur.

Exemple : $z = 3i$ (car $z = 0 + 3i$)

Exemple : Identifier les parties réelle et imaginaire

Pour $z = 7 - 2i$ :

  • $\text{Re}(z) = 7$
  • $\text{Im}(z) = -2$

Leçon 2 : Calculs avec les Nombres Complexes

Opérations fondamentales
Addition

Soient $z = a + bi$ et $z' = a' + b'i$ :

$z + z' = (a + a') + (b + b')i$

Soustraction

$z - z' = (a - a') + (b - b')i$

Multiplication

$z \times z' = (a + bi)(a' + b'i) = (aa' - bb') + (ab' + a'b)i$

Exemple : Multiplication

Soient $z = 1 - 2i$ et $z' = -3 + i$ :

$z \times z' = (1)(-3) + (1)(i) + (-2i)(-3) + (-2i)(i)$

$= -3 + i + 6i - 2i^2 = -3 + 7i + 2 = -1 + 7i$

Conjugué et module
Conjugué

Le conjugué de $z = a + bi$ est $\overline{z} = a - bi$

Module

Le module de $z = a + bi$ est $|z| = \sqrt{a^2 + b^2}$

Exemple : Conjugué et module

Pour $z = 3 + 2i$ :

  • $\overline{z} = 3 - 2i$
  • $|z| = \sqrt{3^2 + 2^2} = \sqrt{13}$

Leçon 3 : Représentation Géométrique

Plan complexe

Un nombre complexe $z = a + bi$ peut être représenté dans le plan par le point $M$ de coordonnées $(a, b)$ :

Représentation d'un nombre complexe dans le plan

Axe réel et axe imaginaire
  • Axe des abscisses : axe réel
  • Axe des ordonnées : axe imaginaire
Affixe d'un point

Le nombre complexe $z$ associé au point $M$ est appelé affixe de $M$.

Exemple : Placement de points

Placer les points d'affixes :

  • $z_A = 1 + i$ → point A(1, 1)
  • $z_B = 1 - i$ → point B(1, -1)
  • $z_C = -2i$ → point C(0, -2)
  • $z_D = -3$ → point D(-3, 0)

Leçon 4 : Formes Trigonométrique et Exponentielle

Forme trigonométrique

Tout nombre complexe $z = a + bi$ peut s'écrire sous la forme :

$z = r(\cos\theta + i\sin\theta)$

Où :

  • $r = |z| = \sqrt{a^2 + b^2}$ (module)
  • $\theta = \arg(z)$ (argument)
Exemple : Forme trigonométrique

Pour $z = 1 + i$ :

  • $r = \sqrt{1^2 + 1^2} = \sqrt{2}$
  • $\theta = \arctan(1/1) = \frac{\pi}{4}$ (modulo $2\pi$)
  • Forme trigonométrique : $z = \sqrt{2}\left(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4}\right)$
Forme exponentielle

En utilisant la formule d'Euler $e^{i\theta} = \cos\theta + i\sin\theta$, on obtient :

$z = re^{i\theta}$

Formule de Moivre

$(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$

Exemple d'application

Calculons $\left(\frac{1}{2} + i\frac{\sqrt{3}}{2}\right)^{2007}$ :

On reconnaît $\cos\frac{\pi}{3} + i\sin\frac{\pi}{3} = e^{i\pi/3}$

Donc $\left(e^{i\pi/3}\right)^{2007} = e^{i\cdot 2007\pi/3} = e^{i\cdot 669\pi} = (e^{i\pi})^{669} = (-1)^{669} = -1$

Leçon 5 : Équations du Second Degré dans $\mathbb{C}$

Résolution d'équations quadratiques

Pour résoudre $az^2 + bz + c = 0$ avec $a \neq 0$ :

Discriminant

$\Delta = b^2 - 4ac$

Solutions

$z = \frac{-b \pm \sqrt{\Delta}}{2a}$

Exemple : Résolution dans $\mathbb{C}$

Résoudre $x^2 + x + 1 = 0$ :

$\Delta = 1^2 - 4(1)(1) = -3 = 3i^2$

Donc $\sqrt{\Delta} = \pm i\sqrt{3}$

Les solutions sont : $x = \frac{-1 \pm i\sqrt{3}}{2}$

Racines carrées d'un nombre complexe

Pour trouver les racines carrées de $z = a + bi$, on cherche $u = x + yi$ tel que $u^2 = z$.

Exemple : Racines carrées

Trouver les racines carrées de $z = -5 + 12i$ :

On pose $u = x + yi$ avec $u^2 = z$

On obtient le système :

  • $x^2 - y^2 = -5$
  • $2xy = 12$
  • $x^2 + y^2 = 13$ (car $|u|^2 = |z|$)

En résolvant, on trouve $u_1 = 2 + 3i$ et $u_2 = -2 - 3i$
