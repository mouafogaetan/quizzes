Instruments d'Optique

Microscope, lunette astronomique et télescope de Newton

Constitution et principe

Un microscope est constitué de deux systèmes convergents :

  • L'objectif : lentille convergente de courte distance focale ($f_1'$ petit)
  • L'oculaire : lentille convergente jouant comme une loupe

L'objet à observer est placé légèrement au-delà du foyer objet de l'objectif, qui en donne une image réelle, agrandie et renversée. Cette image sert d'objet pour l'oculaire qui en donne une image virtuelle, agrandie.

Description: Schéma montrant le trajet des rayons lumineux à travers l'objectif et l'oculaire d'un microscope.

Grossissement et puissance

Le grossissement commercial $G_c$ du microscope est donné par :

$G_c = G_1 \times G_2 = \frac{\delta}{f_1'} \times \frac{1}{f_2'}$

Où :

  • $G_1$ : grossissement de l'objectif
  • $G_2$ : grossissement de l'oculaire
  • $f_1'$ : distance focale de l'objectif
  • $f_2'$ : distance focale de l'oculaire
  • $\delta$ : distance entre les foyers secondaires (intervalle optique), généralement fixée à 16 cm

La puissance $P$ est définie comme le grossissement angulaire lorsque l'image est à l'infini :

$P = \frac{\delta}{f_1' \cdot f_2'}$

Exercice d'application

Un microscope est constitué d'un objectif de distance focale $f_1' = 4mm$ et d'un oculaire de distance focale $f_2' = 20mm$. L'intervalle optique $\delta = 160mm$.

1. Calculez le grossissement commercial de ce microscope.

2. Déterminez sa puissance.

Solution :

1. Grossissement commercial :

$G_c = \frac{\delta}{f_1'} \times \frac{1}{f_2'} = \frac{160}{4} \times \frac{1}{0.02} = 40 \times 50 = 200$

2. Puissance :

$P = \frac{\delta}{f_1' \cdot f_2'} = \frac{160}{0.004 \times 0.02} = \frac{160}{0.00008} = 2 \times 10^6$ dioptries

Constitution et principe

Une lunette astronomique est constituée de :

  • Un objectif convergent de grande distance focale ($f_1'$ grand)
  • Un oculaire convergent de courte distance focale ($f_2'$ petit)

L'objet observé (à l'infini) donne through l'objectif une image réelle située dans le plan focal image de l'objectif. Cette image sert d'objet pour l'oculaire qui en donne une image à l'infini.

Description: Schéma montrant le trajet des rayons lumineux dans une lunette astronomique en mode afocal.

Grossissement

Le grossissement $G$ d'une lunette astronomique afocale (image à l'infini) est donné par :

$G = -\frac{f_1'}{f_2'}$

Où le signe négatif indique que l'image est renversée.

Pour une lunette terrestre, on ajoute un système pour redresser l'image, ce qui modifie légèrement la formule.

Exercice d'application

Une lunette astronomique est constituée d'un objectif de distance focale $f_1' = 800mm$ et d'un oculaire de distance focale $f_2' = 20mm$.

1. Calculez le grossissement de cette lunette.

2. Déterminez la longueur de la lunette (distance entre l'objectif et l'oculaire).

Solution :

1. Grossissement :

$G = -\frac{f_1'}{f_2'} = -\frac{800}{20} = -40$

Le grossissement est de 40 fois et l'image est renversée.

2. Longueur de la lunette :

Pour une lunette afocale, la distance entre objectif et oculaire est :

$L = f_1' + f_2' = 800 + 20 = 820mm$

Télescope de Newton

Le télescope de Newton utilise un miroir concave (objectif) au lieu d'une lentille :

  • Un miroir primaire concave (sphérique ou parabolique)
  • Un miroir secondaire plan qui dévie la lumière sur le côté
  • Un oculaire convergent

Les avantages du télescope par rapport à la lunette :

  • Absence d'aberration chromatique
  • Encombrement réduit pour un même diamètre
  • Coût moindre pour les grandes dimensions

Description: Schéma montrant le trajet de la lumière dans un télescope de Newton avec miroir primaire concave et miroir secondaire plan.

Grossissement

Le grossissement $G$ d'un télescope est donné par :

$G = -\frac{f_1'}{f_2'}$

Où :

  • $f_1'$ : distance focale du miroir primaire
  • $f_2'$ : distance focale de l'oculaire
Exercice d'application

Un télescope de Newton possède un miroir primaire de distance focale $f_1' = 1,2m$ et on utilise un oculaire de distance focale $f_2' = 10mm$.

1. Calculez le grossissement de ce télescope.

2. Si on remplace l'oculaire par un oculaire de $f_2' = 25mm$, quel sera le nouveau grossissement?

Solution :

1. Grossissement avec l'oculaire de 10mm :

$G = -\frac{f_1'}{f_2'} = -\frac{1200}{10} = -120$

2. Grossissement avec l'oculaire de 25mm :

$G = -\frac{1200}{25} = -48$

Plus la distance focale de l'oculaire est petite, plus le grossissement est important.
