Acides Nucléiques et Biosynthèse des Protéines

Structure des Acides Nucléiques

Nucléotides

Les acides nucléiques (ADN et ARN) sont des polymères de nucléotides. Chaque nucléotide est composé de :

  • Une base azotée (Adénine, Thymine/Uracile, Cytosine, Guanine)
  • Un sucre (désoxyribose pour l'ADN, ribose pour l'ARN)
  • Un groupe phosphate

Double hélice d'ADN

Modèle de Watson et Crick : deux brins antiparallèles formant une double hélice. Les bases sont complémentaires :

  • A avec T (2 liaisons hydrogène)
  • G avec C (3 liaisons hydrogène)

Différences ADN/ARN

Réplication de l'ADN

Mécanisme semi-conservatif

La réplication de l'ADN est dite semi-conservative car chaque nouvelle molécule d'ADN est constituée d'un brin parental et d'un brin néoformé.

Étapes de la réplication

  1. Initiation : Ouverture de la double hélice au niveau des origines de réplication par l'hélicase
  2. Élongation : Synthèse des nouveaux brins par l'ADN polymérase en suivant les règles de complémentarité
  3. Terminaison : Les fourches de réplication se rencontrent et les nouvelles molécules se séparent

Enzymes impliquées

  • Hélicase : sépare les deux brins d'ADN
  • ADN polymérase : ajoute les nucléotides complémentaires
  • Primase : synthétise une amorce ARN
  • Ligase : joint les fragments d'Okazaki

Exercice : Brins retardés et continus

Pourquoi parle-t-on de brin continu et de brin discontinu lors de la réplication de l'ADN ?

L'ADN polymérase ne peut synthétiser que dans le sens 5'→3'.

Sur le brin où la synthèse se fait dans le même sens que l'ouverture de la fourche de réplication, elle est continue (brin avancé).

Sur l'autre brin, la synthèse se fait en fragments d'Okazaki (brin retardé) car elle doit s'effectuer en direction opposée à l'ouverture de la fourche.

Relation Gène - Protéine

Le gène

Un gène est une séquence d'ADN qui code pour une protéine ou un ARN fonctionnel. Il contient l'information génétique nécessaire à la synthèse d'une protéine.

Code génétique

Le code génétique est universel, dégénéré et non chevauchant. Il est composé de codons (triplets de nucléotides) qui spécifient les acides aminés.

Il existe 64 codons possibles :

  • 61 codons qui codent pour les 20 acides aminés
  • 3 codons stop (UAA, UAG, UGA) qui signalent la fin de la traduction
  • 1 codon start (AUG) qui initie la traduction et code pour la méthionine

Exercice : Lecture du code génétique

Quel acide aminé est codé par le codon AUG ? Et par le codon UUU ?

AUG code pour la Méthionine (et c'est aussi le codon start).

UUU code pour la Phénylalanine.

Biosynthèse des Protéines

Transcription (ADN → ARNm)

Processus par lequel l'information génétique de l'ADN est copiée en ARN messager (ARNm) :

  1. Initiation : fixation de l'ARN polymérase sur le promoteur
  2. Élongation : synthèse de l'ARNm dans le sens 5'→3'
  3. Terminaison : arrêt de la transcription au niveau du terminateur

Traduction (ARNm → Protéine)

Processus par lequel l'information de l'ARNm est décodée pour synthétiser une protéine :

  1. Initiation : assemblage du ribosome sur l'ARNm
  2. Élongation : ajout d'acides aminés à la chaîne polypeptidique
  3. Terminaison : libération de la protéine lorsqu'un codon stop est atteint

Acteurs moléculaires

  • Ribosomes : complexes ribonucléoprotéiques où se déroule la traduction
  • ARNt (ARN de transfert) : apporte les acides aminés au ribosome
  • ARNm (ARN messager) : porte l'information génétique

Explications :
