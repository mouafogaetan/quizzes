Leçon 5 : Oxydoréduction en Solution Aqueuse

1. Définitions fondamentales

Terme Définition Exemple
Oxydation Transformation chimique où une espèce chimique perd des électrons $Zn \rightarrow Zn^{2+} + 2e^-$
Réduction Transformation chimique où une espèce chimique gagne des électrons $Cu^{2+} + 2e^- \rightarrow Cu$
Oxydant Espèce chimique capable de capter des électrons $Cu^{2+}$, $H_3O^+$
Réducteur Espèce chimique capable de perdre des électrons $Zn$, $Fe$, $Al$
Oxydoréduction Réaction chimique où il y a transfert d'électrons entre espèces $Zn + Cu^{2+} \rightarrow Zn^{2+} + Cu$
Exemple d'application :

Dans la réaction $Fe + 2H_3O^+ \rightarrow Fe^{2+} + H_2 + 2H_2O$ :

  • Le fer ($Fe$) s'oxyde : $Fe \rightarrow Fe^{2+} + 2e^-$ (réducteur)
  • L'ion hydronium ($H_3O^+$) se réduit : $2H_3O^+ + 2e^- \rightarrow H_2 + 2H_2O$ (oxydant)

2. Action des acides sur les métaux

Certains métaux réagissent avec les acides en libérant du dihydrogène ($H_2$).

Métal + Acide → Sel + Dihydrogène

Exemples :

  • $Zn + 2H_3O^+ \rightarrow Zn^{2+} + H_2 + 2H_2O$
  • $Fe + 2H_3O^+ \rightarrow Fe^{2+} + H_2 + 2H_2O$
  • $2Al + 6H_3O^+ \rightarrow 2Al^{3+} + 3H_2 + 6H_2O$

Remarque : Les ions $Cl^-$ et $SO_4^{2-}$ n'interviennent pas dans ces réactions et sont dits "spectateurs".

Exercice 1 :

Écrire l'équation bilan de la réaction entre l'aluminium et l'acide sulfurique ($H_2SO_4$).

L'acide sulfurique en solution aqueuse produit des ions $H_3O^+$.

Équation bilan : $2Al + 6H_3O^+ \rightarrow 2Al^{3+} + 3H_2 + 6H_2O$

En tenant compte de l'ion spectateur $SO_4^{2-}$ :

$2Al + 3H_2SO_4 + 6H_2O \rightarrow 2Al^{3+} + 3SO_4^{2-} + 3H_2 + 6H_2O$

Qui se simplifie en : $2Al + 3H_2SO_4 \rightarrow Al_2(SO_4)_3 + 3H_2$

3. Action des ions métalliques sur les métaux

Un métal peut réduire les ions d'un autre métal si celui-ci est moins réducteur.

Exemple :

Dans une solution de sulfate de cuivre ($Cu^{2+} + SO_4^{2-}$) :

  • Le zinc réduit les ions cuivre : $Zn + Cu^{2+} \rightarrow Zn^{2+} + Cu$
  • Le fer réduit les ions cuivre : $Fe + Cu^{2+} \rightarrow Fe^{2+} + Cu$

Par contre, le cuivre ne réduit pas les ions zinc : $Cu + Zn^{2+} \rightarrow$ pas de réaction

Exercice 2 :

On plonge une lame de zinc dans une solution de sulfate de fer II ($Fe^{2+} + SO_4^{2-}$).

1. Observera-t-on une réaction chimique?

2. Si oui, écrire l'équation bilan de la réaction.

1. Oui, il y aura une réaction car le zinc est un réducteur plus fort que le fer.

2. Équation bilan : $Zn + Fe^{2+} \rightarrow Zn^{2+} + Fe$

Le zinc s'oxyde : $Zn \rightarrow Zn^{2+} + 2e^-$

Les ions fer II se réduisent : $Fe^{2+} + 2e^- \rightarrow Fe$

4. Applications aux situations problèmes

Situation problème 1 :

Les fils de fer changent de couleur car le fer s'oxyde au contact de l'air et de l'eau acide :

$4Fe + 3O_2 + 6H_2O \rightarrow 4Fe^{3+} + 12OH^- \rightarrow 4Fe(OH)_3$ (rouille)

L'aluminium et le cuivre ne sont pas attaqués car ils forment une couche protectrice d'oxyde qui les protège contre une corrosion ultérieure.

Situation problème 2 :

Le sulfate de cuivre anhydre ($CuSO_4$) est blanc. Lorsqu'on ajoute de l'eau, il se hydrate pour former du sulfate de cuivre pentahydraté ($CuSO_4·5H_2O$) qui est bleu.

La coloration bleue caractérise les ions cuivre II hydratés $[Cu(H_2O)_6]^{2+}$.

Exercice 3 :

Expliquer pourquoi l'argent ne réagit pas avec l'acide chlorhydrique alors que le zinc oui.

L'argent est un métal moins réducteur que le zinc. Son potentiel standard d'oxydoréduction est plus élevé, ce qui signifie qu'il a moins tendance à perdre des électrons que le zinc.

L'ion $Ag^+$ est un oxydant plus faible que $H_3O^+$, donc la réaction $Ag + H_3O^+ \rightarrow Ag^+ + H_2$ n'est pas favorable thermodynamiquement.

Au contraire, le zinc est un réducteur fort et l'ion $Zn^{2+}$ est un oxydant faible, donc la réaction $Zn + 2H_3O^+ \rightarrow Zn^{2+} + H_2 + 2H_2O$ est spontanée.
