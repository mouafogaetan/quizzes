Condensé Leçon 3 : Les Composés Oxygénés

1. Définitions et Formules Générales

Un composé oxygéné est un corps pur organique dont la molécule comporte au moins un atome d'oxygène.

Composés Formule générale Groupe fonctionnel
Alcool $R-OH$ Groupe hydroxyle (-OH)
Éther-oxyde $R-O-R'$ Liaison éther (C-O-C)
Aldéhyde $R-CHO$ Carbonyle terminal (-CHO)
Cétone $R-CO-R'$ Carbonyle interne (>C=O)
Acide carboxylique $R-COOH$ Carboxyle (-COOH)
Ester $R-COO-R'$ Ester (-COO-)
Exemple :

Nommer le composé suivant :

$CH_3-CH(CH_3)-CH_2-CH_2-C(CH_3)(OH)-CH_3$

Solution : La chaîne carbonée la plus longue a 7 atomes de carbone. Le carbone fonctionnel (portant -OH) doit avoir l'indice le plus petit possible. Le nom est : 2,5-diméthylheptan-2-ol.

2. Classes d'Alcools

Les alcools sont classés selon le nombre de groupes alkyle liés au carbone fonctionnel :

  • Alcool primaire : Le carbone fonctionnel est lié à 1 groupe alkyle
  • Alcool secondaire : Le carbone fonctionnel est lié à 2 groupes alkyles
  • Alcool tertiaire : Le carbone fonctionnel est lié à 3 groupes alkyles
Exemples :
  • $CH_3-CH_2-OH$ (Éthanol) : Alcool primaire
  • $CH_3-CH(OH)-CH_3$ (Propan-2-ol) : Alcool secondaire
  • $(CH_3)_3C-OH$ (2-méthylpropan-2-ol) : Alcool tertiaire
3. Réactions des Alcools
3.1 Déshydratation intermoléculaire

Formation d'éther-oxyde à partir de deux molécules d'alcool :

$CH_3-CH_2-OH + CH_3-CH_2-OH \xrightarrow{Al_2O_3} CH_3-CH_2-O-CH_2-CH_3 + H_2O$

Éthoxyéthane (éther diéthylique)

3.2 Oxydation

a) Combustion complète :

$C_nH_{2n+1}OH + \frac{3n}{2}O_2 \rightarrow nCO_2 + (n+1)H_2O$

b) Oxydation ménagée :

  • Alcools primaires → Aldéhydes → Acides carboxyliques
  • Alcools secondaires → Cétones
  • Alcools tertiaires → Pas d'oxydation ménagée
Exemples d'oxydation :

Avec excès de dichromate :

$3R-CH_2-OH + 2Cr_2O_7^{2-} + 16H_3O^+ \rightarrow 3R-COOH + 4Cr^{3+} + 27H_2O$

Avec défaut de permanganate :

$5R-CH_2-OH + 2MnO_4^- + 6H_3O^+ \rightarrow 5R-CHO + 2Mn^{2+} + 14H_2O$
3.3 Estérification

Réaction entre un acide carboxylique et un alcool :

$R-COOH + R'-OH \rightleftharpoons R-COO-R' + H_2O$

Caractéristiques : lente, limitée, athermique

4. Propriétés Physiques des Alcools
  • Températures d'ébullition plus élevées que les alcanes de masse molaire comparable (liaisons hydrogène)
  • Solubilité qui diminue avec l'allongement de la chaîne carbonée
  • Solubilité qui augmente avec le nombre de groupes hydroxyle
Comparaison :

Éthanol ($C_2H_5OH$, M=46 g/mol) : Téb = 78°C

Butane ($C_4H_{10}$, M=58 g/mol) : Téb = -0.5°C

La différence s'explique par la présence de liaisons hydrogène dans l'éthanol.

5. Tests de Reconnaissance
Test/Réactif Aldéhyde Cétone
Réactif de Schiff Coloration rose (+) Incolore (-)
2,4-DNPH Précipité jaune (+) Précipité jaune (+)
Papier pH humide Ne rougit pas (-) Ne rougit pas (-)
Réactif de Tollens Miroir d'argent (+) Rien (-)
Exercice d'application :

Dans un laboratoire, on dispose de trois flacons contenant trois alcools : l'alcool benzylique, le propan-2-ol et le méthylpropan-2-ol dont on a perdu les étiquettes.

  • Le premier ne réagit pas avec une solution acidifiée de permanganate de potassium.
  • Le deuxième réagit avec une solution acidifiée de permanganate de potassium en excès pour donner un composé qui rougit le papier pH humide.
  • Le troisième subit une déshydrogénation catalytique pour donner un composé qui donne un précipité jaune avec la 2,4-DNPH et est sans action sur le réactif de Schiff.

Voir la solution

Solution :

  1. Le premier alcool qui ne réagit pas avec le permanganate est un alcool tertiaire : méthylpropan-2-ol $(CH_3)_3C-OH$
  2. Le deuxième qui donne un composé acidique (rougit le papier pH) est un alcool primaire oxydé en acide carboxylique : alcool benzylique $C_6H_5-CH_2-OH$
  3. Le troisième qui donne une cétone (précipité avec 2,4-DNPH mais pas avec Schiff) est un alcool secondaire : propan-2-ol $CH_3-CH(OH)-CH_3$
6. Polymérisation (Rappel de la leçon précédente)

L'indice de polymérisation $n$ se calcule par :

$n = \frac{M_p}{M_m}$

Où $M_p$ est la masse molaire du polymère et $M_m$ la masse molaire du monomère.

Exercice :

Calculer l'indice de polymérisation d'un polymère dérivant du 2-chlorobut-1-ène, sachant que sa masse molaire est de 85 kg/mol.

Masses molaires (g/mol) : H=1 ; C=12 ; Cl=35,5

Voir la solution

Solution :

Masse molaire du monomère (2-chlorobut-1-ène) : $C_4H_7Cl$

Calcul : $M_m = 4 \times 12 + 7 \times 1 + 35.5 = 48 + 7 + 35.5 = 90.5$ g/mol

Masse molaire du polymère : $M_p = 85$ kg/mol $= 85000$ g/mol

Indice de polymérisation : $n = \frac{85000}{90.5} \approx 939$
