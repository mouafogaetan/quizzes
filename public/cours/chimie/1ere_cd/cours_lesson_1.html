Condensé de la Leçon 1 : Les Alcanes

1. Définition et structure des alcanes

Les alcanes sont des hydrocarbures saturés de formule générale $C_nH_{2n+2}$ où $n$ est un entier naturel non nul.

Caractéristiques structurales :
  • Atome de carbone tétravalent (forme 4 liaisons covalentes)
  • Structure tétraédrique autour du carbone (angles de 109,5°)
  • Liaisons simples C-C et C-H uniquement
  • Rotation libre possible autour des liaisons simples
Exemples :

Méthane : CH₄ (n=1) - le plus simple des alcanes

Éthane : C₂H₆ (n=2) - formule semi-développée : CH₃-CH₃

Propane : C₃H₈ (n=3) - formule semi-développée : CH₃-CH₂-CH₃

Exercice d'application :

Donnez la formule brute et la formule semi-développée du butane (n=4).

Formule brute : C₄H₁₀

Formule semi-développée : CH₃-CH₂-CH₂-CH₃

2. Isomérie des alcanes

Des molécules ayant la même formule brute mais des formules développées différentes sont appelées isomères.

À partir du butane (C₄H₁₀), plusieurs isomères de constitution (chaine) sont possibles :

Exemple pour C₄H₁₀ :

CH₃-CH₂-CH₂-CH₃ (butane)

CH₃-CH(CH₃)-CH₃ (2-méthylpropane)

Nombre de carbones Formule brute Nombre d'isomères
1CH₄1
2C₂H₆1
3C₃H₈1
4C₄H₁₀2
5C₅H₁₂3
6C₆H₁₄5
Exercice d'application :

Combien d'isomères existe-t-il pour l'heptane (C₇H₁₆) ?

Il existe 9 isomères pour l'heptane (C₇H₁₆).

3. Nomenclature des alcanes
Règles de nomenclature IUPAC :
  1. Identifier la chaîne carbonée la plus longue (chaîne principale)
  2. Numéroter les atomes de carbone pour donner les plus petits indices aux substituants
  3. Nommer les substituants alkyliques (méthyle, éthyle, propyle, etc.)
  4. Indiquer la position de chaque substituant
  5. Classer les substituants par ordre alphabétique
  6. Utiliser les préfixes di-, tri-, tétra- pour les substituants identiques
Exemples :

CH₃-CH(CH₃)-CH₂-CH₃ → 2-méthylbutane

CH₃-CH₂-CH(CH₃)-CH₂-CH₃ → 3-méthylpentane

CH₃-C(CH₃)₂-CH₂-CH₃ → 2,2-diméthylbutane

Exercice d'application :

Nommez le composé suivant : CH₃-CH₂-CH(CH₃)-CH(CH₃)-CH₃

La chaîne principale a 5 carbones (pentane).

Les substituants sont sur les carbones 2 et 3 : 2,3-diméthylpentane

4. Propriétés physiques des alcanes
  • Insolubles dans l'eau (hydrophobes)
  • Solubles dans les solvants organiques
  • Densité inférieure à celle de l'eau
  • État physique dépendant de la masse molaire :
    • C₁-C₄ : gazeux
    • C₅-C₁₅ : liquides
    • C₁₆ et plus : solides
  • Les isomères ramifiés ont des températures d'ébullition plus faibles que les linéaires
Exercice d'application :

Quel alcane entre le pentane (CH₃-CH₂-CH₂-CH₂-CH₃) et le 2,2-diméthylpropane (C(CH₃)₄) a la température d'ébullition la plus élevée ? Justifiez.

Le pentane a une température d'ébullition plus élevée car sa structure linéaire permet des interactions intermoléculaires plus importantes (forces de London) que la structure ramifiée du 2,2-diméthylpropane.

5. Propriétés chimiques des alcanes
A. Combustion

Réaction exothermique avec le dioxygène :

Combustion complète : $C_nH_{2n+2} + \frac{3n+1}{2}O_2 \rightarrow nCO_2 + (n+1)H_2O + énergie$

Combustion incomplète (manque d'oxygène) : formation de CO et/ou C

Exemple avec le méthane :

Combustion complète : $CH_4 + 2O_2 \rightarrow CO_2 + 2H_2O$

Combustion incomplète : $CH_4 + O_2 \rightarrow C + 2H_2O$ ou $2CH_4 + 3O_2 \rightarrow 2CO + 4H_2O$

B. Réactions de substitution (halogénation)

Les atomes d'hydrogène peuvent être remplacés par des halogènes (Cl, Br) sous l'effet de la lumière UV.

Exemple avec le méthane :

$CH_4 + Cl_2 \xrightarrow{h\nu} CH_3Cl + HCl$ (chlorométhane)

$CH_3Cl + Cl_2 \xrightarrow{h\nu} CH_2Cl_2 + HCl$ (dichlorométhane)

La réaction continue jusqu'à formation de CCl₄ (tétrachlorométhane)

Exercice d'application :

Écrivez l'équation de la réaction entre le propane et le dichlore en présence de lumière UV pour former le 1-chloropropane.

$CH_3-CH_2-CH_3 + Cl_2 \xrightarrow{h\nu} CH_3-CH_2-CH_2Cl + HCl$

6. Préparation du méthane

Au laboratoire, le méthane peut être préparé par action de l'eau sur le carbure d'aluminium :

$Al_4C_3 + 12H_2O \rightarrow 4Al(OH)_3 + 3CH_4$

Exercice d'application :

Quel volume de méthane, mesuré dans les conditions normales de température et de pression, peut-on obtenir à partir de 18g de carbure d'aluminium ?

Données : M(Al₄C₃) = 144 g/mol ; Volume molaire = 22,4 L/mol

Masse molaire de Al₄C₃ = 144 g/mol

Nombre de moles de Al₄C₃ = 18/144 = 0,125 mol

D'après l'équation : 1 mole de Al₄C₃ produit 3 moles de CH₄

Donc 0,125 mole de Al₄C₃ produit 0,375 mole de CH₄

Volume de CH₄ = 0,375 × 22,4 = 8,4 L

7. Exercice de synthèse

On fait réagir du dichlore sur un alcane de masse molaire 44 g/mol. On obtient un composé de masse molaire 115 g/mol.

Tâche 1 :

Donnez les formules brutes des deux composés.

L'alcane a pour formule générale CₙH₂ₙ₊₂ et masse molaire 12n + (2n+2) = 14n+2 = 44 g/mol

Donc 14n = 42 ⇒ n = 3

L'alcane est le propane : C₃H₈

Le composé chloré a une masse molaire de 115 g/mol. La substitution d'un H par un Cl augmente la masse molaire de 35,5 (Cl) - 1 (H) = 34,5 g/mol

Masse molaire du composé chloré = 44 + 34,5 = 78,5 g/mol ≠ 115 g/mol

Il faut donc plusieurs substitutions : 115 - 44 = 71 g/mol

71 / 34,5 ≈ 2, donc deux atomes de chlore ont été substitués

Le composé est donc C₃H₆Cl₂

Tâche 2 :

Donnez les formules développées possibles pour le composé chloré.

Pour C₃H₆Cl₂, les isomères possibles sont :

1,1-dichloropropane : CH₃-CH₂-CHCl₂

1,2-dichloropropane : CH₃-CHCl-CH₂Cl

1,3-dichloropropane : ClCH₂-CH₂-CH₂Cl

2,2-dichloropropane : CH₃-CCl₂-CH₃
